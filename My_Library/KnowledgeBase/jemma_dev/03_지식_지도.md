# 3. My_Library 지식 지도 (한 장)

**프로젝트 루트:** `D:\MONEY lol\My_Library`

| 경로 | 성격 | 잼마가 할 일 |
|------|------|----------------|
| `app/main.py` | 앱 진입 | “어디서부터 흐르는지”만 파악 |
| `app/api/` | HTTP API | 엔드포인트·요청 흐름 |
| `app/core/` | 도메인 로직 (검색, 건강검사 등) | 변경 시 **부사수와** 같이 |
| `app/db/` | 모델·`init_db` | **스키마 = 마이그레이션** 필수 (거의 항상 부사수) |
| `app/static/` | 프론트 (JS/CSS/HTML) | **UI 바꾸면** `index.html` 캐시버스트 |
| `data/library.db` | SQLite | **직접 바이너리 편집 금지** |
| `My_Library\Wiki\` / Raw 등 | **자료·위키** | **원본 content 보호** — 규칙에 따른다. |
| `10_Wiki\` (상위 `MONEY lol`) | **개인 개념 노트** | 앱 **필수**와 **혼동**하지 말기 |
| `jemma\` (프로젝트 루트) | **잼마 전용** · 주제/인덱스 | `learned/`, `topics/`, `bridge_to_library/` — **우리 자료는 읽기·경로·인덱스** (`jemma\README.md`) |
| `jemma_dev\from_connect_ai_brain\` | **Connect AI** 로컬 뇌 | `00_Raw`, `10_Wiki` — **Cursor** `connectAiLab.localBrainPath` |
| `KnowledgeBase\` (여기) | **방법론·팀 룰** | 잼마·인간 공용 **초급·문서** |

**규칙 파일 (반드시 존중):**  
`My_Library\.cursor\rules\` — 특히 `my-library.mdc`, `change-scope`, `collaboration-terms`, `reporting`.
