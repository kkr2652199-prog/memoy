# 병렬 실행

## 설명

병렬 실행은 여러 계산 작업을 동시에 처리하여 전체 처리 시간을 단축하고 시스템 효율성을 높이는 컴퓨팅 기법입니다.

유형/분류:
- 태스크 병렬 처리: 독립적인 여러 작업을 동시에 실행하여 전체 처리량을 높이는 방식
- 데이터 병렬 처리: 동일한 연산을 대규모 데이터 세트의 여러 부분에 동시에 적용하는 방식
- 명령어 수준 병렬 처리: CPU 내부에서 여러 기계어 명령어를 동시에 실행하여 파이프라인 효율을 극대화하는 방식

핵심 원리: 병렬 실행은 단일 문제를 여러 개의 작은 하위 작업으로 분할하고, 이들을 동시에 처리할 수 있는 여러 프로세서나 코어에 할당하여 전체 작업 완료 시간을 줄이는 방식으로 작동합니다. 이를 통해 시스템 자원 활용도를 높이고 처리량을 향상시킵니다.
