# 웹 스크래핑

## 설명

웹 스크래핑은 웹 페이지에서 데이터를 자동으로 추출하는 기술입니다.

유형/분류:
- 정적 스크래핑: 웹 페이지의 HTML 소스 코드에서 직접 데이터를 추출하는 방식.
- 동적 스크래핑: 자바스크립트 등으로 동적으로 생성되는 웹 페이지의 최종 콘텐츠를 브라우저 에뮬레이션을 통해 추출하는 방식.
- 대규모 스크래핑: 여러 페이지나 웹사이트에서 대량의 데이터를 효율적으로 수집하기 위해 분산 또는 병렬 처리 방식을 사용하는 것.

핵심 원리: 웹 스크래핑은 HTTP 요청을 통해 웹 서버로부터 웹 페이지의 HTML/XML 데이터를 받아온 후, 파싱(parsing) 기술을 이용하여 해당 데이터에서 필요한 정보를 식별하고 추출하는 방식으로 작동합니다. 이 과정에서 웹 페이지의 구조와 패턴을 이해하는 것이 중요합니다.

관련 도구: [[Beautiful Soup]], [[Scrapy]], [[Selenium]]
