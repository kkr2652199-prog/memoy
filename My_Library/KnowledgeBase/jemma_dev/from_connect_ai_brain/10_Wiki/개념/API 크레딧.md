# API 크레딧

## 설명

API 크레딧은(는) API 서비스의 접근 및 사용량을 측정하고, 구독 정책에 따라 할당 및 제한될 수 있는 가상의 단위이다.

유형/분류:
- 사용량 기반 크레딧: API 호출 횟수, 데이터 처리량 등 실제 사용량에 따라 차감되는 방식
- 구독 기반 크레딧: 특정 구독 플랜에 따라 일정 기간 동안 정해진 양이 제공되는 방식
- 정책 제한 크레딧: 특정 사용 환경(예: 제3자 도구)에서 사용이 제한되거나 차등 적용되는 방식

핵심 원리: API 크레딧은 서비스 제공자가 서버 자원을 보호하고 서비스 사용량을 통제하기 위해 사용된다. 이는 구독자에게 약속된 사용량과 실제 사용 가능한 양 사이에 불일치를 야기할 수 있으며, 특히 제3자 도구 사용 시 제한을 통해 논란이 될 수 있다.
