# Agentic Coding

## 설명

Agentic Coding은(는) 인공지능 에이전트를 활용하여 코딩 작업을 자동화, 관리 및 최적화하며, 특히 여러 에이전트의 병렬 실행과 워크플로우 조정을 통해 개발 생산성을 향상시키는 접근 방식이다.

유형/분류:
- AI 에이전트 오케스트레이션: 여러 AI 에이전트의 작업을 통합적으로 관리하고 조정하여 복잡한 개발 프로젝트를 수행.
- 병렬 작업 관리: 다수의 코딩 작업을 동시에 실행하고 워크트리를 통해 충돌을 방지하며 효율성을 극대화.
- 클라우드 기반 실행: 로컬 환경에 구애받지 않고 클라우드 서버에서 장시간 또는 고성능 작업을 지속적으로 처리.
- 개발 생산성 향상: 사용자 친화적인 인터페이스와 통합 터미널, 실시간 변경 사항 리뷰 기능 등을 통해 개발 워크플로우를 간소화하고 효율을 증대.

핵심 원리: AI 에이전트들이 독립적으로 또는 협력적으로 코딩 작업을 수행하도록 지시하고, 이들의 실행을 중앙에서 '관제탑'처럼 관리하여 병렬 처리의 효율성을 높이고 잠재적 충돌을 방지한다. 이를 통해 개발자는 복잡한 프로젝트를 보다 체계적이고 신속하게 진행할 수 있다.
