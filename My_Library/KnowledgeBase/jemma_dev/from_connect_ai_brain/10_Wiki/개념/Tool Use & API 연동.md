# Tool Use & API 연동

## 설명

Tool Use & API 연동은(는) 인공지능 모델의 기능을 외부 도구 및 시스템과 통합하여 복잡한 작업을 자동화하고 확장된 워크플로우를 구축하는 전략적 접근 방식이다.

유형/분류:
- AI 워크플로우 설계: 인공지능을 단순한 대화형 인터페이스를 넘어 특정 목적을 위한 작업 흐름에 통합하는 방식.
- 외부 도구 연동: 인공지능 모델이 외부 애플리케이션이나 서비스의 기능을 호출하고 활용하도록 연결하는 방식.
- 마인드셋 전환: 인공지능을 수동적으로 사용하는 것을 넘어 능동적으로 활용 전략을 설계하는 관점의 변화.

핵심 원리: 인공지능 모델이 자체적으로 수행하기 어려운 작업을 외부 도구에 위임하거나, 외부 도구의 데이터를 활용하여 더 정교한 결과를 도출함으로써, 인공지능의 활용 범위를 확장하고 효율성을 극대화한다.
