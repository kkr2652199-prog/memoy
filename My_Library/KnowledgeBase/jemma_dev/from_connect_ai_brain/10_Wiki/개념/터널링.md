# 터널링

## 설명

터널링은(는) 한 네트워크 프로토콜 내에 다른 프로토콜의 데이터를 캡슐화하여 전송하거나, 로컬 서비스를 외부 네트워크에 안전하게 노출하기 위한 가상 통로를 생성하는 기술이다.

유형/분류:
- 로컬 서비스 노출 터널링: 로컬 개발 환경의 웹 서비스나 API를 외부 인터넷에 임시로 공개하여 테스트 및 데모를 가능하게 한다.
- 웹훅 테스트 터널링: 외부 서비스로부터 웹훅을 수신하기 위해 로컬 개발 서버를 인터넷에 연결한다.
- 보안 원격 접근 터널링: 암호화된 채널을 통해 외부에서 내부 네트워크 자원에 안전하게 접근한다.

핵심 원리: 로컬에서 실행 중인 서비스에 대한 요청을 외부 공개 URL을 통해 받아 암호화된 터널을 거쳐 로컬 서비스로 전달한다. 이를 통해 방화벽이나 NAT 뒤에 있는 서비스도 외부에서 접근 가능하게 된다.
