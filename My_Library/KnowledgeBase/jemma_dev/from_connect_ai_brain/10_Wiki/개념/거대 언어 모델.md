# 거대 언어 모델

## 설명

거대 언어 모델은(는) 방대한 양의 텍스트 데이터를 학습하여 인간의 언어를 이해하고 생성하는 능력을 갖춘 인공지능 모델이다.

유형/분류:
- 검색 증강 생성 (RAG): 외부 지식을 모델에 제공하여 환각을 줄이고 답변의 정확성을 높이는 기법.
- 메모리 관리: 모델이 과거 대화 맥락을 기억하고 장기적인 상호작용에서 일관성을 유지하도록 돕는 기법.
- 툴 오케스트레이션: 모델이 외부 도구와 연동하여 실제 세계의 작업을 수행하도록 지원하는 기법.

핵심 원리: 컨텍스트 엔지니어링은 거대 언어 모델의 기억 상실과 환각이라는 단점을 극복하기 위해 외부 지식 제공, 과거 대화 기억, 실제 도구 사용 지원 등을 통해 AI가 스스로 일할 수 있는 환경을 구축하는 것을 핵심 원리로 한다.
