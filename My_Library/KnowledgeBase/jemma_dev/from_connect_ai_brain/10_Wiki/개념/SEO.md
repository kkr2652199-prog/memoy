# SEO

## 설명

SEO은(는) 검색 엔진 결과 페이지에서 웹사이트의 가시성을 높여 유기적인 트래픽의 양과 질을 개선하는 일련의 최적화 과정이다.

유형/분류:
- 온페이지 SEO: 웹 페이지의 콘텐츠, HTML 소스 코드 등을 최적화하여 검색 엔진 순위를 높이는 활동
- 오프페이지 SEO: 웹사이트 외부에서 이루어지는 활동으로, 주로 백링크 구축을 통해 도메인 권위를 높이는 방법
- 기술 SEO: 웹사이트의 기술적 요소를 최적화하여 검색 엔진이 콘텐츠를 더 효율적으로 크롤링하고 색인하도록 돕는 과정
- 로컬 SEO: 특정 지역 기반 검색 결과에서 웹사이트가 더 잘 노출되도록 최적화하는 전략

핵심 원리: 검색 엔진 알고리즘의 작동 방식을 이해하고 사용자의 검색 의도를 파악하여, 고품질의 관련성 높은 콘텐츠를 생성하고 권위 있는 백링크를 구축함으로써 검색 엔진 순위를 향상시킨다.
