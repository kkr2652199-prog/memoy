# ShareGPT 스타일

## 설명

ShareGPT 스타일은(는) 대화형 인공지능 모델의 파인튜닝을 위해 사용자-모델 상호작용을 구조화된 형식으로 표현하는 데이터 표준입니다.

유형/분류:
- 데이터 형식: `.jsonl`과 같이 각 상호작용을 개별 객체로 저장하는 형식
- 활용 목적: 대규모 언어 모델(LLM)의 특정 작업 또는 도메인에 대한 맞춤형 학습
- 콘텐츠 구조: 사용자 질문과 모델 응답을 명확히 구분하여 대화 흐름을 재현
- 플랫폼 연동: [[Hugging Face]]와 같은 모델 허브에 업로드하여 공유 및 활용

핵심 원리: 데이터를 대화 턴(turn) 단위로 표준화하여 모델이 실제 대화 맥락을 학습하고 특정 응답 패턴을 생성하도록 유도합니다.

관련 도구: [[Google Colab]], [[Gemma 4]], [[Hugging Face]], [[LoRA]], [[GGUF]], [[unsloth]]
