# 지능 3단계

## 설명

지능 3단계는 로컬 AI 환경에서 사용자 경험 및 성능 최적화를 위해 설계된 인공지능 작동 모드의 계층적 분류 체계이다.

유형/분류:
- 고속 지능 모드: 키워드 기반 의도 파악, 경량 프롬프트, 제한된 컨텍스트를 활용하여 빠른 응답을 제공하는 모드.
- 실시간 스트리밍 모드: SSE(Server-Sent Events)를 통해 토큰 단위로 답변을 실시간으로 사용자에게 제공하는 모드.
- UI 연동 지능 표시: 로컬 UI 환경에서 사용자의 선택에 따라 AI의 지능 수준 또는 작동 모드를 시각적으로 표시하고 유지하는 기능.

핵심 원리: 지능 3단계는 속도, 컨텍스트 크기, 프롬프트 복잡성 등 다양한 요소를 기반으로 AI의 처리 방식을 동적으로 조절하여 사용자 경험을 최적화한다. 특히 로컬 AI 환경에서 빠른 응답과 실시간 스트리밍을 지원하며, UI를 통해 현재 작동 중인 지능 모드를 시각적으로 표시한다.
