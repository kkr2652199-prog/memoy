# 포터블 실행 파일

## 설명

포터블 실행 파일은(는) 운영 체제에 설치 과정 없이 독립적으로 실행되며, 시스템에 잔여 파일이나 레지스트리 변경을 최소화하는 프로그램 파일이다.

유형/분류:
- 설치 불필요 소프트웨어: 시스템에 설치 과정 없이 즉시 실행 가능한 형태.
- 시스템 무영향 소프트웨어: 실행 후 시스템 레지스트리나 파일 시스템에 영구적인 변경을 거의 남기지 않는 소프트웨어.
- 이동식 미디어 호환: USB 드라이브 등 이동식 저장 장치에 담아 여러 컴퓨터에서 사용 가능한 형태.

핵심 원리: 포터블 실행 파일은 필요한 모든 구성 요소와 라이브러리를 자체적으로 포함하여 외부 의존성을 최소화하고, 실행 환경에 대한 변경을 제한함으로써 시스템의 깔끔함을 유지한다.
