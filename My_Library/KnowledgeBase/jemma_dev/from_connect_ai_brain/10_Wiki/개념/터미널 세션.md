# 터미널 세션

## 설명

터미널 세션은 사용자가 컴퓨터 시스템과 상호작용하기 위해 명령줄 인터페이스(CLI)를 통해 설정하는 대화형 통신 채널이다.

유형/분류:
- 로컬 세션: 사용자의 현재 기기에서 직접 명령을 실행하는 환경.
- 원격 세션: 네트워크를 통해 다른 컴퓨터에 접속하여 명령을 실행하는 환경 (예: SSH).
- 가상 터미널 세션: 단일 물리적 콘솔에서 여러 독립적인 터미널 환경을 제공하는 기능.
- GUI 기반 세션: 그래픽 사용자 인터페이스 내에서 터미널 에뮬레이터를 통해 제공되는 세션.

핵심 원리: 사용자의 텍스트 기반 명령을 시스템에 전달하고, 시스템의 처리 결과를 텍스트 형태로 사용자에게 다시 표시함으로써 양방향 소통을 가능하게 한다.
