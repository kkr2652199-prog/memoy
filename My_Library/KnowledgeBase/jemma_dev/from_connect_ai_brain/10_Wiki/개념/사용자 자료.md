# 사용자 자료

## 설명

사용자 자료은(는) 자료 관리 시스템 내에서 개인적인 메모와 문서와 같이 사용자 개개인의 고유한 정보를 저장하고 관리하는 데 사용되는 데이터 유형이다.

유형/분류:
- 개인 메모: 사용자가 작성한 비공식적인 기록이나 노트.
- 개인 문서: 사용자가 생성하거나 소유한 문서 파일.
- 고유 자료: 특정 사용자에게만 해당되는 독점적인 정보.

핵심 원리: 사용자 자료는 정보 자료와 명확히 구분되어 개인화된 정보 관리를 가능하게 하며, 시스템의 복잡성을 증가시키지 않으면서 개별 사용자의 필요에 맞는 자료를 효율적으로 저장하고 접근할 수 있도록 한다.
