# JS 렌더링

## 설명

JS 렌더링은(는) [[네이버]]나 [[유튜브]]와 같은 동적 웹사이트에서 자바스크립트 실행을 통해 데이터를 추출하기 위해 필요한 웹 페이지 구성 과정이다.

유형/분류:
- 스크래핑 목적 렌더링: 동적으로 생성되는 웹 콘텐츠를 수집하기 위해 자바스크립트 실행을 포함하는 페이지 렌더링.
- 브라우저 에뮬레이션 기반: [[Playwright]]와 같은 도구를 사용하여 실제 브라우저 환경을 모방하여 자바스크립트를 실행하고 페이지를 렌더링하는 방식.

핵심 원리: 동적 웹사이트는 초기 HTML 로드 후 자바스크립트를 실행하여 페이지 내용을 구성하므로, 일반적인 HTTP 요청만으로는 완전한 콘텐츠를 얻을 수 없다. JS 렌더링은 이러한 자바스크립트 코드를 실행하여 최종적으로 브라우저에 표시되는 형태의 데이터를 확보하는 원리이다.

관련 도구: [[httpx]], [[playwright]]
