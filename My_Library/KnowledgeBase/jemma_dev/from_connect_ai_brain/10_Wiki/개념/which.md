# which

## 설명

which은(는) 의문사 또는 관계대명사로 사용되어 선택지를 묻거나 특정 사물이나 정보를 지칭하는 절을 도입하는 영어 단어.

유형/분류:
- 의문사: 질문에서 선택 가능한 여러 항목 중 특정 하나를 지칭할 때 사용.
- 관계대명사: 선행하는 명사(사물, 동물, 개념 등)에 대한 추가 정보를 제공하는 관계절을 이끌 때 사용.
- 한정사: 명사 앞에 위치하여 '어떤' 또는 '어느'와 같은 의미로 특정 선택지를 나타낼 때 사용.

핵심 원리: 문맥에 따라 명사, 대명사, 형용사적 역할을 수행하며, 주로 선택, 식별, 또는 추가 정보 제공의 기능을 통해 문장의 의미를 명확히 한다.
