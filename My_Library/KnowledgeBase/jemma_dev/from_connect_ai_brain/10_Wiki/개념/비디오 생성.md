# 비디오 생성

## 설명

비디오 생성은(는) 텍스트 프롬프트나 이미지를 입력받아 인공지능 모델을 통해 동영상을 자동으로 만들어내는 기술이다.

유형/분류:
- AI 기반 이미지-비디오 변환: 정지 이미지를 동영상 클립으로 변환하는 방식.
- 프롬프트 기반 비디오 생성: 텍스트 설명을 바탕으로 원하는 내용의 동영상을 생성하는 방식.
- 자동화된 대량 비디오 생성: 브라우저 확장 프로그램 등을 활용하여 여러 개의 비디오를 자동으로 생성하고 관리하는 방식.

핵심 원리: 사용자가 이미지를 드래그 앤 드롭하거나 텍스트 프롬프트를 입력하면, 인공지능 모델이 이를 해석하여 비디오를 생성한다. 생성된 비디오는 자동으로 다운로드되고 정리되어 효율적인 워크플로우를 제공한다.
