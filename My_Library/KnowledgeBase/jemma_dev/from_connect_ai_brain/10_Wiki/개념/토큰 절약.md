# 토큰 절약

## 설명

토큰 절약은 인공지능 모델 사용 시 발생하는 비용을 줄이기 위해 입력, 출력, 사고 토큰의 효율적인 관리를 통해 토큰 사용량을 최적화하는 전략이다.

유형/분류:
- 프롬프트 엔지니어링: 정확한 프롬프트 작성과 불필요한 정보 제거(예: 슬래시 클리어, 슬래시 컴팩트)를 통해 입력 토큰을 줄이는 방법.
- 모델 및 기능 활용: 작업에 적합한 모델 선택, 특정 기능(예: 플랜 모드, 클로드 이그노어)을 사용하여 토큰 효율성을 높이는 방법.
- 토큰 구조 분석: 입력, 출력, 사고 토큰의 작동 방식을 이해하여 전반적인 토큰 소비를 예측하고 관리하는 방법.

핵심 원리: 인공지능 모델이 입력, 출력, 사고 토큰을 소비하는 방식을 이해하고, 불필요한 토큰 생성을 최소화하며, 효율적인 프롬프트 작성 및 모델 선택을 통해 전체 토큰 사용량을 줄여 비용을 절감한다.
