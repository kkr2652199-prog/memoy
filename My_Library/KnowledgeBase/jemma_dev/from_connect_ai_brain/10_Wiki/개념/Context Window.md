# Context Window

## 설명

Context Window은(는) 대규모 언어 모델(LLM)이 현재 응답을 생성하기 위해 고려할 수 있는 입력 텍스트(토큰)의 제한된 범위를 의미한다.

유형/분류:
- 고정 컨텍스트: 미리 정해진 최대 토큰 길이 내에서만 정보를 처리하는 방식.
- 슬라이딩 컨텍스트: 대화 진행에 따라 가장 오래된 정보를 제거하고 새로운 정보를 추가하며 윈도우를 이동시키는 방식.
- 확장 컨텍스트: 어텐션 메커니즘 최적화 등을 통해 물리적 토큰 제한을 넘어 더 긴 정보를 효과적으로 활용하는 방식.

핵심 원리: 이 원리는 대규모 언어 모델이 이전 대화 내용이나 입력 정보를 "기억"하여 일관성 있고 맥락에 맞는 응답을 생성할 수 있도록 하며, 대화의 연속성을 유지하는 데 필수적이다.
