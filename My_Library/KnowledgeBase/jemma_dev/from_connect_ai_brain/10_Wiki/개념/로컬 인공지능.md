# 로컬 인공지능

## 설명

로컬 인공지능은(는) 사용자의 개인 컴퓨터에서 직접 실행되며, 외부 클라우드 서비스나 토큰 소진 없이 개인 지식을 기반으로 맞춤형 인공지능 환경을 구축하는 인공지능 구현 방식이다.

유형/분류:
- 개인 맞춤형 AI: 사용자의 고유한 지식과 데이터를 활용하여 '제2의 두뇌'처럼 기능하도록 구축된다.
- 오프라인 실행 AI: 인터넷 연결이나 외부 서버 의존 없이 사용자의 로컬 환경에서 직접 작동한다.
- 비용 효율적 AI: 외부 API 사용에 따른 토큰 소진이나 추가 비용 없이 무제한으로 활용 가능하다.

핵심 원리: 로컬 인공지능은 사용자가 선택한 인공지능 모델을 자신의 컴퓨터에 직접 설치하고 실행하여, 개인 데이터를 활용한 학습 및 추론을 로컬 환경에서 수행한다. 이를 통해 외부 서비스의 제약 없이 맞춤형 인공지능 기능을 제공한다.
