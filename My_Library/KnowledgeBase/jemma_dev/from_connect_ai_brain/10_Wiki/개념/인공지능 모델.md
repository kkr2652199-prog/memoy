# 인공지능 모델

## 설명

인공지능 모델은(는) 특정 작업을 수행하도록 대량의 데이터를 학습하여 패턴을 인식하고 의사결정 능력을 갖춘 계산 구조이다.

유형/분류:
- 로컬 AI 모델: 사용자의 개인 컴퓨터에서 직접 실행되어 데이터 주권과 토큰 비용 절감의 이점을 제공하는 모델.
- 클라우드 기반 AI 모델: 원격 서버에서 호스팅되어 강력한 컴퓨팅 자원과 확장성을 활용하는 모델.
- 사전 학습 모델: 대규모 일반 데이터셋으로 미리 학습되어 특정 작업에 미세 조정될 수 있는 모델.
- 맞춤형 AI 모델: 특정 사용자의 데이터나 요구사항에 맞춰 추가 학습되거나 커스터마이징된 모델.

핵심 원리: 인공지능 모델은 입력 데이터를 기반으로 학습된 알고리즘을 통해 예측, 분류, 생성 등의 작업을 수행하며, 이 과정에서 모델의 내부 파라미터가 최적화되어 문제 해결 능력을 향상시킨다.
