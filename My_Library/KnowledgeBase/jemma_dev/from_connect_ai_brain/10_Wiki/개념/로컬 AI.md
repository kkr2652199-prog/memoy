# 로컬 AI

## 설명

로컬 AI은(는) 사용자 기기에서 직접 실행되며, 실시간 스트리밍 응답과 낮은 지연 시간을 위해 최적화된 인공지능 시스템이다.

유형/분류:
- 스트리밍 응답 최적화: 토큰 단위의 [[SSE]]를 활용하여 답변을 실시간으로 제공하는 사용자 경험에 중점을 둔다.
- 성능 가속화: [[quick_intent]]와 같은 키워드 기반 의도 파악, 경량 프롬프트, 제한된 컨텍스트를 사용하여 처리 지연을 최소화한다.
- UI 통합 및 사용자 설정: [[LM Studio]]와 같은 로컬 환경 UI에서 특정 지능 관련 기능을 표시하고, 사용자 선택을 유지하는 기능을 포함한다.
- 작업 형식 처리: 빠른 경로에서 특정 작업 형식 오버라이드를 통해 의도 파악에 반영하여 효율적인 처리를 가능하게 한다.

핵심 원리: 로컬 AI는 사용자 기기 내에서 빠른 응답과 실시간 상호작용을 제공하기 위해 경량 프롬프트와 제한된 컨텍스트를 활용하여 지연 시간을 줄이는 데 중점을 둔다. 이는 특히 스트리밍 기반의 대화형 사용자 경험을 강화한다.

관련 도구: [[quick_intent]], [[LM Studio]], [[SSE]]
