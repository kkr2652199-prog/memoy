# 강화 학습

## 설명

강화 학습은(는) 에이전트가 특정 환경에서 행동을 수행하고 그 결과로 보상을 받으며 최적의 정책을 학습하는 기계 학습 패러다임이다.

유형/분류:
- 모델 기반 강화 학습: 환경의 역학을 모델링하여 최적의 행동을 계획하는 방식.
- 모델 프리 강화 학습: 환경 모델 없이 시행착오를 통해 직접 정책을 학습하는 방식.
- 가치 기반 강화 학습: 각 상태 또는 상태-행동 쌍의 가치를 추정하여 최적의 행동을 선택하는 방식.
- 정책 기반 강화 학습: 행동을 직접 선택하는 정책 함수를 최적화하는 방식.

핵심 원리: 에이전트는 환경과 상호작용하며 현재 상태에서 어떤 행동을 취할지 결정하고, 그 행동의 결과로 주어지는 보상을 통해 학습하여 장기적인 누적 보상을 최대화하는 전략을 찾아낸다. 이는 시행착오를 통해 최적의 행동 정책을 점진적으로 개선하는 과정이다.
