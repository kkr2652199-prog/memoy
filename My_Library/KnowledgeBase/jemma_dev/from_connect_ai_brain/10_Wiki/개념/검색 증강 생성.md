# 검색 증강 생성

## 설명

검색 증강 생성은 거대 언어 모델의 환각 및 기억 상실 문제를 극복하고 정확성을 높이기 위해 외부 지식을 검색하여 답변 생성에 활용하는 기술이다.

유형/분류:
- 외부 지식 통합: 거대 언어 모델이 학습 데이터에 없는 최신 정보나 특정 도메인 지식을 활용하여 답변을 생성하도록 지원.
- 환각 감소: 부정확하거나 존재하지 않는 정보를 생성하는 거대 언어 모델의 경향을 줄여 답변의 신뢰도를 높임.
- 정보 최신성 유지: 실시간 또는 주기적으로 업데이트되는 외부 데이터를 활용하여 거대 언어 모델의 정보가 항상 최신 상태를 유지하도록 함.

핵심 원리: 거대 언어 모델이 답변을 생성하기 전에 관련 외부 문서를 검색하여 최신 정보나 특정 도메인 지식을 확보하고, 이를 기반으로 더 정확하고 신뢰할 수 있는 응답을 생성한다. 이는 모델의 환각을 줄이고 기억 상실 문제를 보완한다.
