# 자동 자막 생성

## 설명

자동 자막 생성은 음성 인식 기술을 활용하여 영상 또는 오디오 콘텐츠의 대화를 텍스트로 변환하고, 이를 시간 정보와 동기화하여 자막 형태로 자동으로 만들어내는 기술이다.

유형/분류:
- 실시간 자막 생성: 라이브 스트리밍이나 실시간 회의에서 음성을 즉시 텍스트 자막으로 변환하는 방식
- 사전 녹화 자막 생성: 이미 녹화된 영상이나 오디오 파일의 음성을 분석하여 자막을 생성하는 방식
- 언어별 자막 생성: 단일 언어 음성을 해당 언어 자막으로 변환하거나, 다국어 번역을 포함하여 다른 언어 자막을 생성하는 방식

핵심 원리: 음성 신호를 디지털화한 후, [[음성 인식]] (ASR) 모델이 음성 패턴을 분석하여 해당 언어의 단어와 문장으로 변환한다. 이 텍스트는 영상의 타임코드와 동기화되어 화면에 표시될 자막 파일로 출력된다.
