# RAG

## 설명

RAG은(는) 대규모 언어 모델(LLM)이 외부 지식 기반에서 관련 정보를 검색하여 답변의 정확성과 신뢰성을 높이는 인공지능 기술이다.

유형/분류:
- 로컬 RAG 시스템: 사용자의 개인 지식 기반에서 정보를 검색하여 LLM의 생성을 돕는 시스템.
- 지식 관리 RAG: 개인 또는 조직의 방대한 비정형 데이터를 체계화하고 활용하여 LLM 기반의 지식 검색 및 생성을 지원하는 시스템.
- 제2의 뇌 시스템: 개인의 모든 디지털 정보를 수집, 정리, 연결하여 LLM을 통해 지능적으로 활용하는 시스템.

핵심 원리: RAG 시스템은 사용자가 저장한 다양한 원본 데이터를 검색하고, 이를 기반으로 LLM이 새로운 정보를 생성하거나 기존 지식을 구조화하여 답변의 정확성과 깊이를 향상시킨다. LLM은 원본 데이터를 직접 수정하지 않고 참조한다.
