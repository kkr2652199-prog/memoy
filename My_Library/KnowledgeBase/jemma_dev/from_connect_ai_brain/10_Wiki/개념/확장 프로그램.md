# 확장 프로그램

## 설명

확장 프로그램은(는) 웹 브라우저나 다른 응용 프로그램에 새로운 기능이나 개선된 사용자 경험을 추가하는 소프트웨어 구성 요소이다.

유형/분류:
- 웹 브라우저 확장: 웹 브라우저의 기능을 보완하거나 새로운 기능을 추가하여 사용자 편의성을 높이는 소프트웨어.
- 콘텐츠 자동화 도구: 텍스트 스크립트와 같은 입력을 기반으로 이미지 등 특정 유형의 콘텐츠 생성을 자동화하는 프로그램.
- 생산성 향상 도구: 반복적인 작업을 자동화하거나 특정 기능을 간소화하여 사용자의 작업 효율을 높이는 프로그램.

핵심 원리: 확장 프로그램은 호스트 응용 프로그램의 인터페이스와 API를 활용하여 특정 기능을 추가하며, 사용자의 입력(예: 텍스트 대본)을 처리하여 원하는 결과물(예: 이미지)을 자동으로 생성한다. 이를 통해 기존 응용 프로그램의 한계를 넘어선 맞춤형 작업 흐름을 제공한다.
