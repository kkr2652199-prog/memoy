# LoRA

## 설명

LoRA(Low-Rank Adaptation)은(는) 대규모 사전 학습 모델의 파인튜닝 효율성을 높이기 위해 저랭크 행렬 어댑터를 추가하여 학습 가능한 파라미터 수를 줄이는 기법이다.

유형/분류:
- 모델 파인튜닝: 사전 학습된 대규모 언어 모델을 특정 작업이나 데이터셋에 맞게 효율적으로 조정하는 데 사용된다.
- 파라미터 효율적 학습 (PEFT): 전체 모델 가중치를 업데이트하는 대신, 적은 수의 추가 파라미터만 학습시켜 컴퓨팅 자원과 시간을 절약한다.
- 어댑터 기반 학습: 기존 모델의 가중치를 고정하고, 그 위에 작은 저랭크 어댑터 모듈을 삽입하여 학습을 진행한다.

핵심 원리: LoRA는 사전 학습된 모델의 각 트랜스포머 레이어에 두 개의 작은 행렬(저랭크 어댑터)을 병렬로 추가하고, 이 어댑터의 가중치만 학습시켜 전체 모델의 파라미터 업데이트 부담을 최소화한다. 이를 통해 모델의 성능 저하 없이 파인튜닝 과정을 가속화하고 메모리 사용량을 줄일 수 있다.
