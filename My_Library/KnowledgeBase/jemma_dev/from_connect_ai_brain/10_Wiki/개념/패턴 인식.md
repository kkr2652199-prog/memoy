# 패턴 인식

## 설명

패턴 인식은 데이터에서 반복되는 경향이나 규칙을 식별하는 과정을 의미한다. 이번 영상에서는 로또 1226회에 대한 여러 유튜브 채널의 예상 번호를 비교하고 분석하였다. 지난주 1225회의 결과를 바탕으로 각 채널의 선택 결과를 살펴보았으며, F 채널과 C 채널의 상승세가 눈에 띄었다. G 채널은 제외되었고, 이번 주에는 네 개의 채널과 AI의 예측 결과를 종합하여 분석하였다. 특히, 아무도 선택하지 않은 번호와 장기 미출연 번호를 주목하였다. 새 자료에서는 로또 1195회차 당첨번호를 분석하고, 과거 40회차 데이터를 바탕으로 패턴과 규칙을 찾아내어 1196회차 예상 번호를 도출하였다.
