# AI 정원사

## 설명

AI 정원사는 사용자의 아이디어와 정보를 체계적으로 정리하고 관리하여 개인의 지식 네트워크를 확장하는 LLM 기반 지식 관리 시스템입니다.

유형/분류:
- 개인 지식 관리 (PKM): 사용자의 정보를 체계적으로 정리하고 관리하는 데 중점을 둡니다.
- LLM 기반 도구: 대규모 언어 모델의 능력을 활용하여 정보를 처리하고 위키 형태로 가꿉니다.
- 정보 큐레이션: 수집된 로우 데이터를 정리하고 의미 있는 지식으로 변환하는 역할을 수행합니다.
- 지식 네트워크 확장: 개인의 전문성과 경험을 바탕으로 새로운 연결고리를 발견하고 지식의 폭을 넓힙니다.

핵심 원리: 사용자가 수집한 정보를 로우 폴더에 저장하면, AI가 이를 정리하여 위키 형태로 가꿉니다. 사용자는 '수집', '질문', '정리' 명령어를 통해 AI의 도움을 받아 지식을 관리합니다.
