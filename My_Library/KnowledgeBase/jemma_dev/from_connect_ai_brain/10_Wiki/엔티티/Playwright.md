# Playwright

## 개요

Playwright은(는) 동적 웹사이트의 JavaScript 렌더링을 통해 웹 스크래핑 효율을 높이는 브라우저 자동화 프레임워크.

핵심 특징:
- JavaScript 렌더링 지원: 네이버와 같은 동적 웹사이트에서 JavaScript 기반 콘텐츠를 정확하게 렌더링하여 본문 추출 실패 문제를 해결합니다.
- 동적 데이터 추출 강화: 기존 requests.get 방식으로는 어려운 유튜브 채널 URL 처리 등 복잡한 동적 데이터를 효과적으로 추출할 수 있도록 돕습니다.
- 웹 스크래핑 시스템 개선: 기존 웹 스크래핑 시스템의 한계를 극복하고, 데이터 추출 성공률을 높이는 핵심적인 개선 방안으로 제안됩니다.
