# RAG

## 개요

RAG은(는) 대규모 언어 모델(LLM)이 외부 지식 기반에서 관련 정보를 검색하여 보다 정확하고 신뢰할 수 있는 응답을 생성하도록 돕는 인공지능 기술입니다.

핵심 특징:
- 정보 검색 기반 응답 생성: LLM이 학습하지 않은 최신 또는 특정 도메인 지식을 외부에서 검색하여 답변의 정확성과 관련성을 높입니다.
- 환각 현상 감소: 검증된 외부 데이터를 활용함으로써 LLM의 잘못된 정보 생성(환각)을 줄이고 답변의 신뢰도를 향상시킵니다.
- 지식 관리 시스템(KMS) 통합: 도서관과 같은 방대한 지식 구조 내의 데이터를 LLM이 효과적으로 탐색하고 분석하여 활용할 수 있도록 지원합니다.
- 도메인 특화 질의응답 개선: 특정 분야(예: 도서관 카테고리, 앱 구조)에 대한 심층적인 분석 및 가이드라인 제공을 가능하게 합니다.
