# Microsoft Edge

## 개요

Microsoft Edge은(는) Microsoft에서 개발한 크로미움 기반의 웹 브라우저.

핵심 특징:
- 크로미움 기반: 웹 표준 호환성과 성능을 제공하며, 다양한 웹 기술을 지원한다.
- 확장 프로그램 생태계: 사용자가 기능을 추가하고 맞춤 설정할 수 있도록 다양한 브라우저 확장 프로그램을 지원한다.
- AI 도구 플랫폼: 2026년에는 이미지 드래그 앤 드롭 및 프롬프트 입력을 통해 AI 비디오를 자동으로 생성하는 [[Auto Hunyuan]] 확장 프로그램을 지원할 예정이다.
