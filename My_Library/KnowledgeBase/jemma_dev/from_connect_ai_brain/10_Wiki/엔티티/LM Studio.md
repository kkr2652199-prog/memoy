# LM Studio

## 개요

LM Studio은(는) 사용자가 자신의 컴퓨터에 대규모 언어 모델(LLM)을 설치하고 관리할 수 있도록 돕는 로컬 LLM 관리 도구.

핵심 특징:
- 로컬 LLM 관리: 클라우드 기반이 아닌 사용자 로컬 환경에서 LLM을 직접 구축하고 관리하는 기능을 제공.
- 오픈소스 LLM 설치 지원: 젬마 4 E2B와 같은 다양한 오픈소스 LLM을 사용자의 컴퓨터에 쉽게 설치할 수 있도록 지원.
- AI 에이전트 구축 활용: '더 컨스트럭트' 플랫폼에서 맞춤형 지식을 학습시킨 차별화된 AI 에이전트를 만드는 데 핵심 도구로 사용.
