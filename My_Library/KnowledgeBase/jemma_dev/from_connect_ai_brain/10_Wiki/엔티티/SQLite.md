# SQLite

## 개요

SQLite은(는) 개인 지식 도서관 애플리케이션 'My_Library'의 핵심 데이터 저장소로 사용되며, 파일 기반으로 작동하는 경량 관계형 데이터베이스 관리 시스템.

핵심 특징:
- 개인 지식 도서관의 데이터 저장소: 'My_Library' 프로젝트에서 172개의 자료와 관련 메타데이터를 저장하는 데 활용됩니다.
- 자료 임베딩 저장: `nomic-embed-text-v2-moe` 모델로 생성된 모든 자료의 임베딩을 위한 데이터베이스 테이블을 구축하고 관리합니다.
- 하이브리드 검색 지원: BM25+벡터 하이브리드 검색 기능 구현을 위한 기반 데이터를 저장하여 검색 효율성을 높입니다.
- 신규 자료 자동 임베딩 기능 개발: 신규 자료 섭취 시 자동으로 임베딩을 생성하고 저장하는 기능 개발의 핵심 구성 요소입니다.

관련: [[FastAPI]], [[Ollama]], [[D3.js]]
