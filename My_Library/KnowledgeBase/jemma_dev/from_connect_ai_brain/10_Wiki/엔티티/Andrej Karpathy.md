# Andrej Karpathy

## 개요

Andrej Karpathy은(는) 인공지능 분야의 저명한 연구자로, 특히 대규모 언어 모델(LLM)을 활용한 지식 관리 및 '제2의 뇌' 구축 방법론에 영감을 준 인물이다.

핵심 특징:
- '제2의 뇌' 구축 방법론: Andrej Karpathy의 이름을 딴 이 방법론은 옵시디언과 LLM(Claude Code)을 활용하여 개인 지식 관리 시스템을 구축하는 데 중점을 둡니다.
- LLM 기반 지식 자동화: Claude Code와 같은 대규모 언어 모델을 사용하여 사용자의 원본 자료(기사, PDF 등)를 분석하고 자동으로 위키 페이지를 생성하는 시스템을 제안합니다.
- 로컬 RAG 시스템: 사용자의 로컬 환경에서 검색 증강 생성(RAG) 원리를 적용하여, LLM이 원본 파일을 직접 수정하지 않고 참조하며 지식을 구조화합니다.
- 옵시디언 활용: 옵시디언의 'raw' 폴더에 자료를 저장하고, 태그 및 연결 기능을 활용하여 지식 구조를 효과적으로 구축하는 방식을 포함합니다.
