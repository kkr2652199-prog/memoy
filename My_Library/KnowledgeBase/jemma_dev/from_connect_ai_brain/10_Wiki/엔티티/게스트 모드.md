# 게스트 모드

## 개요

게스트 모드은(는) 공용 컴퓨터 사용 환경에서 개인 정보 보호를 강화하기 위해 제공되는 임시 로그인 기능.

핵심 특징:
- 개인 정보 보호 강화: 공용 컴퓨터 사용 시 사용자의 로그인 정보나 활동 기록이 기기에 저장되지 않도록 보호합니다.
- 공용 컴퓨터 최적화: 여러 사용자가 공유하는 환경에서 개인 정보 유출 위험을 최소화하도록 설계되었습니다.
- 임시 세션 관리: 세션 종료 시 모든 브라우징 데이터 및 로그인 정보가 자동으로 삭제되어 흔적을 남기지 않습니다.
