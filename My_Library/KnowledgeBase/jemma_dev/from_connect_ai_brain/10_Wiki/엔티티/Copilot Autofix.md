# Copilot Autofix

## 개요

Copilot Autofix는 GitHub 플랫폼에서 코드 수정 과정을 가속화하고 평균 문제 해결 시간(MTTR)을 70% 단축시키는 AI 기반 도구입니다.

핵심 특징:
- MTTR 단축: 평균 문제 해결 시간(MTTR)을 70% 단축시켜 개발 효율성을 크게 향상시킵니다.
- 코드 수정 가속화: AI 기반으로 코드 작성, 테스트, 수정 과정 중 특히 수정 단계를 가속화합니다.
- GitHub 통합: 개발자 협업 플랫폼인 GitHub 내에서 작동하여 개발 프로세스를 지원합니다.
