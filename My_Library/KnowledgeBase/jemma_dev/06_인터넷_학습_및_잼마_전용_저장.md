# 인터넷으로 공부하기 + **잼마 전용 뇌**에 저장 (필수)

## 1) 교본이 “웹”일 때 — 어디를 보나 (시작용)

| 목표 | 공식·신뢰도 높은 쪽 (영문이어도 번역기 병행) |
|------|---------------------------------------------|
| Python 기초 | <https://docs.python.org/3/tutorial/> (튜토리얼 앞 몇 장) |
| HTML/CSS/JS (앱 UI) | MDN: <https://developer.mozilla.org/ko/> |
| FastAPI (우리 백엔드) | <https://fastapi.tiangolo.com/ko/> (한국어 있음) |
| SQL/SQLite 개념 | SQLite 문서 요약 + SQL “기초 쿼리”만 |
| Git (나중에) | <https://git-scm.com/doc> — **처음엔** `add` / `commit` / `status` 정도만 |

**방법:**  
- **한 세션 = 한 주제** (예: “오늘은 변수만”).  
- 읽은 뒤 **끄적이지 말고** 아래 **잼마 전용 뇌**에 **한 번만** 정리한다.

---

## 2) **잼마 전용 뇌**란? (비유 → 실제 파일)

- **진짜 “뇌”가 아니라**, 팀이 정한 **잼마만의 기억 창고**다.  
- **위치(고정):**  
  `D:\MONEY lol\My_Library\KnowledgeBase\jemma_dev\jemma_learned_log.md`  
- 인터넷·책·강의에서 **새로 습득한 것**은 **꼭** 이 파일에 **짧게** 남긴다. (긴 복붙 X, **요약** O)

### 기록 포맷 (한 블록 = 한 번 배움)

```markdown
### YYYY-MM-DD
- **출처(링크 제목 + URL):** 
- **한 줄 요약:** 
- **우리 앱에 쓰면:** (없으면 "아직 없음")
- **다음에 볼 키워드:** 
```

- **잼마(LM Studio 대화)**에서 나온 좋은 정리도, **같은 파일**에 **같은 포맷**으로 옮긴다. (대화는 날아가고, **파일은 남는다**)

### (선택) 주제·인덱스로 나누기

- **한 줄 뇌의 기준**은 위 `jemma_learned_log.md`가 **그대로**다.  
- 그걸 **주제별**로 덧쌓거나, **우리 자료 경로**만 정리하려면 `My_Library\jemma\` — `learned/`, `topics/`, `bridge_to_library/` (앱 `Wiki/`, `data/` **원문 복제·DB 직접 수정 금지**). 루트 설명: `My_Library\jemma\README.md`

## 3) 반드시 지킬 말 (잼마 호출 시 복붙)

아래를 **잼마 세션 맨 위**에 덧붙인다.

```text
인터넷/강의로 배운 뒤에는 KnowledgeBase\jemma_dev\jemma_learned_log.md 에
날짜·출처·한 줄 요약을 반드시 남겨라. 이것이 팀의 '잼마 전용 뇌'다.
```

---

## 4) 부사수(Cursor)와 역할

- **잼마:** 개념 설명·계획·기록 **초안** + `jemma_learned_log`에 **쓰기 제안**.  
- **Cursor:** 레포·경로·검증 명령. **app/data 직접 뇌에 넣는 척 하지 말고**, 방법론은 `jemma_dev`에 둔다.
