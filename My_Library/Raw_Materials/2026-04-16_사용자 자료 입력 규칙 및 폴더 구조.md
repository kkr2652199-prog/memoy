[가이드] 사용자 자료 입력 규칙 & 폴더 구조
1. 추천 폴더/소분류 구조

■ 내_지식 (직접 만든 프로세스/생각)

워크플로우: 이미지 제작 7단계, 영상 편집 루틴

대본_규칙: 유튜브 대본 구조, 썸네일 카피 공식

Gemma_지침: 전용 프라이빗 지침, 질문 템플릿

아이디어_브레인: 갑자기 떠오른 아이디어, 메모

■ 수집_꿀팁 (외부에서 배운 정보)

Grok_꿀팁: Grok 사용법, 실전 프롬프트 모음

Claude_꿀팁: Claude Code 사용법, 벤치마킹 정리

AI_도구_정리: Ollama, 설정법 등 실전 노트

■ 정제_노트 (배운 내용을 내 식으로 요약)

경제사냥꾼_정리: 영상 핵심 내용 내 언어로 재정리

트랜드_정리: 최신 트렌드 분석 + 내 사업 적용점

2. 실전 데이터 입력 4대 원칙

① 타입 설정: material_type = "user" 선택
② 분류 입력: 위 리스트의 대분류/중분류를 직접 입력
③ 제목 형식: [분류] 명확한 제목
(예: [워크플로우] 이미지 제작 7단계 체크리스트)
④ 내용 작성: 마크다운(불렛 포인트 등) 활용 시 AI가 더 잘 읽음
