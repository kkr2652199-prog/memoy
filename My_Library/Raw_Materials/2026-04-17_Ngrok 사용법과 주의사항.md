========================================================
🌐 Ngrok 빠른 사용법 (이미 설치 완료!)
========================================================

✅ 현재 상태:
- Ngrok 설치 완료 (버전 3.24.0)
- 인증 토큰 설정 완료
- 서버 실행 중 (포트 5001)

========================================================
🚀 사용 방법 (2단계)
========================================================

1단계: 서버 실행 확인
--------------------
브라우저에서 http://localhost:5001 접속
→ 정상적으로 페이지가 보이면 OK!

2단계: Ngrok 터널 시작
--------------------
방법 A: 배치 파일 실행 (추천)
  → start_ngrok_tunnel.bat 더블클릭

방법 B: 직접 명령어
  → 터미널에서: ngrok http 5001

========================================================
📋 결과 확인
========================================================

Ngrok 실행 후 다음과 같은 화면이 나타납니다:

Session Status                online
Account                       Your Name (Plan: Free)
Version                       3.24.0
Region                        Asia Pacific (ap)
Latency                       45ms
Web Interface                 http://127.0.0.1:4040
Forwarding                    https://abc123.ngrok-free.app -> http://localhost:5001

⭐ 중요: "Forwarding" 줄의 URL이 공개 URL입니다!
   예: https://abc123.ngrok-free.app

========================================================
💡 실제 사용 예시
========================================================

1. 모바일에서 테스트
   → 스마트폰 브라우저에서 공개 URL 접속

2. 친구에게 데모 보여주기
   → 공개 URL을 카카오톡/이메일로 전송

3. 웹훅 테스트
   → 공개 URL을 웹훅 설정에 입력

========================================================
🔧 유용한 명령어
========================================================

Ngrok 웹 인터페이스 열기:
  → 브라우저에서 http://127.0.0.1:4040 접속
  → 요청 로그, 통계 등 확인 가능

Ngrok 중지:
  → 터미널에서 Ctrl+C

========================================================
⚠️ 보안 주의사항
========================================================

1. 임시 테스트용으로만 사용
2. URL을 신뢰할 수 있는 사람에게만 공유
3. 사용 후 반드시 Ngrok 세션 종료 (Ctrl+C)
