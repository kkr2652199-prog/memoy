# 📖 개인 지식 도서관 (My Library)

AI 사서가 관리하는 개인 지식 관리 시스템입니다.

## 개요

자료를 넣으면 AI가 자동으로 읽고, 분류하고, 요약하고, 기존 지식과 연결해줍니다.
모든 지식은 대분류 → 중분류 → 소분류 체계로 정리되며, 교차 참조를 통해 지식 간 관계를 파악할 수 있습니다.

## 구조

| 경로 | 설명 |
|------|------|
| `Raw_Materials/` | 원본 자료 보관소. AI가 읽기만 하며 절대 수정하지 않습니다. |
| `Wiki/` | AI가 정리한 지식 페이지들. 대분류별 하위 폴더로 구성됩니다. |
| `index.md` | 전체 목차. AI가 자동으로 관리합니다. |
| `log.md` | 모든 작업 기록이 날짜순으로 남습니다. |
| `.cursorrules` | AI 사서의 행동 규칙이 정의된 파일입니다. |

## 사용법

1. **자료 섭취**: `Raw_Materials/`에 자료를 넣고 "정리해" 또는 "섭취해"라고 말하세요.
2. **검색**: 궁금한 것을 질문하면 AI가 위키를 검색해 출처와 함께 답변합니다.
3. **건강 상태 확인**: "건강 상태 확인"이라고 하면 고아 페이지, 모순, 누락된 참조를 점검합니다.
