# BM25 알고리즘

## 설명

BM25 알고리즘은 정보 검색에서 문서의 중요도를 평가하는 데 사용되는 통계적 모델로, 주로 검색 엔진에서 쿼리와 문서 간의 관련성을 측정하는 데 활용됩니다. 이 알고리즘은 단어 빈도, 문서 길이, 그리고 전체 문서 집합에서의 단어 분포를 고려하여 점수를 산출합니다. 최근에는 마인드볼트라는 오픈소스 도구가 등장하여, 안드레이 카파시의 LLM 위키 패턴을 기반으로 AI가 프로젝트 자료를 자동으로 분석하고 위키 형태로 정리하는 기능을 제공합니다. 이 도구는 검색, 그래프, 위키의 세 가지 레이어로 구성되어 있으며, 기존의 AI 도구와 통합하여 사용자의 작업 효율성을 극대화합니다. 특히, 마인드볼트는 토큰 소비를 약 60배 절감할 수 있는 기능을 갖추고 있습니다.
