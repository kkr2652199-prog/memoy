# Retrieval Augmented Generation

## 설명

Retrieval Augmented Generation (RAG)은 [[LLM]]의 한계를 극복하기 위해 외부 지식 소스를 활용하여 답변의 정확성과 신뢰성을 높이는 기술입니다. 최근 Andrej Karpathy의 [[LM-Milky]] 아이디어를 활용한 개인 AI 지식 위키 구축 방식은 RAG의 한 형태로 볼 수 있습니다. 이 방식은 AI가 직접 위키를 생성하고 유지보수하여 LLM의 휘발성 기억 문제를 해결하며, 이는 외부 지식 기반의 정보 검색 및 생성을 통해 이루어집니다. [[Personal Knowledge Management]] 시스템에서도 이러한 접근 방식은 AI의 지식 활용 능력을 극대화하는 데 기여합니다.
