# MOC

## 설명

[[MOC]](Maps of Content)는 여러 [[Atomic Notes]]를 특정 주제나 프로젝트 중심으로 묶어 구조화하는 상위 개념의 지식 지도입니다. 이는 복잡한 지식 영역을 탐색하고 이해하는 데 도움을 주는 효과적인 [[지식 구조 설계]] 기법입니다. MOC는 [[상호 참조성]]을 극대화하여 지식 간의 관계를 시각화하고, [[지식 관리 시스템]] 내에서 사용자가 원하는 정보를 빠르게 찾을 수 있도록 합니다. 2026년 4월 19일 자료에서 제시된 '도서관(지식 구조)'의 체계적인 관리에 유용합니다.
