# 확장 프로그램

## 설명

확장 프로그램은 웹 브라우저의 기능을 확장하거나 개선하기 위해 설계된 소프트웨어 모듈입니다. 최근 [[구글 플로우]](Nano Banana 2) 자동 생성 크롬 확장 프로그램이 출시되어 사용자가 쉽게 콘텐츠를 제작할 수 있도록 돕는 도구로 주목받고 있습니다. 이 프로그램은 텍스트 파일에서 대본을 읽어와 자동으로 이미지를 생성하는 방식으로 사용자의 편리함을 극대화합니다. 한편, [[오토플로우]]는 [[구글 플로우]] 기반의 [[AI 영상 자동화]]를 위한 일괄 처리 [[RPA 툴]]로, 직접적인 확장 프로그램은 아니지만 유사한 자동화 기능을 제공합니다.
