# 콘텐츠 기획 방식

## 설명

콘텐츠 기획 방식은 AI 사서 잼마가 사용자님의 자료를 기반으로 고품질의 대본을 생성하기 위한 전략적인 접근법을 의미합니다. 잼마는 폴더별로 세밀하고 일관된 규칙을 적용하여 자료를 [[구조화]]함으로써 효과적인 [[콘텐츠 기획 방식]]을 지원합니다. 이러한 방식은 사용자 스타일의 대본을 효율적으로 제작하며, [[스토리텔링 패턴]]을 활용하여 매력적인 콘텐츠를 기획하는 데 필수적입니다.
