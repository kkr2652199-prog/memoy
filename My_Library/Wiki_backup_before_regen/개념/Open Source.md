# Open Source

## 설명

오픈소스는 소프트웨어의 소스 코드가 공개되어 누구나 자유롭게 사용, 수정, 배포할 수 있도록 허용하는 개발 모델입니다. 최근 AI 에이전트 관리 도구인 Codex 앱의 경우, 그 [[CLI]] 부분이 [[Open Source]]로 제공되어 큰 장점으로 부각되고 있습니다. 이는 사용자들이 도구를 더욱 유연하게 활용하고 커뮤니티 기여를 통해 발전시킬 수 있는 기반을 마련합니다. [[Agentic Coding]]과 같은 분야에서 오픈소스는 투명성과 협업을 촉진하는 중요한 요소입니다.
