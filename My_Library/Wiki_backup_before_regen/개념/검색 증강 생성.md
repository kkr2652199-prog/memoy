# 검색 증강 생성

## 설명

검색 증강 생성(RAG)은 [[컨텍스트 엔지니어링]]의 핵심 기술로, [[거대 언어 모델]](LLM)의 정보 부족이나 환각 문제를 해결하기 위해 외부 지식 기반에서 관련 정보를 검색하여 답변 생성에 활용합니다. 이를 통해 LLM은 최신 정보에 접근하고 더 정확하며 신뢰할 수 있는 응답을 제공하여 AI의 성능과 유용성을 크게 향상시킵니다. RAG의 효과는 '지식 구조 설계' 및 '지식 관리 시스템(KMS)'의 품질에 크게 좌우되며, 이는 외부 지식 기반의 체계적인 구축과 관리가 중요함을 시사합니다.
