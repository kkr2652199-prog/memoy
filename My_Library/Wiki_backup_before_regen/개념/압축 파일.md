# 압축 파일

## 설명

압축 파일은 여러 개의 파일이나 폴더를 하나의 파일로 묶어 저장하는 형식으로, 파일 크기를 줄이고 관리하기 쉽게 만들어줍니다. 일반적으로 ZIP, RAR 등의 형식이 있으며, 압축 해제 프로그램을 통해 원래의 파일로 복원할 수 있습니다. 이를 통해 저장 공간을 절약하고 파일 전송을 간편하게 할 수 있습니다. 또한, 컴퓨터의 저장 용량을 늘리기 위해 디스크 정리와 같은 방법을 활용할 수 있으며, 윈도우 파일 압축 기능을 통해 추가적인 공간 절약이 가능합니다.
