# 스토리텔링 프레임워크

## 설명

[[스토리텔링 프레임워크]]는 독자나 시청자의 흥미를 유지하며 메시지를 체계적으로 전달하기 위한 구조화된 접근 방식입니다. 이는 정보를 단순히 나열하는 것을 넘어, 매력적인 '흐름'을 만들어내어 콘텐츠의 몰입도를 극대화하는 데 사용됩니다. 다양한 [[서사 구조]]와 기법들을 포함하며, [[콘텐츠 기획]] 과정에서 효과적인 메시지 전달 전략을 수립하는 데 필수적인 도구입니다. 이 프레임워크는 복잡한 아이디어를 쉽게 이해시키고 감정적인 연결을 형성하는 데 기여합니다.
