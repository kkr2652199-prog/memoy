# TTS

## 설명

TTS(텍스트 음성 변환)는 텍스트를 자연스러운 음성으로 변환하는 기술로, 다양한 응용 분야에서 사용되고 있습니다. AI 크래프터의 영상 제작 기법을 통해 구글의 최신 도구인 VEO 3.1을 활용하여 고퀄리티 영상을 무료로 제작하는 방법이 소개되었습니다. 이 과정에서 JSON 기법을 이용하여 영상의 일관성을 유지하고, 다양한 API를 활용하여 비용을 최소화하는 전략이 제시됩니다.
