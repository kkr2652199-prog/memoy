# LLM

## 설명

LLM(대규모 언어 모델)은 자연어 처리(NLP) 분야에서 사용되는 강력한 AI 모델로, 대량의 텍스트 데이터를 학습하여 인간과 유사한 방식으로 언어를 이해하고 생성하는 능력을 갖추고 있습니다. 최근에는 안드레이 카파시의 LLM 위키 패턴을 기반으로 한 오픈소스 도구인 마인드볼트가 주목받고 있으며, 이 도구는 AI가 프로젝트 자료를 자동으로 분석하고 위키 형태로 정리하는 기능을 제공합니다. 또한, Obsidian과 Claude Code를 활용하여 개인 지식 관리 시스템을 설정하는 방법도 소개되고 있으며, 사용자는 원시 정보를 입력하고 Claude Code가 이를 분석하여 위키 페이지를 생성하도록 할 수 있습니다.
