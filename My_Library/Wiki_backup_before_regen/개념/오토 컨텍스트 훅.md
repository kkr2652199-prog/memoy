# 오토 컨텍스트 훅

## 설명

오토 컨텍스트 훅은 AI 코딩 도구가 세션이 바뀔 때마다 프로젝트 맥락을 잃어버리는 문제를 해결하기 위해 개발된 기술이다. 이를 통해 AI는 이전 세션의 맥락을 기억하고, 매번 처음부터 파일을 탐색할 필요가 없어진다. '마인드볼트'라는 시스템은 안드레이 카파시의 LLM 위키 패턴을 기반으로 한 오픈소스 도구로, AI가 프로젝트 자료를 자동으로 분석하고 위키 형태로 정리하는 기능을 제공한다. 이 도구는 검색, 그래프, 위키의 세 가지 레이어로 구성되어 있으며, 기존의 AI 도구와 통합하여 사용자의 작업 효율성을 극대화한다. AB 테스트 결과, 마인드볼트를 사용했을 때는 토큰 소비가 사실상 0에 가까워지고 응답 시간이 즉시로 단축되었다.
