# 거대 언어 모델

## 설명

거대 언어 모델(LLM)은 방대한 데이터를 학습하여 인간과 유사한 텍스트를 생성하고 이해하는 AI 모델입니다. 그러나 LLM은 때때로 기억 상실이나 환각과 같은 치명적인 단점을 보일 수 있습니다. 이러한 한계를 극복하고 모델의 신뢰성과 유용성을 높이기 위해 [[컨텍스트 엔지니어링]]과 같은 기술이 중요하게 활용됩니다. [[컨텍스트 엔지니어링]]은 LLM의 성능을 보완하고 확장하는 데 기여합니다.
