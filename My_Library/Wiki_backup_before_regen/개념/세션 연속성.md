# 세션 연속성

## 설명

세션 연속성은 사용자가 시스템과 상호작용하는 동안 지속적인 경험을 제공하는 개념이다. 이는 사용자의 이전 입력과 상태를 기억하여, 다음 상호작용에서 일관된 맥락을 유지하는 데 중요한 역할을 한다. 그러나 최근 마인드볼트라는 AI 코딩 도구의 폐기 결정에서 세션 연속성을 포함한 세 가지 핵심 기능이 실패했음을 밝혔으며, 문제의 근본 원인은 원래 개념을 잘못 이해한 데 있다고 지적되었다. LLM의 이해와 합성이 부족하여 기능의 본질이 사라졌다는 점은 주목할 만하다.
