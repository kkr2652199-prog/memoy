# 디자인 시스템

## 설명

Anthropic의 Claude Design은 Opus 4.7의 시각적 기능을 기반으로 웹, 앱, 슬라이드 디자인을 자동 생성하는 새로운 [[AI 디자인]] 앱입니다. 이 도구는 회사 정보, 기존 웹사이트 코드 또는 디자인 자산을 활용하여 [[디자인 시스템]]을 구축할 수 있게 합니다. 구축된 [[디자인 시스템]]을 통해 사용자는 일관된 디자인의 새로운 프로젝트를 효율적으로 만들 수 있습니다. 이는 디자인 프로세스의 [[일관성]]과 효율성을 크게 향상시킵니다.
