# 파일 시스템

## 설명

파일 시스템은 컴퓨터에서 데이터를 저장하고 관리하는 구조를 의미한다. 파일 시스템은 파일의 생성, 삭제, 읽기, 쓰기 등의 작업을 지원하며, 데이터의 조직화와 접근을 용이하게 한다. 최근에는 원격 개발 환경을 구축하기 위해 VS 코드의 리모트 SSH 기능을 활용하는 방법이 주목받고 있다. 이는 기존의 터뮤스 앱을 통한 SSH 연결에서 느꼈던 불편함을 해결하며, 개인 개발자에게 적합한 이유를 제시하고 있다.
