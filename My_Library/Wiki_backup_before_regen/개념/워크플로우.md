# 워크플로우

## 설명

워크플로우는 특정 작업이나 프로세스가 어떻게 진행되는지를 나타내는 체계적인 흐름이다. 사용자 자료 입력 규칙과 폴더 구조에 대한 가이드를 제공하는 것이 워크플로우의 효율성을 높이는 데 중요하다. 추천 폴더 구조는 '내_지식', '수집_꿀팁', '정제_노트'로 나뉘며, 각 폴더는 특정 주제에 따라 세분화된다. 실전 데이터 입력 시에는 타입 설정, 분류 입력, 제목 형식, 내용 작성의 4대 원칙을 준수해야 하며, 이는 사용자들이 자료를 체계적으로 정리하고 쉽게 접근할 수 있도록 돕는다. 새 자료는 부루 자동 매칭 프로그램의 사용법을 소개하며, CSV 파일을 활용하여 이미지와 영상을 자동으로 매칭하는 방법을 설명한다.
