# 일괄 편집

## 설명

[[일괄 편집]]은 여러 파일을 동시에 처리하는 기능을 의미하며, 2026-01-01에 소개된 'Every tool you could want to edit images in bulk' 자료에서 강조된 핵심 기능입니다. iLoveIMG는 이러한 [[대량 이미지 편집]] 기능을 제공하는 무료 온라인 웹 애플리케이션입니다. 이 플랫폼은 사용자들이 이미지를 쉽고 편리하게 편집하고 수정할 수 있도록 설계되었습니다. [[이미지 최적화]], 생성, [[이미지 변환]], 보안 등 광범위한 도구 세트를 제공하며, 100% 보안을 보장합니다.
