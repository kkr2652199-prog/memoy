# 자동화 프로그램

## 설명

자동화 프로그램은 사용자가 반복적인 작업을 효율적으로 수행할 수 있도록 돕는 소프트웨어 도구이다. 최근 출시된 구글 플로우(Nano Banana 2) 자동 생성 크롬 확장 프로그램은 사용자가 텍스트 파일에서 대본을 읽어와 자동으로 이미지를 생성하는 기능을 제공하여 콘텐츠 제작을 간소화한다. 이 프로그램은 5시간의 작업을 통해 개발되었으며, 사용자는 구글 드라이브에서 다운로드 후 크롬 확장 프로그램으로 등록하여 사용할 수 있다. 또한, '그록 자동화' 확장 프로그램은 영상 제작에 소요되는 시간을 획기적으로 줄이며, 반복적인 편집 작업을 자동화하여 창작에 집중할 수 있도록 돕는다. 새 자료는 부루 자동 매칭 프로그램의 사용법을 소개하며, CSV 파일을 활용하여 이미지와 영상을 자동으로 매칭하는 방법을 설명한다.
