# 개발 프로세스

## 설명

[[개발 프로세스]]는 GitHub와 같은 [[개발자 플랫폼]]을 통해 혁신되고 있습니다. GitHub는 개발자, 에이전트, 코드가 한 플랫폼에서 함께 작업하는 [[협업 개발]]을 지원하여 미래의 개발 방식을 제시합니다. 특히 GitHub Copilot과 같은 [[AI 기반 도구]]는 코드 작성, 테스트, 수정 과정을 가속화하여 개발 효율성을 크게 향상시킵니다. Copilot Autofix는 문제 해결 시간을 단축시켜 전체 개발 주기를 최적화하는 데 중요한 역할을 합니다.
