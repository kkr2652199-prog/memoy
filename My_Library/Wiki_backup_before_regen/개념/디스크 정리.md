# 디스크 정리

## 설명

디스크 정리는 컴퓨터의 저장 용량을 확보하기 위해 불필요한 파일과 시스템 파일을 삭제하는 과정입니다. 이 과정을 통해 사용자는 저장 공간을 효율적으로 관리하고, 필요 없는 데이터를 제거하여 시스템 성능을 향상시킬 수 있습니다. 디스크 정리는 간단한 방법으로 수행할 수 있으며, 추가적인 프로그램이나 명령어를 활용하여 더 효과적으로 용량을 늘릴 수 있습니다. 특히, 시스템 파일 정리와 같은 방법도 포함되어 있으며, 명령 프롬프트를 이용한 파일 삭제, 최대 절전 모드 비활성화, 윈도우 파일 압축 등의 추가적인 방법이 있습니다.
