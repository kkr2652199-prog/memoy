# 개발자 플랫폼

## 설명

GitHub는 개발자, 에이전트, 코드가 한 플랫폼에서 함께 작업하는 [[협업 개발]]의 미래를 제시하는 대표적인 [[개발자 플랫폼]]입니다. 이 플랫폼은 도구와 트렌드가 변화해도 협업의 가치는 변치 않는다는 철학을 바탕으로 합니다. 특히 GitHub Copilot과 같은 [[AI 기반 도구]]를 통해 코드 작성, 테스트, 수정 과정을 가속화하며, Copilot Autofix는 평균 문제 해결 시간(MTTR) 단축에 기여합니다. 사용자는 플랫폼 내에서 코드, 저장소, 사용자, 이슈, 풀 리퀘스트 등을 검색할 수 있습니다.
