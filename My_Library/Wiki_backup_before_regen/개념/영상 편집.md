# 영상 편집

## 설명

영상 편집은 다양한 영상 클립을 조합하고 수정하여 최종 영상을 만드는 과정입니다. 이 과정에서 사용자 자료 입력 규칙과 폴더 구조에 대한 가이드를 제공하는 것이 중요하며, 추천 폴더 구조는 '내_지식', '수집_꿀팁', '정제_노트'로 나뉩니다. 실전 데이터 입력 시에는 타입 설정, 분류 입력, 제목 형식, 내용 작성의 4대 원칙을 준수해야 하며, 이는 사용자들이 자료를 체계적으로 정리하고 쉽게 접근할 수 있도록 돕습니다.
