# 소프트웨어 개발

## 설명

소프트웨어 개발은 컴퓨터 프로그램과 애플리케이션을 설계, 구현, 테스트 및 유지 관리하는 포괄적인 과정입니다. 최근 [[AI 활용]]이 증가하며 코드 작성 생산성 향상에 기여하고 있지만, '[[vibe coding]]'과 같이 AI에 과도하게 의존하여 문제 해결에 대한 깊은 사고가 부족해지는 위험성도 지적됩니다. [[Cursor]]와 같은 AI 기반 코드 편집기는 세계적 수준의 소프트웨어를 개발하는 팀들이 신뢰하며, 즉시적인 코드 [[동기화]]와 [[병렬 실행]]을 제공하여 개발 워크플로우의 효율성을 극대화하고 더 빠르고 정확한 코딩을 돕습니다.
