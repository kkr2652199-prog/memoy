# CLI

## 설명

CLI(Command Line Interface)는 텍스트 기반의 명령어를 입력하여 컴퓨터를 제어하는 사용자 인터페이스입니다. AI 에이전트 관리 도구인 Codex 앱은 다양한 AI 모델을 지원하며, 특히 그 [[CLI]] 부분이 [[Open Source]]라는 큰 장점을 가집니다. 이는 사용자들이 명령줄을 통해 에이전트 작업을 효율적으로 관리하고 자동화할 수 있게 해주며, 오픈소스 특성 덕분에 커스터마이징 및 확장이 용이합니다. [[Agentic Coding]] 환경에서 CLI는 강력한 제어 수단으로 활용됩니다.
