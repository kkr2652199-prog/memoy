# 인코딩

## 설명

인코딩은 데이터를 특정 형식으로 변환하여 저장하거나 전송하는 과정을 의미한다. 현재 웹 스크래핑 시스템에서 인코딩 문제는 네이버와 유튜브 같은 동적 사이트에서 데이터 추출 실패의 원인 중 하나로 지적되고 있다. 이러한 문제는 특히 다양한 문자 세트를 사용하는 웹 페이지에서 발생할 수 있으며, 해결책으로는 JS 렌더링을 위한 httpx와 playwright 도입이 제안되고 있다.
