# 장르 선택

## 설명

장르 선택은 한국 전통 야담 대본 작성을 위한 첫 번째 단계로, 사용자가 대본의 장르를 결정하는 과정이다. 이 단계에서 사용자는 주제와 제목을 제시받고, 이를 바탕으로 인트로와 전체 대본을 작성하게 된다. 각 단계는 구체적인 규칙과 예시를 포함하고 있으며, 대본은 Save the Cat 구조를 기반으로 한다. 최종적으로 대본은 약 90분 분량으로 구성되며, 각 비트의 시간 배분과 내용이 포함된다.
