# LoRA

## 설명

LoRA (Low-Rank Adaptation)는 대규모 언어 모델(LLM)의 효율적인 파인튜닝을 위한 기술입니다. 이 기술은 모델의 전체 가중치를 업데이트하는 대신, 적은 수의 추가 파라미터만을 훈련하여 모델의 성능을 특정 작업에 맞게 조정합니다. 최근 실습 자료에서는 [[Unsloth]] 라이브러리를 활용하여 [[Google Colab]] 환경에서 [[Gemma 4]] 모델을 파인튜닝하는 과정을 상세히 다루고 있습니다 [ID 001]. 이 실습은 사용자가 자신만의 [[데이터셋]]을 [[Hugging Face]]에 업로드하고 이를 파인튜닝에 적용하는 방법을 안내하며, [[Gemma 4]] 모델의 [[멀티모달]] 능력을 확인하는 내용도 포함합니다 [ID 001]. 이는 LoRA와 같은 효율적인 파인튜닝 기법이 실제 LLM 적용에 어떻게 활용될 수 있는지를 보여줍니다.
