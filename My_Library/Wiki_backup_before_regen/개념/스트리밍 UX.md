# 스트리밍 UX

## 설명

스트리밍 UX는 [[로컬 AI]] 업데이트의 핵심 개선 사항 중 하나입니다. 이는 토큰 단위의 SSE(Server-Sent Events) 방식을 통해 AI의 답변이 실시간으로 사용자에게 보이는 경험을 제공합니다. send-stream과 chat_stream 기능을 활용하여 대화의 흐름을 끊김 없이 이어가며 사용자 상호작용을 향상시킵니다. 이러한 실시간 스트리밍은 AI 응답 지연을 체감상 줄여주어 더욱 자연스러운 대화 환경을 구축하는 데 기여합니다.
