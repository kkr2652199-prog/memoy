# Workspace Agent

## 설명

Workspace Agent는 AI 에이전트 작업을 효율적으로 관리하고 조율하는 도구와 관련된 개념입니다. [[Codex 앱]]은 이러한 Workspace Agent의 한 예시로, 다양한 AI 모델을 지원하며 특히 CLI 부분이 오픈소스라는 강점을 가집니다. 이는 여러 에이전트 작업을 동시에 관리하는 '관제탑' 역할을 수행하며, 병렬 실행 시 워크트리 기능을 통해 작업 간의 충돌을 방지합니다. [[Claude Code]]와 같은 다른 AI 에이전트 도구와 비교될 때, 더 넓은 모델 지원 범위와 오픈소스 CLI가 주요 차별점입니다.
