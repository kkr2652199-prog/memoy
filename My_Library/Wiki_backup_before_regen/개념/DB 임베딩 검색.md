# DB 임베딩 검색

## 설명

[[DB 임베딩 검색]]은 [[대규모 언어 모델]]의 [[기억 한계]]를 극복하고 [[장기 기억]]을 구현하기 위한 핵심 기능입니다. 이 시스템은 과거 대화 내용이나 중요한 정보를 데이터베이스에 임베딩 형태로 저장하고, 필요할 때 검색하여 모델의 [[Context Window]]로 주입합니다. 이를 통해 [[Gemma 4]]와 같은 LLM이 이전 대화를 기억하고 [[대화 맥락 유지 및 기억 능력]]을 향상시켜 [[AI 규칙 준수 문제]]를 해결하는 데 기여합니다. 이는 모델이 지속적으로 일관된 응답을 생성하고 사용자 경험을 개선하는 데 필수적인 외부 시스템적 접근 방식입니다.
