# 툴 오케스트레이션

## 설명

툴 오케스트레이션은 [[컨텍스트 엔지니어링]]의 핵심 기술 중 하나로, [[거대 언어 모델]](LLM)이 다양한 외부 도구와 상호작용하고 이를 조율하여 복잡한 작업을 수행할 수 있도록 합니다. 이는 LLM이 단순히 텍스트를 생성하는 것을 넘어, 실제 세계의 정보를 검색하거나 특정 작업을 실행하는 등 능동적으로 행동할 수 있게 만듭니다. 툴 오케스트레이션을 통해 AI는 더욱 강력하고 다재다능한 문제 해결 능력을 갖추게 됩니다.
