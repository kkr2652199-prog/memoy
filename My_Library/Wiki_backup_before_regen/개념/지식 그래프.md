# 지식 그래프

## 설명

지식 그래프는 개체와 그들 간의 관계를 시각적으로 표현한 구조로, 정보의 연결성을 강화하고 데이터의 의미를 이해하는 데 도움을 준다. 최근 안드레이 카파시의 LLM 위키 패턴을 기반으로 한 마인드볼트라는 오픈소스 도구가 소개되었으나, 이 도구의 약속된 기능들이 작동하지 않아 프로젝트를 폐기하기로 결정한 사례가 있다. 마인드볼트는 AI가 프로젝트 자료를 자동으로 분석하고 위키 형태로 정리하는 기능을 제공하지만, 세 가지 핵심 기능인 토큰 절약, 세션 연속성, 정확한 컨텍스트 주입이 모두 실패했음을 설명하며, 문제의 근본 원인은 원래 개념을 잘못 이해한 데 있다고 밝혀졌다. 또한, Andrej Karpathy는 LLM Wiki라는 프레임워크를 발표했으나, 이 시스템은 자기 인식이 부족하다는 문제를 지적하였다. 새 자료는 LLM Wiki가 개인 지식 베이스를 구축하기 위한 시스템으로, 대화 속 지식을 파일로 저장하여 다음 대화에서 이어갈 수 있도록 돕는다고 설명한다.
