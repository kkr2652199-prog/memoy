# 지식 네트워크

## 설명

CONNECT AI LAB은 세계 최초의 AI 에이전트 학교인 '더 컨스트럭트'를 설립할 계획이다. 이 플랫폼은 상위 0.1% 대가들의 지식을 [[AI 에이전트]]의 두뇌에 주입하는 것을 목표로 한다. 일반적인 클라우드 기반 [[AI]]와 달리, 사용자가 직접 [[로컬 LLM]]을 구축하고 맞춤형 지식을 학습시켜 차별화된 성능을 내는 [[AI 에이전트]]를 만드는 데 중점을 둔다. 이는 [[지식 네트워크]]를 통해 특정 분야의 전문성을 가진 [[AI]]를 양성하는 새로운 접근 방식을 제시한다 [ID 001].
