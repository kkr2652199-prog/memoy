# 배달대행사

## 설명

[[배달대행사]]는 음식이나 물품을 고객에게 배달하는 서비스를 제공하는 사업체로, 매 분기 [[부가세 신고]] 시 복잡한 [[정산서]] 데이터 처리로 많은 시간과 노력을 소모합니다. '시간단축! 배달대행 부가세 대리 처리 서비스'는 이러한 [[배달대행사]] 사장이 직접 겪었던 비효율적인 수작업 방식과 본업 방해 문제를 해결하기 위해 개발된 '부가세 해방' 솔루션입니다. 이 프로그램은 여러 업체의 [[정산서]]를 며칠 밤낮으로 정리해야 했던 고통을 덜어주며, [[정산서]] 엑셀 파일을 [[카카오톡]]이나 이메일로 보내면 [[홈택스]] [[일괄 등록]]용 최종 파일을 손쉽게 생성해줍니다. 이를 통해 [[배달대행사]]는 복잡한 세무 업무 부담을 줄이고 단 몇 번의 클릭으로 효율적인 [[부가세 신고]]를 완료하며 본업에 집중할 수 있습니다.
