# Agentic Coding

## 설명

Agentic Coding은 AI 에이전트를 활용하여 코딩 작업을 수행하는 개념을 포괄합니다. 최근 자료에 따르면, Codex 앱과 같은 AI 에이전트 관리 도구는 Claude Code와 같은 다른 AI 모델과 비교되며, 특히 다양한 AI 모델을 지원하는 강점을 가집니다. Codex 앱은 여러 [[Agentic Coding]] 작업을 동시에 관리하는 '관제탑' 역할을 수행하며, [[병렬 작업]] 실행 시 워크트리를 통해 충돌을 방지하는 기능을 제공합니다. 이 도구의 [[CLI]] 부분이 [[Open Source]]라는 점은 큰 장점으로 언급됩니다.
