# 커스텀 하네스

## 설명

커스텀 하네스는 특정 프로젝트의 요구에 맞춰 설계된 하네스 시스템으로, 성능 향상을 위해 활용된다. 메타 시니어 엔지니어가 설명한 바와 같이, 클로드 코드와 같은 도구들은 이미 하네스를 내장하고 있어 이를 통해 커스텀 하네스를 추가할 수 있다. 하네스의 기본 개념, 오픈 소스 하네스의 활용, 그리고 커스텀 하네스를 만드는 방법에 대한 이해는 프로젝트의 규칙과 아키텍처 결정을 문서화하여 AI에게 전파하는 데 중요하다. 새 자료에서는 하네스 엔지니어링의 개념과 활용 방법을 상세히 다루며, 특히 프로젝트에 맞는 커스텀 하네스를 추가하여 성능을 향상시킬 수 있는 방법을 강조한다.
