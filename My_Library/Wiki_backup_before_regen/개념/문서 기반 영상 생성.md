# 문서 기반 영상 생성

## 설명

문서 기반 영상 생성은 텍스트 문서나 스크립트를 입력하여 인공지능이 자동으로 영상 콘텐츠를 만들어내는 기술입니다. 이 방식은 대본 작성부터 영상 제작까지의 과정을 효율적으로 자동화하여 [[AI 영상 제작]] 시간을 크게 단축시킵니다. 다양한 AI 영상 도구들을 비교할 때, 문서 기반 영상 생성 기능의 유무와 성능은 중요한 평가 요소가 됩니다. 특히 교육 콘텐츠나 정보 전달 목적의 영상 제작에서 이 기능은 핵심적인 역할을 수행할 수 있습니다.
