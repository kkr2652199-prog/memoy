# 이미지 생성

## 설명

이미지 생성은 컴퓨터 알고리즘을 사용하여 이미지 콘텐츠를 자동으로 만들어내는 과정입니다. 최근 [[그록 AI]]가 유료로 전환되면서 [[메타 AI]]가 무료 대체 도구로 소개되었으며, 이는 이미지와 영상 생성 기능을 제공하지만 한국에서는 VPN을 통해 접근해야 합니다. [[Gemini]]와 같은 다른 AI 시스템 또한 이미지 및 동영상 생성 기능을 핵심 기능으로 제공하여 사용자가 다양한 시각 콘텐츠를 손쉽게 제작할 수 있도록 돕습니다. 특히 [[NotebookLM]]과 [[Google Gemini]]를 활용하면 핀터레스트 스타일의 대량 이미지를 무료로 생성하여 유튜브 플레이리스트와 같은 AI 기반 콘텐츠 제작에 활용할 수 있으며, 이는 효율적인 [[워크플로우]] 구축에 기여합니다.
