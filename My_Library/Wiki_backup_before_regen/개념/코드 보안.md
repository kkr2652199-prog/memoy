# 코드 보안

## 설명

[[코드 보안]]은 [[개발 프로세스]] 전반에 걸쳐 중요한 요소이며, [[개발자 플랫폼]]의 발전과 함께 진화하고 있습니다. GitHub는 개발자, 에이전트, 코드가 한 플랫폼에서 함께 작업하는 [[협업 개발]] 환경을 제공하며, 이는 보안 취약점의 조기 발견 및 해결에도 기여할 수 있습니다. GitHub Copilot과 같은 [[AI 기반 도구]]는 코드 작성 및 수정 과정을 가속화하여, 잠재적인 보안 문제를 더 빠르게 식별하고 수정하는 데 도움을 줄 수 있습니다. 특히 Copilot Autofix는 평균 문제 해결 시간(MTTR)을 단축함으로써, 발견된 보안 취약점에 대한 신속한 대응을 가능하게 합니다.
