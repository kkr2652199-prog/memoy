# 자기 개선 시스템

## 설명

자기 개선 시스템은 사용자의 작업 이력을 기억하고 스스로 학습하는 기능을 갖춘 혁신적인 도구를 포함한다. 예를 들어, 헤르메스 에이전트는 기존 AI 코딩 도구와 달리 서버에 상주하며 텔레그램, 디스코드 등 다양한 플랫폼에서 접근 가능하다. 이 에이전트는 복잡한 작업을 수행할 때마다 자동으로 스킬 파일을 생성하고, 이를 기반으로 다음 작업을 효율적으로 처리한다. 또한, 사용자 모델링 시스템을 통해 개인 맞춤형 서비스를 제공하여 자기 개선의 효과를 극대화한다.
