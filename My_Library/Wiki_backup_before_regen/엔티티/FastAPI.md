# FastAPI

## 개요

FastAPI는 My_Library라는 개인 지식 도서관 애플리케이션 개발에 사용된 핵심 프레임워크입니다. 이 프로젝트는 2026년 4월 19일 기준으로 172개의 자료를 관리하며, [[SQLite]], [[Ollama]] (특히 [[llama3]] 모델) 및 [[D3.js]]와 함께 사용되어 구축되었습니다. FastAPI는 이 애플리케이션의 백엔드 및 API 로직을 담당하며, 효율적인 데이터 처리와 검색 기능을 지원합니다.
