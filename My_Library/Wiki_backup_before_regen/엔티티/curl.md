# curl

## 개요

curl은 다양한 프로토콜을 사용하여 데이터를 전송하는 데 사용되는 강력한 명령줄 도구입니다. 웹 서버에서 파일을 다운로드하거나 데이터를 업로드하는 등 네트워크 통신에 널리 활용됩니다. 특히, [[Cursor]]와 같은 소프트웨어를 설치할 때 설치 스크립트를 실행하거나 파일을 다운로드하는 데 사용됩니다. 개발 환경 구축 시 필수적인 도구로, 간단한 명령어를 통해 효율적인 데이터 전송을 가능하게 합니다.
