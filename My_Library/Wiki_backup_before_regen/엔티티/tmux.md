# tmux

## 개요

[[tmux]]는 터미널 멀티플렉서로, 단일 터미널 창에서 여러 세션을 관리하고 분할할 수 있게 해주는 유용한 도구이다. 최근 [[Claude Code]] 사용을 위한 최강 터미널 환경을 구축하는 데 필수적인 요소로 소개되었다. [[iTerm2]], [[lazygit]], [[zoxide]] 등과 함께 활용되어 개발자의 작업 효율성을 극대화한다. [[tmux]]를 사용하면 터미널 세션을 유지한 채로 연결을 끊었다가 다시 연결할 수 있어, 장시간 작업이나 원격 작업 시 특히 유용하다.
