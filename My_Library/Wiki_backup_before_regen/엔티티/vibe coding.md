# vibe coding

## 개요

이 개념은 최근 영상 'The Vibe Coding Trap (Do This Instead)'에서 다루어졌으며, 개발자가 AI를 사용하여 코드를 작성할 때의 사고 부족 문제를 강조한다. 실제 문제는 AI 사용 여부가 아니라 그 과정에서의 사고 부족에 있다고 주장하며, 예를 들어 호텔 예약 플랫폼을 개발할 경우 AI가 생성한 코드가 실제 사용 시 서버에 부하를 주는 문제를 일으킬 수 있다. 따라서 개발자는 AI에 의존하기보다는 문제 해결에 집중해야 한다.
