# LM Studio

## 개요

LM Studio는 로컬 환경에서 대규모 언어 모델(LLM)을 실행하고 관리할 수 있게 해주는 도구로, 사용자가 자신의 컴퓨터에서 다양한 LLM을 다운로드하고 실행하여 오프라인 환경에서도 AI 모델을 활용할 수 있도록 지원합니다. 2026년 4월 20일자 '세계 최초의 AI 에이전트 학교를 만들겁니다.' 자료에서 [[더 컨스트럭트]] 플랫폼이 로컬 LLM 구축에 중점을 둔다고 언급되어 LM Studio의 활용 가능성을 시사했습니다. 특히 2026년 4월 21일자 `로컬_AI_업데이트_이전·이후_분석_보고_1`에 따르면, `[[POST /api/chat/send-stream]]`을 통한 스트리밍 UX 개선 및 지연 감소 기능은 UI가 로컬 환경 또는 [[LM Studio]]일 때 활성화되는 것으로 명시되어, LM Studio가 이러한 최적화된 로컬 AI 경험의 핵심 요소임을 보여줍니다.
