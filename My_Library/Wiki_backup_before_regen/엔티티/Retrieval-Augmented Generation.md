# Retrieval-Augmented Generation

## 개요

Retrieval-Augmented Generation (RAG)은 대규모 언어 모델(LLM)의 성능을 향상시키는 프레임워크로, 외부 지식 기반에서 관련 정보를 검색하여 답변을 생성하는 방식입니다. 이 기술은 모델이 대화 맥락을 유지하고 장기적인 '기억' 능력을 강화하여, [[Gemma 4]]와 같은 모델이 사용자 질문에 대해 더욱 정확하고 풍부한 정보를 제공할 수 있도록 돕습니다. RAG는 LLM의 환각 현상을 줄이고 최신 정보에 접근할 수 있게 하는 핵심적인 방법론으로 평가받습니다.
