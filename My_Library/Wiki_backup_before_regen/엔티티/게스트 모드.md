# 게스트 모드

## 개요

게스트 모드는 주로 공용 컴퓨터 환경에서 사용자의 개인 정보 보호를 강화하기 위해 설계된 기능이다. 이 모드는 [[Google]] 계정 로그인 페이지와 같은 서비스에서 제공되며, 사용자가 로그인 없이 웹을 탐색하거나 일시적으로 계정에 접근할 때 개인 데이터가 기기에 저장되지 않도록 돕는다. 이를 통해 사용자는 자신의 검색 기록, 쿠키, 기타 활동 정보가 공용 컴퓨터에 남지 않도록 할 수 있다. 게스트 모드는 특히 보안이 중요한 환경에서 유용하게 활용된다.
