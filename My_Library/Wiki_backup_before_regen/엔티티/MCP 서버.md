# MCP 서버

## 개요

MCP 서버는 클라우드 기반의 서버 솔루션으로, 다양한 애플리케이션과 서비스의 운영을 지원한다. 최근에는 클로드 코드 사용 시 발생하는 비용을 줄이는 방법에 대한 관심이 높아지고 있다. 개발자 평균 비용은 하루 6달러로, 한 달에 180달러가 넘는 경우도 있다. 이를 위해 입력, 출력, 사고 토큰의 구조를 이해하고, 클로드 이그노어 설정, 슬래시 클리어 및 슬래시 컴팩트 활용, 작업에 맞는 모델 선택, 정확한 프롬프트 작성, 플랜 모드 활용 등의 방법이 제안된다.
