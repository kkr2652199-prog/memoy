# GitHub Copilot

## 개요

[[GitHub Copilot]]은 GitHub이 제시하는 협업 개발의 미래에서 핵심적인 역할을 하는 AI 기반 개발자 도구이다. 이 도구는 코드 작성, 테스트, 수정 과정을 가속화하여 개발 워크플로우를 효율적으로 만든다. 특히 [[Copilot Autofix]]와 같은 기능을 통해 개발자들이 직면하는 문제의 평균 해결 시간을 단축하는 데 기여한다. GitHub의 철학인 '협업의 가치는 변치 않는다'는 기조 아래, 개발자와 에이전트, 코드가 함께 작업하는 환경을 지원한다.
