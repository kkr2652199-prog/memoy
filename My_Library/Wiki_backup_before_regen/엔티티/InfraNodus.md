# InfraNodus

## 개요

InfraNodus는 연구 아이디어를 시각화하고 지식의 공백을 식별하는 도구로, Andrej Karpathy의 LLM Wiki 프레임워크와 관련이 있다. LLM Wiki는 연구 자료를 정리하고 개념 간의 연결을 생성하지만, 자기 인식이 부족하다는 문제를 지적받았다. 이를 해결하기 위해 InfraNodus를 활용하면 사용자가 원하는 통찰력을 생성하기 위한 지식 그래프를 구축할 수 있다. 이 도구는 연구자들이 더 효과적으로 정보를 분석하고 활용할 수 있도록 돕는다. 또한, LLM Wiki는 사용자가 원하는 통찰력을 생성하기 위해 지식 그래프가 필요하다는 점이 강조되었다.
