# OpenClaw

## 개요

OpenClaw는 Ollama 플랫폼을 사용하여 Claude Code 및 기타 오픈 모델과 함께 빠르고 쉽게 실행할 수 있는 제품입니다. Ollama는 사용자가 몇 분 안에 이러한 모델들을 설정하고 구동할 수 있도록 지원하며, `ollama launch openclaw`와 같은 간단한 명령어를 통해 설치 및 구성 과정을 자동화합니다. 특히, OpenClaw는 Ollama를 통해 Claude Code와 같은 오픈 모델들과 함께 손쉽게 배포 및 활용될 수 있어, 개발 및 활용의 진입 장벽을 낮춥니다.
