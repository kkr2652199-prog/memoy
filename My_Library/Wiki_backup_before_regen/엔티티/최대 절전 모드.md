# 최대 절전 모드

## 개요

최대 절전 모드는 컴퓨터의 전원을 끄지 않고도 전력 소비를 줄이는 기능이다. 이 기능을 활성화하면 현재 작업 중인 내용을 저장하고 시스템 상태를 유지한 채로 전원이 꺼진다. 사용자는 필요할 때 이 상태에서 쉽게 컴퓨터를 다시 켜서 작업을 이어갈 수 있다. 최근 자료에서는 최대 절전 모드를 비활성화하여 저장 공간을 늘리는 방법이 언급되었다.
