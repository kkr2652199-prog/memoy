# Cursor

## 개요

Cursor는 세계적 수준의 소프트웨어를 개발하는 팀들이 매일 신뢰하는 AI 기반 코드 편집기입니다. 이 도구는 즉각적인 코드 동기화와 병렬 실행 기능을 제공하여 개발 워크플로우의 효율성을 극대화합니다. 모든 개발 도구와 단계에 걸쳐 활용될 수 있는 유연성을 제공하며, 개발 워크플로우 전반에 걸쳐 AI의 도움을 받을 수 있습니다. 간단한 `[[curl]]` 명령어를 통해 쉽게 설치할 수 있으며, [[설치 스크립트]]를 활용하여 효율적인 개발 환경을 구축할 수 있습니다. 이를 통해 개발자들은 더 빠르고 정확하게 코딩할 수 있습니다.
