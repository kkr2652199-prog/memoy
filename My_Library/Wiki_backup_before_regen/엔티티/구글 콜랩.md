# 구글 콜랩

## 개요

구글 콜랩은 클라우드 기반의 주피터 노트북 환경으로, 인공지능 모델 파인튜닝 실습에 널리 활용되는 서비스입니다. 특히 [[잼마 4]] 모델을 파인튜닝하는 과정에서 코랩 환경을 설정하고 모델을 불러오는 데 사용됩니다. 사용자는 [[구글 콜랩]]을 통해 멀티모달 능력을 확인하고, [[허깅 페이스]]에 업로드된 데이터셋을 활용하여 모델을 훈련할 수 있습니다.
