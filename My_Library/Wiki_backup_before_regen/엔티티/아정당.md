# 아정당

## 개요

아정당은 컴퓨터의 저장 용량을 효율적으로 관리하는 방법을 제공하는 기업이다. 이들은 디스크 정리와 특정 프로그램을 활용하여 불필요한 파일을 삭제하고, 용량이 큰 폴더를 쉽게 식별할 수 있는 솔루션을 제안한다. 또한, 명령 프롬프트를 이용한 파일 삭제, 최대 절전 모드 비활성화, 윈도우 파일 압축 등의 방법을 통해 사용자들이 저장 공간을 늘릴 수 있도록 돕는다.
