# AI 크래프터 스튜디오

## 개요

'AI 크래프터 스튜디오'는 유튜브 조회수를 높이기 위한 AI 기반의 분석 툴이다. 이 프로그램은 [[YouTube Data API V3]]를 활용하여 조회수가 급상승한 영상의 패턴을 분석하고, 사용자가 입력한 키워드에 따라 관련 영상을 신속하게 찾아준다. 이를 통해 유튜버는 효과적인 콘텐츠 전략을 세울 수 있으며, 복잡한 유료 툴 대신 무료로 제공되어 유튜버들에게 유용한 도구가 될 것이다. 이 툴은 [[YouTube API Key]]를 사용하여 데이터를 수집하며, 유튜버들이 콘텐츠 기획에 활용할 수 있는 인사이트를 제공한다.
