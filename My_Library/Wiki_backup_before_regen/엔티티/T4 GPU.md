# T4 GPU

## 개요

T4 GPU는 구글 코랩과 같은 클라우드 환경에서 인공지능 모델 파인튜닝 실습에 흔히 사용되는 그래픽 처리 장치입니다. [[구글 콜랩]]을 활용하여 [[잼마 4]] 모델을 파인튜닝하는 과정에서 효율적인 연산 처리를 위해 활용될 수 있습니다. 이는 복잡한 모델 훈련 및 멀티모달 능력 확인과 같은 작업에 필수적인 컴퓨팅 자원을 제공합니다.
