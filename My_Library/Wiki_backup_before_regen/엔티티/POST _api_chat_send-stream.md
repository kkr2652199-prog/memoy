# POST /api/chat/send-stream

## 개요

POST /api/chat/send-stream은 2026년 4월 21일자 `로컬_AI_업데이트_이전·이후_분석_보고_1`에서 언급된 로컬 AI 업데이트 이후의 핵심 API 엔드포인트입니다. 이 엔드포인트는 스트리밍 UX를 위해 토큰 단위의 SSE(Server-Sent Events) 방식으로 AI 답변을 실시간으로 제공하며, `chat_stream` 기능과 함께 작동합니다. 지연을 줄이기 위해 키워드 기반 `quick_intent`, 경량 `LOCAL_STREAM_*` 프롬프트, 약 1000자의 컨텍스트, 그리고 `max_items 3`과 같은 최적화 전략이 적용되었습니다. 이 스트리밍 지능 기능은 UI가 로컬 환경이거나 [[LM Studio]]일 때만 활성화됩니다.
