# LLM 위키

## 개요

'LLM 위키'는 AI가 스스로 글쓰기 원칙을 학습하고 정보를 체계적으로 정리하여 대본을 자동으로 작성하는 혁신적인 시스템이다. 이 시스템은 사용자가 수집한 정보를 로우 폴더에 저장하고, AI가 이를 정리하여 위키 형태로 가꾸는 방식으로 작동한다. 사용자는 '수집', '질문', '정리'라는 세 가지 명령어를 통해 AI 정원사를 활용할 수 있으며, 이는 개인의 지식을 효과적으로 관리할 수 있도록 지원한다. 최근 마인드볼트라는 AI 코딩 도구의 기능이 제대로 작동하지 않아 프로젝트가 폐기되었으며, 이는 LLM의 이해와 합성이 부족했던 것이 원인으로 지적되었다.
