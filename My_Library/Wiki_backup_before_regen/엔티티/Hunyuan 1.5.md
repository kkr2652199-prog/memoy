# Hunyuan 1.5

## 개요

Hunyuan 1.5는 비디오 생성의 효율성을 높이기 위해 Auto Hunyuan이라는 브라우저 확장 프로그램과 함께 사용됩니다. 이 도구는 사용자가 이미지를 드래그 앤 드롭하고 프롬프트를 입력하면 자동으로 비디오를 생성하며, 완료된 클립을 자동으로 다운로드하고 정리해줍니다. Grok의 무료 사용 중단 이후, Hunyuan 1.5를 활용한 비디오 생성이 수작업으로는 비효율적이라는 문제를 해결하기 위해 이 확장 프로그램이 개발되었으며, 2026년 4월 5일에 출시되었습니다.
