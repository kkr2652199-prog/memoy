# Google

## 개요

구글은 최근 [[Gemma 4]]라는 고품질 AI 모델을 출시하여 사용자가 로컬 컴퓨터에서 인터넷 연결이나 API 비용 없이도 AI 기능을 활용할 수 있도록 했다. 이와 더불어, 구글 검색은 복잡한 알고리즘을 통해 사용자의 쿼리에 가장 관련성 높은 결과를 제공하며, 개인정보 처리방침을 통해 데이터를 안전하게 관리한다. 또한, 구글은 계정 로그인 페이지를 통해 사용자가 이메일이나 휴대전화 번호로 접근할 수 있도록 하며, 보안 강화를 위해 CAPTCHA 기능을 제공한다. 특히 공용 컴퓨터 사용자를 위한 [[게스트 모드]]를 지원하여 개인 정보 보호를 강화하는 등 다양한 서비스를 제공하고 있다.
