# Brian Kernighan

## 개요

Brian Kernighan은 컴퓨터 과학자이자 프로그래머로, 주로 C 프로그래밍 언어와 UNIX 운영 체제의 개발에 기여한 것으로 잘 알려져 있다. 그는 또한 'The C Programming Language'라는 유명한 교과서를 공동 저술하였다. 최근 영상 'The Vibe Coding Trap (Do This Instead)'에서는 'vibe coding'의 개념과 그에 대한 오해를 다루며, 개발자가 AI를 사용하여 코드를 작성할 때의 사고 부족 문제를 강조하였다. 이 영상은 AI의 코드 생성이 실제 사용 시 서버에 부하를 주는 문제를 일으킬 수 있음을 경고하며, 개발자들이 AI에 의존하지 말고 문제 해결에 집중해야 한다.
