# LLM 위키

## 설명

LLM 위키는 AI가 고품질의 전문적인 콘텐츠를 생성하기 위해 자료를 수집, 정리, 관리하는 'AI 도서관' 개념의 지식 관리 시스템이다.

유형/분류:
- AI 기반 지식 관리 시스템: AI가 자료 수집, 정리, 질문 답변, 모순 점검 등 지식 관리 전반을 수행한다.
- 고품질 콘텐츠 생성 도구: 위키백과 요약 수준을 넘어 감정 곡선, 팩트 체크를 포함한 전문적인 대본 등을 생성한다.
- AI 도서관 구축 방법론: AI가 스스로 자료를 쌓고 유지보수하며 지식을 심화시키는 시스템을 구축하는 방법론이다.

핵심 원리: AI가 'AI 사서' 역할을 수행하며 자료를 수집, 정리하고 질문에 답변하며 모순을 점검한다. 도서관에 자료가 축적될수록 AI의 결과물은 깊어지고, AI가 스스로 시스템을 유지보수하여 관리 부담을 줄인다.
