# DB 임베딩 검색

## 설명

DB 임베딩 검색은(는) 텍스트, 이미지 등 다양한 형태의 데이터를 벡터 공간에 임베딩하여 저장하고, 쿼리 임베딩과의 유사도를 기반으로 데이터베이스에서 가장 관련성 높은 정보를 찾아내는 기술입니다.

유형/분류:
- 의미 기반 검색: 쿼리의 의미와 가장 유사한 데이터를 벡터 공간에서 찾아내는 방식.
- 대화 컨텍스트 검색: 이전 대화 기록을 임베딩하여 현재 질의에 필요한 맥락 정보를 검색하는 방식.

핵심 원리: 데이터를 고차원 벡터 공간에 임베딩하여 의미적 유사성을 벡터 간의 거리로 표현합니다. 이후 쿼리 임베딩과 저장된 데이터 임베딩 간의 유사도(예: 코사인 유사도)를 계산하여 가장 가까운 데이터를 검색합니다.
