# 하이브리드 검색

## 설명

하이브리드 검색은 키워드 기반의 희소 검색(BM25)과 의미 기반의 밀집 벡터 검색을 결합하여 검색 정확도와 관련성을 높이는 기술이다.

유형/분류:
- BM25-벡터 결합: 키워드 매칭과 임베딩 벡터 유사도를 동시에 활용하여 검색 결과를 도출하는 방식
- 다단계 벡터 검색: 임베딩 클라이언트 구축, 자료 임베딩, 검색 쿼리 임베딩 등 여러 단계를 거쳐 벡터 유사도를 계산하는 방식

핵심 원리: 희소 검색으로 초기 관련 문서를 빠르게 필터링하고, 밀집 벡터 검색으로 의미론적 유사성이 높은 문서를 찾아내어 두 결과를 통합함으로써 사용자 질의에 대한 포괄적이고 정확한 응답을 제공한다. 이는 자료의 임베딩을 미리 생성하고 저장하여 효율적인 유사도 비교를 가능하게 한다.

관련 도구: `nomic-embed-text-v2-moe`, [[Ollama]]
