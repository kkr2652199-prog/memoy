# Open Source

## 설명

Open Source은(는) 소스 코드가 공개되어 누구나 자유롭게 접근, 사용, 수정 및 배포할 수 있도록 허용하는 개발 및 배포 모델이다.

유형/분류:
- 오픈 소스 소프트웨어: 소스 코드가 공개되어 자유로운 사용, 수정, 배포가 가능한 모든 소프트웨어.
- 오픈 소스 라이브러리/프레임워크: 특정 기능 구현을 위해 공개된 코드 묶음 또는 개발 환경.
- 오픈 소스 도구: 특정 작업을 수행하는 데 사용되는 공개된 소프트웨어 도구.

핵심 원리: 소스 코드의 투명한 공개를 통해 사용자 및 개발자 커뮤니티의 협력적인 개선과 확장을 촉진하며, 이를 통해 더 높은 유연성과 신뢰성을 확보한다.
