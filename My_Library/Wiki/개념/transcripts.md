# transcripts

## 설명

transcripts은(는) 음성 또는 영상 콘텐츠의 대화나 소리를 텍스트 형태로 변환하여 기록한 문서이다.

유형/분류:
- 수동 생성 transcripts: 사람이 직접 듣고 작성하여 정확도가 높은 텍스트 기록
- 자동 생성 transcripts: 음성 인식 기술을 활용하여 자동으로 생성된 텍스트 기록
- 자막: 영상 콘텐츠의 대화를 화면에 표시하기 위해 시간 정보와 함께 제공되는 텍스트

핵심 원리: transcripts는 음성 콘텐츠의 소리를 텍스트로 변환하는 과정을 통해 생성되며, 이는 수동 작업 또는 음성 인식 기술을 통해 이루어진다.
