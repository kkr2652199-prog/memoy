# 키프레임

## 설명

키프레임은(는) 애니메이션, 영상 편집, 3D 모델링 등 시간 기반 미디어에서 객체의 속성 변화를 정의하는 특정 시점의 핵심 프레임이다.

유형/분류:
- 애니메이션 키프레임: 캐릭터나 객체의 움직임, 형태 변화 등 특정 시점의 중요한 상태를 정의하여 중간 프레임 생성을 위한 기준점을 제공.
- 영상 편집 키프레임: 영상 클립의 불투명도, 크기, 위치, 오디오 볼륨 등 속성의 변화를 특정 시점에 지정하여 시간 경과에 따른 효과를 제어.
- 3D 애니메이션 키프레임: 3D 객체의 변형(이동, 회전, 크기 조절)이나 재질 속성 변화를 특정 시간대에 기록하여 애니메이션을 생성.

핵심 원리: 키프레임은 시간 축 상의 특정 지점에서 객체의 속성 값을 설정하며, 시스템은 이 키프레임들 사이의 중간 값을 자동으로 보간(interpolation)하여 부드러운 변화나 움직임을 생성한다.
