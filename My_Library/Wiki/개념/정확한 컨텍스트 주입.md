# 정확한 컨텍스트 주입

## 설명

정확한 컨텍스트 주입은(는) 대규모 언어 모델(LLM)이 특정 작업에 대한 깊은 이해와 정확한 합성을 수행할 수 있도록, 가장 관련성 높고 필수적인 정보를 선별하여 제공하는 기법이다.

유형/분류:
- 정보 선별 (Information Selection): LLM에 주입될 데이터의 관련성을 평가하고 불필요한 부분을 제거하는 과정.
- 의미론적 정렬 (Semantic Alignment): 주입된 컨텍스트가 LLM의 내부 지식 표현과 일관되도록 조정하여 오해를 방지.
- 토큰 효율성 (Token Efficiency): 최소한의 토큰으로 최대의 정보 가치를 전달하여 처리 비용을 최적화.

핵심 원리: LLM이 당면한 문제의 본질을 정확히 파악하고 일관된 추론을 할 수 있도록, 외부 지식이나 이전 대화 기록 등에서 핵심 정보를 추출하여 모델의 입력 프롬프트에 효과적으로 통합하는 것을 목표로 한다.
