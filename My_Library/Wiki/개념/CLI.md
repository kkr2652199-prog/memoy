# CLI

## 설명

CLI은(는) 텍스트 명령어를 입력하여 컴퓨터 프로그램이나 운영체제를 제어하는 사용자 인터페이스이다.

유형/분류:
- 시스템 관리 CLI: 운영체제 및 서버 자원을 제어하고 관리하는 데 사용된다.
- 애플리케이션 CLI: 특정 소프트웨어 애플리케이션의 기능을 명령어로 실행하고 설정하는 데 사용된다.
- 개발 도구 CLI: 소프트웨어 개발 및 배포 워크플로우를 자동화하고 관리하는 데 사용된다.

핵심 원리: 사용자가 입력한 텍스트 명령어를 커맨드 라인 인터프리터(셸)가 해석하여 해당 작업을 실행하며, 이를 통해 정밀한 제어와 작업 자동화가 가능하다.
