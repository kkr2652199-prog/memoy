# 컨텍스트 엔지니어링

## 설명

컨텍스트 엔지니어링은 거대 언어 모델(LLM)의 기억 상실과 환각 문제를 극복하고 AI가 자율적으로 작동할 수 있는 환경을 구축하기 위해 외부 지식 제공, 메모리 관리, 도구 활용 등을 설계하는 기술이다.

유형/분류:
- 검색 증강 생성(RAG): AI에 외부 지식을 제공하여 답변의 정확성을 높임.
- 메모리 관리: AI가 과거 대화나 정보를 기억하고 활용할 수 있도록 지원함.
- 툴 오케스트레이션: AI가 실제 도구를 사용하여 작업을 수행하도록 조정하고 제어함.

핵심 원리: 컨텍스트 엔지니어링은 LLM의 한계인 기억 상실과 환각을 보완하기 위해 AI에게 필요한 정보를 적시에 제공하고, 과거 상호작용을 기억하게 하며, 외부 도구를 활용할 수 있도록 시스템을 설계하여 AI의 자율성과 정확성을 극대화하는 것을 목표로 한다.
