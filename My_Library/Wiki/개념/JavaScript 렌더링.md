# JavaScript 렌더링

## 설명

JavaScript 렌더링은(는) 웹 브라우저가 JavaScript 코드를 실행하여 웹 페이지의 콘텐츠와 구조를 동적으로 생성하거나 수정하는 과정이다.

유형/분류:
- 클라이언트 사이드 렌더링 (CSR): 브라우저에서 JavaScript를 실행하여 페이지를 구성하고 업데이트한다.
- 서버 사이드 렌더링 (SSR): 서버에서 JavaScript를 실행하여 초기 HTML을 생성한 후 클라이언트에 전송한다.
- 정적 사이트 생성 (SSG): 빌드 시점에 JavaScript를 사용하여 HTML 파일을 미리 생성하여 제공한다.
- 하이드레이션: SSR로 생성된 HTML에 클라이언트 사이드 JavaScript 기능을 연결하여 상호작용 가능하게 만든다.

핵심 원리: 웹 페이지 로드 시 또는 사용자 상호작용에 따라 JavaScript 엔진이 코드를 해석하고 실행하여 DOM(문서 객체 모델)을 조작함으로써 화면에 표시될 내용을 동적으로 변경한다.

관련 도구: [[React]], [[Angular]], [[Vue.js]], [[Next.js]]
