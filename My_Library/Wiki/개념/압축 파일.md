# 압축 파일

## 설명

압축 파일은(는) 데이터 크기를 줄여 저장 공간을 절약하고 전송 효율을 높이기 위해 특정 알고리즘으로 처리된 하나 이상의 파일 묶음이다.

유형/분류:
- 무손실 압축: 원본 데이터를 손상 없이 완벽하게 복원할 수 있는 방식.
- 손실 압축: 일부 데이터를 제거하여 압축률을 높이지만, 원본 데이터의 완벽한 복원이 불가능한 방식.
- 아카이브 파일: 여러 파일을 하나로 묶어 관리하기 쉽게 만들며, 압축 기능이 포함될 수 있는 형식.

핵심 원리: 파일 내의 중복되거나 예측 가능한 패턴을 찾아내어 더 효율적인 방식으로 재구성하거나, 중요도가 낮은 데이터를 제거하여 원본 파일의 정보량을 줄이는 방식으로 작동한다. 이를 통해 파일 크기를 줄여 저장 및 전송 효율을 높인다.

관련 도구: [[WinRAR]], [[7-Zip]], [[반디집]], [[ZIP]]
