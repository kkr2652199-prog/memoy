# Atomic Notes

## 설명

Atomic Notes은(는) 단일하고 명확한 아이디어를 담고 있으며, 지식 관리 시스템(KMS) 내에서 효율적인 지식 구조 설계를 위해 상호 연결될 수 있도록 설계된 지식 단위이다.

유형/분류:
- 지식 구조 요소: 특정 주제나 개념을 독립적으로 표현하며, 전체 지식 체계의 구성에 기여하는 최소 단위.
- KMS 핵심 구성: 지식 관리 시스템에서 정보의 검색, 연결 및 재활용성을 높이는 데 필수적인 기본 블록.

핵심 원리: 각 노트는 하나의 아이디어에 집중하여 명확성을 유지하고, 다른 노트와의 유기적인 링크를 통해 복잡한 지식 네트워크를 구축함으로써 지식 구조 설계의 유연성과 확장성을 제공한다.

관련 도구: [[Zettelkasten]], [[지식 관리 시스템]], [[LLM]]
