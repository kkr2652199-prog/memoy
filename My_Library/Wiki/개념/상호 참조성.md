# 상호 참조성

## 설명

상호 참조성은 정보 시스템 내에서 서로 관련된 항목이나 개념들을 연결하여 사용자가 정보를 쉽게 탐색하고 맥락을 이해할 수 있도록 돕는 원리입니다.

유형/분류:
- 내부 참조: 동일 시스템 내에서 다른 항목으로 연결하여 정보의 흐름을 제공합니다.
- 외부 참조: 시스템 외부의 관련 자료나 정보원으로 연결하여 정보의 범위를 확장합니다.
- 개념적 참조: 특정 개념이나 주제를 공유하는 정보들을 연결하여 심층적인 이해를 돕습니다.
- 구조적 참조: 지식 기반 내의 계층적 또는 네트워크 관계를 기반으로 정보를 연결합니다.

핵심 원리: 명시적인 링크나 관계를 통해 정보 조각들을 연결함으로써, 사용자가 관련 정보를 쉽게 찾아보고 지식 기반 전체의 일관성과 응집력을 높이는 데 기여합니다. 이는 정보의 발견 가능성을 향상시키고, 복잡한 주제에 대한 이해를 심화시킵니다.
