# 데이터 섭취

## 설명

데이터 섭취는 다양한 소스에서 데이터를 수집하여 저장 시스템으로 가져오고, 후속 처리를 위해 준비하는 과정이다.

유형/분류:
- 자동화된 섭취: 신규 자료가 발생하면 자동으로 시스템에 통합되어 처리되는 방식.
- 배치 섭취: 대량의 데이터를 한 번에 수집하여 초기 시스템 구축에 활용하는 방식.
- 스트리밍 섭취: 실시간으로 발생하는 데이터를 지속적으로 수집하여 처리하는 방식.

핵심 원리: 데이터 섭취는 다양한 형태의 데이터를 효율적으로 수집하여 저장하고, 검색 및 분석을 위해 필요한 전처리 과정을 거쳐 시스템에 통합하는 것을 목표로 한다. 이 과정에서 데이터의 임베딩 생성과 같은 준비 작업이 포함될 수 있다.

관련 도구: [[FastAPI]], [[SQLite]], [[Ollama]], [[벡터 임베딩]], [[LLM]]
