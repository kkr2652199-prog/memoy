# 프리트레인된 인공지능 에이전트

## 설명

프리트레인된 인공지능 에이전트은(는) 특정 분야의 전문가 지식 또는 광범위한 기초 지식을 사전에 주입받아 초기 학습을 마친 후, 사용자의 맞춤형 지식 학습을 통해 특정 작업을 수행하도록 설계된 인공지능 시스템.

유형/분류:
- 클라우드 기반 에이전트: 대규모 데이터셋으로 사전 학습되어 범용적인 기능을 제공하는 인공지능 에이전트.
- 로컬 맞춤형 에이전트: 사용자가 직접 로컬 환경에서 LLM을 구축하고 특정 지식을 학습시켜 차별화된 성능을 내도록 최적화된 에이전트.

핵심 원리: 초기에는 전문가 지식을 주입하여 기초를 다지고, 이후 사용자가 로컬 환경에서 맞춤형 LLM을 구축하고 특정 지식을 학습시켜 고유한 성능을 발휘하도록 한다.
