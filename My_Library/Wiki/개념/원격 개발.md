# 원격 개발

## 설명

원격 개발은(는) 개발자가 로컬 컴퓨터가 아닌 원격 서버나 클라우드 환경에서 소프트웨어 개발 작업을 수행하는 방식이다.

유형/분류:
- 클라이언트-서버 기반: 로컬 개발 환경에서 원격 서버에 접속하여 개발 작업을 수행하는 방식으로, [[VS Code]]의 [[Remote SSH]] 기능 등이 이에 해당한다.
- 클라우드 기반: 클라우드 서비스 제공자가 제공하는 개발 환경을 활용하여 웹 브라우저나 전용 클라이언트를 통해 개발하는 방식이다.
- 가상 머신 기반: 로컬 또는 원격 서버에 설치된 가상 머신 내에서 개발 환경을 구축하고 사용하는 방식이다.

핵심 원리: 로컬 장치의 개발 도구(예: 코드 에디터)가 [[SSH]]와 같은 프로토콜을 통해 원격 서버에 접속하여, 실제 코드 실행, 파일 관리, 디버깅 등의 작업을 원격 서버에서 처리하고 그 결과를 로컬 화면에 표시한다. 이를 통해 개발자는 강력한 원격 서버의 자원을 활용하면서도 익숙한 로컬 개발 환경의 편의성을 유지할 수 있다.

관련 도구: [[VS Code]], [[SSH]], [[Termius]]
