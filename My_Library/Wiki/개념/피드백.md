# 피드백

## 설명

피드백은(는) 특정 행동이나 과정의 결과에 대한 정보를 받아들여 향후 행동이나 시스템을 조정하고 개선하는 과정이다.

유형/분류:
- 환경적 피드백: 외부 환경(예: 경제적 불안감)이 개인의 행동 변화를 유도하는 정보.
- 성과 개선 피드백: 특정 목표 달성을 위해 행동의 효율성을 평가하고 조정하는 정보.
- 자기 성찰 피드백: 개인의 인식이나 태도를 변화시키는 내면의 정보.

핵심 원리: 피드백은(는) 외부 환경이나 내부 상태에서 발생하는 정보를 바탕으로 현재의 상태를 평가하고, 목표 달성을 위해 필요한 행동 변화나 학습을 유도하여 시스템이나 개인의 발전을 촉진한다.
