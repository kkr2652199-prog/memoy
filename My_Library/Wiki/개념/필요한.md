# 필요한

## 설명

필요한은(는) 특정 목적 달성, 기능 수행 또는 문제 해결을 위해 반드시 존재하거나 충족되어야 하는 필수적인 조건, 자원 또는 정보를 의미한다.

유형/분류:
- 기능적 필요성: 특정 시스템이나 주체가 주어진 기능을 올바르게 수행하기 위해 요구되는 요소.
- 목표 지향적 필요성: 설정된 목표를 성공적으로 달성하기 위해 필수적인 자원, 지식 또는 행동.
- 관계적 필요성: 특정 관계를 효과적으로 유지하고 발전시키기 위해 요구되는 역할, 역량 또는 상호작용.

핵심 원리: 필요한 것은 현재 상태와 목표 상태 사이의 격차를 식별하고, 이 격차를 해소하기 위한 필수 요소를 정의하며, 해당 요소를 확보하거나 충족시켜 원하는 결과를 도출 가능하게 한다.
