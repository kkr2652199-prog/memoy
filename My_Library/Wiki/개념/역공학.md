# 역공학

## 설명

역공학은(는) 완성된 제품이나 시스템을 분석하여 그 설계, 구조, 작동 방식, 구성 요소 등을 역으로 추론하고 이해하는 과정이다.

유형/분류:
- 소프트웨어 역공학: 프로그램의 소스 코드 없이 실행 파일을 분석하여 알고리즘, 데이터 구조, 기능 등을 파악하는 것
- 하드웨어 역공학: 전자 기기나 기계 장치를 분해하여 회로도, 부품 구성, 제조 공정 등을 알아내는 것
- 프로토콜 역공학: 통신 프로토콜의 작동 방식을 분석하여 데이터 형식, 메시지 교환 규칙 등을 이해하는 것

핵심 원리: 대상 시스템의 외부 동작을 관찰하고 내부 구조를 분해하여 숨겨진 설계 의도와 구현 세부 사항을 밝혀낸다. 이를 통해 원본 설계 정보를 재구성하거나 개선, 호환성 확보, 보안 취약점 분석 등에 활용한다.
