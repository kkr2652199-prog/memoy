# 프롬프트 생성

## 설명

프롬프트 생성은(는) 인공지능 모델, 특히 이미지 생성 모델의 특정 출력을 유도하기 위해 상세한 지시사항이나 설명을 담은 텍스트를 만드는 과정이다.

유형/분류:
- 이미지 프롬프트 생성: 시각적 결과물을 얻기 위해 이미지 생성 모델에 제공되는 설명을 만드는 과정.
- 스크립트 기반 프롬프트 생성: 대본이나 시나리오와 같은 텍스트 입력을 바탕으로 각 장면에 맞는 프롬프트를 도출하는 방식.
- 캐릭터 중심 프롬프트 생성: 특정 캐릭터의 고유한 특징과 역할을 상세히 묘사하여 프롬프트에 반영하는 기법.

핵심 원리: 입력된 텍스트 대본과 캐릭터 이미지를 분석하여 주요 요소를 추출하고, 이를 기반으로 캐릭터의 특징, 역할, 배경 시각화 등 다양한 구성 요소를 조합하여 상세한 프롬프트를 생성한다. 모든 프롬프트 내용은 영어로 작성된다.
