# 오프라인 모델

## 설명

오프라인 모델은(는) 인터넷 연결이나 외부 클라우드 서비스 없이 사용자 기기 내에서 직접 작동하도록 설계된 인공지능 모델이다.

유형/분류:
- 로컬 실행 모델: 외부 서버와의 통신 없이 사용자 기기 내에서 직접 AI 연산을 수행하는 모델.
- 독립형 모델: 인터넷 연결이나 외부 API 호출 없이 자체적으로 모든 기능을 수행할 수 있는 모델.
- 경량화 모델: 평균적인 개인용 컴퓨터에서도 효율적으로 실행될 수 있도록 최적화된 모델.
- 다중 모달 모델: 텍스트뿐만 아니라 이미지와 같은 다양한 유형의 데이터를 처리하고 분석할 수 있는 모델.

핵심 원리: 오프라인 모델은 인터넷 연결 없이 사용자 기기의 하드웨어 자원을 활용하여 AI 연산을 직접 수행하며, 외부 서버에 대한 의존성을 제거하여 지연 시간을 줄이고 데이터 프라이버시를 강화한다.
