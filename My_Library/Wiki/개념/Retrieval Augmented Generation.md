# Retrieval Augmented Generation

## 설명

Retrieval Augmented Generation은(는) 대규모 언어 모델(LLM)이 외부 지식 기반에서 관련 정보를 검색하여 응답 생성 능력을 향상시키는 기술이다.

유형/분류:
- 지식 기반 RAG: 구조화되거나 비구조화된 외부 지식 기반(예: 위키)을 활용하여 정보를 검색하는 방식
- 개인화 RAG: 개인의 데이터, 경험, 사고 과정 등을 기록한 지식 기반을 활용하여 사용자 맞춤형 응답을 생성하는 방식
- 자율 유지보수 RAG: AI가 직접 외부 지식 기반을 생성, 업데이트 및 검증하여 지식의 정확성과 최신성을 유지하는 방식

핵심 원리: 대규모 언어 모델(LLM)의 휘발성 기억 문제를 해결하기 위해, AI가 외부 지식 위키를 직접 생성하고 유지보수하며, 질문에 대한 답변을 생성하기 전에 이 위키에서 관련 정보를 검색하여 활용한다.
