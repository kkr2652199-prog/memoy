# 파일 구조

## 설명

파일 구조은(는) 저장 장치에 파일과 디렉토리를 논리적으로 배열하고 관리하는 체계적인 방식이다.

유형/분류:
- 계층적 파일 구조: 디렉토리와 서브디렉토리를 사용하여 파일을 트리 형태로 구성하는 가장 일반적인 방식이다.
- 플랫 파일 구조: 모든 파일이 단일 디렉토리에 저장되며, 서브디렉토리가 없는 단순한 구조이다.
- 데이터베이스 기반 파일 구조: 파일 시스템이 아닌 데이터베이스 내에서 파일의 메타데이터와 내용을 관리하는 방식이다.

핵심 원리: 파일 구조는 데이터를 효율적으로 저장하고 검색하며 관리할 수 있도록 논리적인 조직을 제공한다. 이를 통해 사용자 및 운영 체제가 특정 파일을 신속하게 찾고 데이터 무결성을 유지할 수 있다.
