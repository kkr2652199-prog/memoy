# 장기 기억

## 설명

장기 기억은(는) 정보를 장기간 저장하고 필요할 때 인출하여 활용하는 인지 능력 또는 시스템의 기능이다.

유형/분류:
- 대화 맥락 기억: 인공지능 시스템이 특정 대화 세션 내에서 이전 발언과 정보를 유지하여 일관된 상호작용을 가능하게 하는 능력.
- 모델 학습 기억: 대규모 데이터셋을 통해 학습된 방대한 지식과 패턴으로, 인공지능 모델의 기본적인 이해와 추론 능력을 구성하는 장기적인 정보 저장 형태.

핵심 원리: 정보를 부호화하여 저장하고, 필요할 때 인출하는 과정으로, 인간의 뇌에서는 시냅스 연결의 강화 및 재구성을 통해, 인공지능 시스템에서는 학습된 모델 가중치와 외부 저장 메커니즘을 통해 구현된다.
