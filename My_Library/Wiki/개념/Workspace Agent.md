# Workspace Agent

## 설명

Workspace Agent은(는) 여러 AI 에이전트의 작업을 효율적으로 관리하고 오케스트레이션하는 도구이다.

유형/분류:
- AI 에이전트 오케스트레이션: 여러 AI 에이전트의 작업을 조정하고 관리한다.
- 병렬 작업 관리: 여러 에이전트 작업을 동시에 실행하고 충돌을 방지한다.
- 클라우드 실행 환경: 로컬 환경에 구애받지 않고 작업을 지속적으로 수행할 수 있도록 지원한다.

핵심 원리: Workspace Agent는 여러 AI 에이전트의 작업을 중앙에서 관제탑처럼 관리하며, 워크트리를 통해 병렬 실행 시 발생할 수 있는 충돌을 방지한다. 또한 클라우드 서버에서 작업을 실행하여 로컬 컴퓨터의 상태와 무관하게 작업을 지속적으로 수행할 수 있도록 한다.
