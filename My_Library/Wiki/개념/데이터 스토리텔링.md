# 데이터 스토리텔링

## 설명

데이터 스토리텔링은(는) 데이터를 기반으로 통찰력을 발견하고, 이를 설득력 있는 서사 구조와 시각화를 통해 청중에게 효과적으로 전달하는 과정.

유형/분류:
- 설명적 스토리텔링: 데이터가 보여주는 현상이나 사실을 명확하게 설명하여 이해를 돕는다.
- 탐색적 스토리텔링: 데이터 내의 패턴, 추세 또는 관계를 청중이 스스로 탐색하고 발견하도록 유도한다.
- 설득적 스토리텔링: 특정 행동이나 의사결정을 유도하기 위해 데이터를 활용하여 주장을 뒷받침한다.
- 예측적 스토리텔링: 과거 데이터를 기반으로 미래의 가능성이나 추세를 예측하고 제시한다.

핵심 원리: 데이터 분석 결과와 인간적인 서사를 결합하여 복잡한 정보를 이해하기 쉽게 만들고, 청중의 공감을 얻어 메시지의 영향력을 극대화한다.
