# 지식 큐레이션

## 설명

지식 큐레이션은 다양한 출처에서 가치 있는 정보와 지식을 체계적으로 식별, 수집, 평가, 조직화하여 특정 대상이나 목적에 맞게 접근 가능하고 유용하도록 제시하는 과정이다.

유형/분류:
- 콘텐츠 큐레이션: 특정 주제에 대한 외부 콘텐츠를 선별하고 공유하여 가치를 더하는 활동.
- 데이터 큐레이션: 데이터의 품질, 접근성, 재사용성을 보장하기 위해 데이터를 관리하고 유지보수하는 과정.
- 개인 지식 큐레이션: 개인이 자신의 학습, 연구, 업무를 위해 정보를 수집, 정리, 연결하여 개인 지식 기반을 구축하는 활동.

핵심 원리: 지식 큐레이션은 신뢰할 수 있는 정보를 선별하고, 맥락에 맞게 재구성하며, 접근성을 높여 사용자가 필요한 지식을 쉽게 발견하고 활용할 수 있도록 돕는다. 이는 정보 과부하를 줄이고 지식의 가치를 극대화하는 데 중점을 둔다.
