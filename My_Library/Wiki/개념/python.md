# python

## 설명

python은(는) 범용 고수준 프로그래밍 언어로, 간결한 문법과 다양한 라이브러리를 통해 웹 개발, 데이터 과학, 인공지능 등 여러 분야에서 활용됩니다.

유형/분류:
- 프로세스 관리: 실행 중인 파이썬 애플리케이션을 제어하고 종료하는 방법.
- 애플리케이션 실행: 특정 모듈을 지정하여 파이썬 프로그램을 시작하는 방식.
- 명령줄 인터페이스: 운영 체제의 셸을 통해 파이썬 애플리케이션을 조작하는 방식.

핵심 원리: 주어진 명령어들은 운영 체제의 명령줄을 활용하여 특정 파이썬 애플리케이션의 실행을 강제로 중단하고, 작업 디렉토리를 변경한 후, 해당 애플리케이션을 다시 시작하는 일련의 과정을 가능하게 한다. 이는 파이썬 기반 서비스의 운영 및 관리에 필수적인 절차이다.
