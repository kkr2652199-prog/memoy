# BM25+벡터 하이브리드 검색

## 설명

BM25+벡터 하이브리드 검색은(는) 키워드 기반의 통계적 검색(BM25)과 의미 기반의 벡터 유사도 검색을 결합하여 검색 관련성을 향상시키는 고급 정보 검색 방식이다.

유형/분류:
- 스코어 융합: BM25 점수와 벡터 유사도 점수를 통합하여 최종 순위를 결정.
- 리랭킹: 한 가지 방법으로 1차 검색 후 다른 방법으로 결과를 재순위화.
- 캐스케이딩: 한 가지 검색 방법의 결과를 다른 검색 방법의 입력으로 활용하여 점진적으로 개선.
- 병렬 검색: 두 가지 검색을 동시에 수행하고 그 결과를 병합.

핵심 원리: 단어의 정확한 일치(BM25)와 의미적 유사성(벡터)을 모두 고려하여 사용자의 질의 의도를 더 정확하게 파악하고, 키워드 매칭의 한계와 순수 벡터 검색의 단점을 보완하여 더 포괄적이고 관련성 높은 결과를 제공한다.

관련 도구: [[Elasticsearch]], [[OpenSearch]], [[Pinecone]], [[Weaviate]]
