# 스토리텔링 기법

## 설명

스토리텔링 기법은 정보와 서사를 효과적으로 구성하여 청중의 몰입을 유도하고 메시지를 전달하는 커뮤니케이션 전략이다.

유형/분류:
- 콘텐츠 기획 기반: 특정 주제나 목적에 맞춰 서사 구조를 설계하는 방식
- 정보 구조화 중심: 복잡한 정보를 이해하기 쉽게 논리적으로 배열하는 기법
- 서사 전개 활용: 시간적 흐름이나 인과 관계를 통해 이야기를 흥미롭게 이끌어가는 방법
- 야사/정보 결합형: 흥미로운 일화나 비공식적인 이야기를 정보 전달과 융합하는 방식

핵심 원리: 청중의 공감을 얻고 기억에 남는 메시지를 전달하기 위해, 사실적 정보와 감성적 서사를 유기적으로 결합하여 이야기를 구성한다. 이는 정보의 단순한 나열을 넘어 몰입감 있는 경험을 제공하는 데 중점을 둔다.
