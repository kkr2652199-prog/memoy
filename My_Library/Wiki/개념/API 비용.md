# API 비용

## 설명

지식 관리 전문가로서, 제공해주신 '새 자료'의 내용이 'API 비용' 개념과 직접적인 관련이 없어 해당 자료를 기반으로 위키 개요를 갱신할 수 없습니다. '새 자료'는 Gemini의 기능 활용법에 대한 내용이며, 'API 비용'에 대한 정보는 포함하고 있지 않습니다.

정확하고 신뢰할 수 있는 지식 관리를 위해, 'API 비용' 개념을 갱신하려면 해당 주제와 관련된 자료를 제공해주시거나, 제공된 자료(ID: 144)를 활용하여 'Gemini 기능 활용법'과 같은 적절한 개념의 위키 개요를 갱신하도록 지시해주시기 바랍니다.
