# 와이어 프레임

## 설명

와이어 프레임은(는) 웹사이트나 애플리케이션의 구조와 레이아웃을 시각적으로 표현하여 콘텐츠의 배치와 기능적 흐름을 정의하는 초기 단계의 설계 도구이다.

유형/분류:
- 저충실도 와이어프레임: 종이나 화이트보드에 손으로 스케치하여 기본적인 레이아웃과 페이지 흐름을 빠르게 구상하는 방식.
- 중간충실도 와이어프레임: 디지털 도구를 사용하여 더 상세한 레이아웃과 요소 배치, 기본적인 상호작용을 표현하는 방식.
- 고충실도 와이어프레임: 실제 사용자 인터페이스에 가깝게 상세한 요소와 텍스트를 포함하며, 최종 디자인에 근접한 수준으로 제작되는 방식.

핵심 원리: 와이어프레임은 시각적 디자인 요소(색상, 폰트 등)를 배제하고 정보의 우선순위, 사용자 인터페이스의 구성, 기능적 상호작용에 집중하여 디자인 초기 단계에서 구조적 문제를 식별하고 개선하는 데 활용된다. 이를 통해 개발 전 사용자 경험(UX)을 효율적으로 검토하고 피드백을 반영할 수 있다.
