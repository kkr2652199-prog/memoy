# 이미지 압축

## 설명

이미지 압축은 디지털 이미지의 파일 크기를 줄여 저장 공간을 절약하고 전송 속도를 높이는 기술이다.

유형/분류:
- 손실 압축: 원본 이미지의 일부 데이터를 영구적으로 제거하여 높은 압축률을 달성하지만, 품질 저하가 발생할 수 있다.
- 무손실 압축: 이미지 데이터를 손실 없이 압축하여 원본 이미지를 완벽하게 복원할 수 있으나, 압축률은 손실 압축보다 낮다.

핵심 원리: 이미지 내의 통계적 중복성이나 인간 시각 시스템의 인지적 한계를 활용하여 불필요하거나 덜 중요한 정보를 제거하거나 더 효율적인 방식으로 인코딩한다. 이를 통해 데이터 양을 줄여 파일 크기를 최소화한다.
