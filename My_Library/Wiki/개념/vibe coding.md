# vibe coding

## 설명

'Vibe coding'은 개발자가 AI를 사용하여 코드를 작성할 때 발생하는 사고 부족 문제를 지적하는 개념이다. 이 개념은 AI의 코드 생성 능력에 의존하는 것이 아니라, 개발자가 문제를 깊이 이해하고 분석하는 것이 중요하다는 점을 강조한다. 예를 들어, 호텔 예약 플랫폼을 개발할 때 AI가 생성한 코드가 실제 사용 시 서버에 부하를 주는 문제를 일으킬 수 있다. 따라서 개발자는 AI에 의존하기보다는 자신의 사고 과정을 강화해야 한다.
