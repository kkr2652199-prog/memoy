# AB 테스트

## 설명

AB 테스트은(는) 두 가지 이상의 변형(A와 B)을 비교하여 어떤 변형이 특정 목표에 더 나은 성과를 내는지 통계적으로 검증하는 실험 방법이다.

유형/분류:
- A/B/n 테스트: 두 개 이상의 변형을 동시에 비교하여 최적의 버전을 찾는 방식.
- 다변량 테스트: 여러 요소의 조합이 결과에 미치는 영향을 동시에 분석하는 방식.
- 분할 URL 테스트: 다른 URL에 위치한 페이지 버전을 비교하여 최적의 버전을 찾는 방식.

핵심 원리: 사용자 그룹을 무작위로 두 개 이상의 집단으로 나누어 각 집단에 다른 버전의 요소를 노출시킨 후, 사전에 정의된 지표를 통해 각 버전의 성과를 측정하고 통계적 유의미성을 분석하여 최적의 버전을 결정한다.
