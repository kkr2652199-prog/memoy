# 창분할

## 설명

창분할은(는) 단일 터미널 화면이나 애플리케이션 작업 영역을 여러 개의 독립적인 영역(패인)으로 나누어 동시에 여러 작업을 수행할 수 있도록 하는 기능이다.

유형/분류:
- 터미널 멀티플렉서 방식: [[tmux]]와 같이 터미널 세션을 관리하며 창과 패인을 분할하는 방식.
- 터미널 에뮬레이터 내장 방식: [[iTerm2]]와 같이 터미널 에뮬레이터 자체에서 제공하는 화면 분할 기능.
- 방향별 분할: 화면을 수직(좌우) 또는 수평(상하)으로 나누어 작업 영역을 구성하는 방식.

핵심 원리: 단일 화면 공간 내에서 여러 개의 독립적인 명령줄 인터페이스나 애플리케이션 뷰를 동시에 보고 상호작용할 수 있도록 하여 사용자의 생산성을 향상시킨다. 이는 각 분할 영역이 독립적인 프로세스를 실행하거나 다른 정보를 표시할 수 있게 한다.

관련 도구: [[tmux]], [[iTerm2]]
