# 외부화

## 설명

외부화은(는) 내부에 있는 암묵적 지식이나 의도를 명확하고 구체적인 형태로 표현하여 외부 시스템이나 타인이 이해하고 활용할 수 있도록 하는 과정이다.

유형/분류:
- 명시적 지시 외부화: 구체적인 요구사항과 제약 조건을 명확히 기술하여 AI가 수행할 작업을 정의하는 방식.
- 맥락 정보 외부화: AI가 작업을 올바르게 이해하고 처리하도록 돕는 배경 정보나 관련 용어를 제공하는 방식.
- 의도 구체화 외부화: 모호한 요청을 피하고 원하는 결과물을 명확히 상상하고 표현하여 AI의 오해를 줄이는 방식.

핵심 원리: 내부의 추상적인 아이디어나 요구사항을 외부 시스템이 처리할 수 있는 구체적이고 명확한 형태로 전환하는 것을 목표로 한다. 이는 정보의 모호성을 제거하고, 필요한 모든 맥락과 세부 사항을 명시함으로써 효율적인 상호작용과 정확한 결과 도출을 가능하게 한다.
