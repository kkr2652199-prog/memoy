# 데이터 구조

## 설명

데이터 구조은(는) 컴퓨터 과학에서 효율적인 접근과 수정을 위해 데이터를 조직, 관리 및 저장하는 방식이다.

유형/분류:
- 선형 데이터 구조: 데이터를 순차적으로 저장하며, 배열, 연결 리스트, 스택, 큐 등이 있다.
- 비선형 데이터 구조: 데이터 간의 복잡한 관계를 표현하며, 트리, 그래프, 해시 테이블 등이 있다.
- 추상 데이터 타입: 특정 연산을 정의하고 구현 방식은 숨기는 데이터 타입으로, 스택이나 큐 등이 이에 해당한다.

핵심 원리: 데이터 구조는 특정 연산의 효율성을 최적화하고, 데이터 간의 관계를 명확히 하여 프로그램의 성능을 향상시킨다. 이는 데이터를 저장하고 검색하는 방식에 따라 알고리즘의 복잡도에 직접적인 영향을 미친다.
