# 로컬 LLM

## 설명

로컬 LLM은(는) 사용자의 개인 컴퓨터나 로컬 서버에서 직접 실행되어, 클라우드 기반 AI와 달리 맞춤형 지식 학습 및 제어가 가능한 대규모 언어 모델이다.

유형/분류:
- 개인 맞춤형 에이전트 구축용: 사용자가 특정 목적에 맞춰 지식을 주입하고 성능을 최적화하는 AI 에이전트를 개발하는 데 활용된다.
- 오픈소스 모델 기반: [[젬마 4 E2B]]와 같이 공개된 모델을 로컬 환경에 설치하여 활용하는 형태이다.
- 개발 및 학습 환경: AI 에이전트의 기초 지식과 학습 능력을 평가하고 훈련하는 플랫폼에서 사용된다.

핵심 원리: 로컬 LLM은 외부 서버 의존 없이 사용자의 하드웨어에서 직접 모델을 구동하며, 이를 통해 데이터 프라이버시를 강화하고 특정 도메인 지식을 직접 학습시켜 차별화된 성능을 구현한다.

관련 도구: [[올라마]], [[LM 스튜디오]], [[젬마 4 E2B]]
