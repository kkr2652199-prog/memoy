# 지식그래프

## 설명

지식그래프는 다양한 형태의 지식 자원을 구조화하고 의미적으로 연결하여 효율적인 검색과 새로운 통찰을 도출할 수 있도록 돕는 지식 관리 체계이다.

유형/분류:
- 개인 지식 관리: 개인의 자료를 체계적으로 수집, 저장, 검색하며 지식 자산을 구축하는 시스템.
- 임베딩 기반 검색: 자료의 의미를 벡터 공간에 표현하여 유사성을 기반으로 정보를 탐색하는 방식.
- 하이브리드 검색: 키워드 기반 검색(BM25)과 벡터 유사성 검색을 결합하여 검색 정확도와 관련성을 높이는 방식.

핵심 원리: 지식그래프는 자료의 의미를 임베딩으로 변환하여 벡터 공간에 저장하고, 이를 기반으로 BM25와 벡터 검색을 결합한 하이브리드 방식으로 정보를 탐색하고 연결한다.

관련 도구: [[FastAPI]], [[SQLite]], [[Ollama]], [[D3.js]]
