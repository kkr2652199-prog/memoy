# 트랜스크립트

## 설명

트랜스크립트은(는) 웹 스크래핑 과정을 통해 동적 웹사이트에서 추출된 텍스트 형태의 데이터이다.

유형/분류:
- 동적 웹 콘텐츠 트랜스크립트: JavaScript 렌더링이 필요한 동적 웹사이트(네이버, 유튜브 등)에서 데이터를 추출하는 과정 및 결과.
- 인코딩 처리 트랜스크립트: 다양한 문자 인코딩 문제를 해결하여 정확한 텍스트를 확보하는 과정 및 결과.
- 구조화된 데이터 트랜스크립트: 추출된 비정형 데이터를 구조화하고 다른 데이터와 연결하는 과정 및 결과.

핵심 원리: 동적 웹사이트의 JavaScript 렌더링을 처리하고, 다양한 문자 인코딩 문제를 해결하여 정확한 텍스트 데이터를 추출하는 것을 목표로 한다. 또한, 추출된 데이터의 구조화 및 다른 정보와의 연결성을 확보하는 원리를 따른다.
