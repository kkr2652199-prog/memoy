# 인포그래픽

## 설명

인포그래픽은(는) 정보, 데이터 또는 지식을 시각적으로 표현하여 빠르고 명확하게 전달하는 그래픽 표현 방식이다.

유형/분류:
- 통계 인포그래픽: 숫자 데이터와 통계를 차트, 그래프 등으로 시각화하여 복잡한 정보를 이해하기 쉽게 전달한다.
- 프로세스 인포그래픽: 특정 과정이나 절차의 단계를 순서대로 보여주어 복잡한 흐름을 직관적으로 이해하도록 돕는다.
- 타임라인 인포그래픽: 시간의 흐름에 따른 사건이나 변화를 시각적으로 정리하여 보여주며, 역사나 프로젝트 진행 상황 등을 효과적으로 나타낸다.

핵심 원리: 시각적 요소를 통해 복잡한 데이터를 단순화하고, 패턴과 트렌드를 직관적으로 파악할 수 있도록 돕는다. 텍스트와 이미지, 차트 등을 결합하여 정보 전달의 효율성과 기억력을 높이는 데 중점을 둔다.
