# 지식 베이스

## 설명

지식 베이스는 특정 주제나 분야에 대한 정보를 체계적으로 수집, 저장, 관리하여 필요할 때 검색하고 활용할 수 있도록 구성된 저장소이다.

유형/분류:
- 개인 지식 베이스: 개인의 아이디어와 정보를 체계적으로 정리하고 관리하는 시스템으로, [[LLM 위키]]와 같이 AI의 도움을 받아 지식 네트워크를 확장한다.
- 조직 지식 베이스: 기업이나 단체 내에서 공유되는 정보, 절차, FAQ 등을 저장하여 팀원들이 쉽게 접근하고 활용하도록 돕는 중앙 집중식 저장소.
- AI 기반 지식 베이스: 인공지능 기술을 활용하여 정보 수집, 분류, 검색, 요약 등을 자동화하고 지식 간의 새로운 연결고리를 발견하는 시스템.

핵심 원리: 사용자가 수집한 정보를 저장하고, AI가 이를 체계적으로 정리하여 위키 형태로 가꾸며, '수집', '질문', '정리'와 같은 명령어를 통해 지식을 활용하고 확장한다.
