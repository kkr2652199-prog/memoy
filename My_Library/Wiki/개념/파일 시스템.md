# 파일 시스템

## 설명

파일 시스템은 운영 체제가 저장 장치에 데이터를 효율적으로 저장하고 검색하며 관리하기 위해 사용하는 논리적인 구조 및 규칙의 집합이다.

유형/분류:
- 로컬 파일 시스템: 단일 물리적 저장 장치 내에서 파일과 디렉토리를 관리하며, 운영 체제에 직접 연결된 저장 장치에 사용된다.
- 네트워크 파일 시스템: 네트워크를 통해 원격 컴퓨터의 파일에 접근하고 이를 마치 로컬 파일처럼 사용할 수 있도록 지원한다.
- 분산 파일 시스템: 여러 서버에 분산된 데이터를 단일 논리적 공간처럼 관리하여 대규모 데이터 저장 및 처리에 활용된다.

핵심 원리: 파일 시스템은 데이터를 파일과 디렉토리 형태로 계층적으로 조직하고, 각 파일의 메타데이터(크기, 생성일 등) 및 접근 권한을 관리하며, 저장 공간을 효율적으로 할당하고 회수하는 방식으로 작동한다.
