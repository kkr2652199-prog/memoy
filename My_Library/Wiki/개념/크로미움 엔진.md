# 크로미움 엔진

## 설명

크로미움 엔진은(는) 구글이 개발한 오픈 소스 웹 브라우저 엔진으로, 웹 콘텐츠를 렌더링하고 웹 애플리케이션을 실행하는 핵심 구성 요소이다.

유형/분류:
- 웹 브라우저 엔진: 웹 페이지의 HTML, CSS, JavaScript 등을 해석하여 화면에 표시하는 소프트웨어 구성 요소.
- 오픈 소스 프로젝트: 누구나 소스 코드를 열람, 수정, 배포할 수 있도록 공개된 소프트웨어 개발 프로젝트.
- 렌더링 엔진: 웹 표준에 따라 웹 콘텐츠를 시각적으로 표현하는 역할을 수행.
- 애플리케이션 프레임워크: 웹 기반 애플리케이션의 개발 및 실행 환경을 제공.

핵심 원리: HTML 파서가 웹 페이지의 구조를 분석하고, CSS 엔진이 스타일을 적용하며, JavaScript 엔진(V8)이 스크립트를 실행하여 최종적으로 렌더링 트리를 구성하고 화면에 그린다. 이 과정에서 웹 표준을 준수하여 다양한 웹 기술을 지원한다.

관련 도구: [[Google Chrome]], [[Microsoft Edge]], [[Opera]], [[Brave Browser]]
