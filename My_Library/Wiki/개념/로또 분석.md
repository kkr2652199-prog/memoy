# 로또 분석

## 설명

로또 분석은 로또 번호 예측을 위한 다양한 방법론과 접근 방식을 포함하는 분야이다. 로또PD는 이번 주 로또 번호 예측을 AI와 여러 채널의 선택 번호를 비교하여 분석하였다. 지난주 1222회 결과에서 C 채널이 높은 적중률을 보였으며, 이번 주 1223회에서는 특정 번호들이 선택되지 않아 재외수로 고려해야 한다고 언급했다. 각 구간별 예상 번호와 그에 대한 분석도 제공되었으며, 데이터 기반의 접근을 강조하였다. 새 자료에서는 로또 1195회차 당첨번호를 분석하고, 과거 40회차 데이터를 바탕으로 패턴과 규칙을 찾아내어 1196회차 예상 번호를 도출하였다.
