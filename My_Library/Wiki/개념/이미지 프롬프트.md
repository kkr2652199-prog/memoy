# 이미지 프롬프트

## 설명

이미지 프롬프트은(는) 텍스트 설명을 기반으로 인공지능이 시각적 이미지를 생성하도록 지시하는 입력문이다.

유형/분류:
- 콘텐츠 제작 프롬프트: 영상 콘텐츠 등 특정 목적을 위한 시각 자료 생성에 사용되는 지시문.
- AI 이미지 생성 지시: 인공지능 모델에게 원하는 이미지의 특징, 스타일 등을 명시하여 결과물을 유도하는 텍스트 입력.
- 시각화 보조 프롬프트: 텍스트 기반의 원고나 아이디어를 구체적인 시각적 형태로 변환하는 데 활용되는 지시.

핵심 원리: 이미지 프롬프트는 사용자가 원하는 이미지의 특징, 스타일, 내용 등을 텍스트로 상세히 기술하여 인공지능 모델이 이를 해석하고 해당 조건에 부합하는 시각적 결과물을 생성하도록 유도한다. 이는 텍스트 원고를 시각적 요소로 변환하는 핵심 단계로 기능한다.
