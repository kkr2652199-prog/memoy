# BM25 알고리즘

## 설명

BM25 알고리즘은(는) 정보 검색 시스템에서 문서와 질의 간의 관련성을 점수화하여 순위를 매기는 데 사용되는 확률적 검색 모델이다.

유형/분류:
- 확률적 검색 모델: 문서 내 단어의 빈도와 역문서 빈도 등을 통계적으로 분석하여 관련성을 추정.
- TF-IDF 기반 확장: 기존 TF-IDF 모델의 한계를 보완하여 단어 빈도 포화 및 문서 길이 정규화 문제를 개선.
- 랭킹 함수: 검색 질의에 대한 문서의 적합도를 계산하여 검색 결과의 순서를 결정하는 핵심 요소.

핵심 원리: 질의어가 문서에 나타나는 빈도, 질의어가 포함된 문서의 수, 문서의 길이 등을 복합적으로 고려하여 관련성 점수를 계산한다. 특히, 단어 빈도의 과도한 증가가 점수에 미치는 영향을 완화하고, 문서 길이에 따른 불이익을 조정하여 검색 정확도를 높인다.

관련 도구: [[Elasticsearch]], [[Apache Lucene]], [[Solr]], [[MindVault]]
