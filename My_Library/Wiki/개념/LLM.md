# LLM

## 설명

LLM은(는) 대량의 텍스트 데이터를 학습하여 자연어를 이해하고 생성하는 인공지능 모델이다.

유형/분류:
- 지식 생성: 원본 정보를 이해하고 자동으로 위키 페이지를 생성한다.
- 정보 처리: 다양한 형식의 원본 자료를 읽고 분석하여 지식을 추출한다.
- 검색 증강 생성 (RAG): 로컬 저장소의 정보를 참조하여 응답을 생성하는 데 활용된다.

핵심 원리: LLM은 대량의 원본 자료를 분석하여 내용을 이해하고, 이를 바탕으로 새로운 지식 페이지를 자동으로 생성하며 기존 정보를 직접 수정하지 않고 참조하여 지식 구조를 구축한다.
