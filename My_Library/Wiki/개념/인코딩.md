# 인코딩

## 설명

인코딩은 정보(데이터)를 특정 형식이나 표준에 맞춰 변환하는 과정으로, 특히 웹 스크래핑에서 문자 집합 문제를 해결하는 데 사용된다.

유형/분류:
- 데이터 변환: 정보를 특정 형식으로 변환하는 일반적인 과정.
- 문자 집합 처리: 텍스트 데이터의 올바른 표현을 위한 문자 인코딩 방식 적용.
- 자동 감지: 알 수 없는 인코딩을 자동으로 식별하는 기술.

핵심 원리: 웹 페이지의 메타 정보나 콘텐츠를 분석하여 실제 사용된 문자 인코딩을 파악하고, 이를 기반으로 데이터를 정확하게 디코딩하여 올바른 텍스트를 얻는 것이 핵심 원리이다.
