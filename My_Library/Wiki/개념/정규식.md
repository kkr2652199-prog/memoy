# 정규식

## 설명

정규식은(는) 특정 문자열 패턴을 정의하고 검색, 추출, 대체 등 문자열 조작을 수행하는 데 사용되는 강력한 표현식 언어이다.

유형/분류:
- 검색 및 매칭: 텍스트 내에서 특정 패턴과 일치하는 부분을 찾는다.
- 데이터 추출: 복잡한 문자열에서 원하는 정보를 구조적으로 뽑아낸다.
- 유효성 검사: 입력된 데이터가 미리 정의된 형식 규칙을 따르는지 확인한다.
- 문자열 대체: 패턴에 일치하는 부분을 다른 문자열로 변경하거나 제거한다.

핵심 원리: 정규식은 메타문자와 리터럴 문자의 조합으로 구성된 패턴을 사용하여 대상 문자열을 스캔하고, 패턴과 일치하는 부분을 찾아내어 다양한 텍스트 처리 작업을 수행한다.

관련 도구: [[grep]], [[sed]], [[awk]], [[Perl]], [[Python]], [[JavaScript]], [[PHP]], [[Java]]
