# 시청자 패턴

## 설명

시청자 패턴은 특정 콘텐츠가 높은 조회수나 참여도를 얻는 데 기여하는 시청자들의 행동, 선호도, 그리고 콘텐츠 소비 경향에서 나타나는 반복적이거나 예측 가능한 특징이다.

유형/분류:
- 급상승 패턴: 조회수가 급격히 증가하는 영상에서 발견되는 성공적인 콘텐츠의 특징이나 트렌드.
- 콘텐츠 소비 패턴: 시청자들이 특정 유형의 콘텐츠를 소비하는 방식, 선호하는 형식, 그리고 참여도에 대한 전반적인 경향.
- 키워드 기반 패턴: 특정 키워드와 연관된 콘텐츠의 성공 요인이나 시청자 반응에서 나타나는 고유한 경향.

핵심 원리: [[유튜브 API]]를 활용하여 조회수가 급상승한 영상 데이터를 수집하고, 이를 분석하여 특정 키워드와 관련된 성공적인 콘텐츠의 특징을 식별한다.

관련 도구: [[AI 크래프터 스튜디오]], [[유튜브 API]]
