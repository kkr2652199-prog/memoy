# 쇼츠 대본

## 설명

쇼츠 대본은 짧은 형식의 영상 콘텐츠 제작을 위해 특정 스타일과 서사 구조에 맞춰 기획되거나 재각색된 대본이다.

유형/분류:
- 스타일 맞춤형 대본: 야담과 같이 특정 장르나 분위기에 맞춰 재각색된 대본.
- 서사 구조 기반 대본: 미스터리-추적-진실과 같은 명확한 스토리 아크를 중심으로 구성된 대본.
- 교훈적 메시지 대본: 시대적 배경과 교훈적 여운을 담아 시청자에게 메시지를 전달하는 대본.

핵심 원리: 기존 콘텐츠를 특정 스타일과 서사적 틀에 맞춰 각색하며, 짧은 시간 안에 몰입감 있는 이야기를 전달하기 위해 미스터리, 추적, 진실, 시대적 배경, 교훈적 메시지 등의 요소를 효과적으로 활용한다. 성공 사례와 패턴 분석을 통해 최적화된 스토리텔링을 구현한다.
