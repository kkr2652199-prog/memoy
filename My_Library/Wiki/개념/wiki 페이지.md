# wiki 페이지

## 설명

wiki 페이지은(는) 다양한 원본 정보를 바탕으로 LLM에 의해 자동으로 생성되어 지식 구조를 형성하는 지식 단위.

유형/분류:
- 자동 생성 지식 단위: LLM이 원본 자료를 바탕으로 자동으로 생성.
- 구조화된 정보: 태그 및 연결 기능을 통해 지식 구조를 형성.
- 로컬 RAG 구성 요소: 검색 증강 생성 시스템의 핵심 부분.

핵심 원리: Claude Code는 'raw' 폴더의 원본 정보를 읽고 이해하여 자동으로 위키 페이지를 생성하며, 옵시디언의 태그 및 연결 기능을 활용하여 지식 구조를 만듭니다.
