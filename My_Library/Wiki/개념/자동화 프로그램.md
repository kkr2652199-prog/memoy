# 자동화 프로그램

## 설명

자동화 프로그램은(는) 특정 작업을 사람의 개입 없이 자동으로 수행하도록 설계된 소프트웨어 또는 시스템이다.

유형/분류:
- 콘텐츠 생성 자동화: 텍스트 대본을 기반으로 이미지를 자동으로 생성하는 것과 같이, 다양한 형태의 콘텐츠 제작 과정을 자동화하는 프로그램.
- 브라우저 확장 프로그램: 웹 브라우저에 기능을 추가하여 특정 웹 기반 작업을 자동화하거나 효율화하는 프로그램.
- 스크립트 기반 자동화: 미리 정의된 스크립트나 규칙에 따라 작업을 순차적으로 실행하여 반복적인 프로세스를 자동화하는 프로그램.

핵심 원리: 미리 작성된 텍스트 대본을 입력받아 이를 기반으로 인물 및 배경 이미지를 자동으로 생성하며, 사용자는 소스 코드 수정을 통해 작업 속도를 조절할 수 있다.
