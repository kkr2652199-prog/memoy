# 데이터 수집

## 설명

데이터 수집은 특정 목적을 위해 다양한 출처로부터 정보를 체계적으로 모으는 과정이다.

유형/분류:
- LLM 기반 지식 수집: 대형 언어 모델(LLM)을 활용하여 텍스트, PDF 등 다양한 형태의 데이터를 분류하고 연결하여 지식 기반을 구축하는 방식.
- 비정형 데이터 수집: 텍스트 파일, PDF와 같이 구조화되지 않은 형태의 데이터를 수집하는 과정.
- 자동화된 연구 데이터 수집: AI 도구를 사용하여 수집된 데이터를 자동으로 처리하고 연결하여 연구 효율성을 높이는 방법.

핵심 원리: 대형 언어 모델(LLM)이 텍스트 파일, PDF 등 다양한 형태의 데이터를 입력받아 이를 분류하고 상호 연결하여 개인 지식 기반을 구축한다. 이를 통해 사용자는 복잡한 질문에 대한 깊이 있는 연구 결과를 효율적으로 도출할 수 있다.
