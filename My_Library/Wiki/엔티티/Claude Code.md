# Claude Code

## 개요

Claude Code은(는) Andrej Karpathy의 방법을 기반으로 Obsidian 내에서 다양한 원본 정보를 읽고 자동으로 구조화된 위키 페이지를 생성하여 로컬 RAG 시스템을 구축하는 AI 도구입니다.

핵심 특징:
- 자동 위키 페이지 생성: 기사, PDF, 노트, 스크립트, 이미지 등 다양한 원본 자료를 분석하여 자동으로 위키 페이지를 생성합니다.
- 옵시디언 통합: 옵시디언의 'raw' 폴더에 저장된 자료를 활용하고, 태그 및 연결 기능을 통해 지식 구조를 구축합니다.
- 로컬 RAG 시스템 구축: Andrej Karpathy의 개념을 바탕으로 사용자의 로컬 환경에서 검색 증강 생성 시스템을 구현합니다.
- 다양한 활용 시나리오: 비디오 스크립트 분석, 디자이너 및 제품 관리자의 지식 관리 등 '제2의 뇌' 구축에 활용될 수 있습니다.
- 원본 비수정 원칙: LLM이 원본 파일을 직접 수정하지 않고 참조하여 지식을 생성합니다.
