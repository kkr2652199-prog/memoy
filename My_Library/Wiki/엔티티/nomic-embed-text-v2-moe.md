# nomic-embed-text-v2-moe

## 개요

nomic-embed-text-v2-moe은(는) My_Library 개인 지식 도서관 애플리케이션에서 자료 임베딩을 위해 활용되는 텍스트 임베딩 모델이다.

핵심 특징:
- My_Library 자료 임베딩: My_Library 프로젝트 내 172개 자료의 텍스트 임베딩을 완료하여 벡터 검색의 기반을 마련.
- 하이브리드 검색 지원: BM25+벡터 하이브리드 검색 시스템의 벡터 검색 1~3단계 구현에 핵심적인 역할을 수행.
- 자동 임베딩 기능 개발: 신규 자료 섭취 시 자동 임베딩 기능 개발의 기반 기술로 활용 중.
