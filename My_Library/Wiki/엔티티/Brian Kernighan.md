# Brian Kernighan

## 개요

Brian Kernighan은(는) 유닉스 운영체제 개발에 기여하고 C 프로그래밍 언어의 초기 버전을 공동 설계한 저명한 캐나다의 컴퓨터 과학자이다.

핵심 특징:
- C 언어 공동 설계: 데니스 리치와 함께 'The C Programming Language' 저서를 통해 C 언어의 사실상 표준을 확립.
- 유닉스 개발 기여: 벨 연구소에서 유닉스 운영체제의 초기 개발에 중요한 역할을 수행.
- AWK 언어 공동 개발: 알프레드 에이호, 피터 와인버거와 함께 텍스트 처리 및 보고서 생성 언어인 AWK를 개발.
- 프로그래밍 서적 저술: 'The Practice of Programming', 'The Unix Programming Environment' 등 다수의 영향력 있는 프로그래밍 서적을 저술.
