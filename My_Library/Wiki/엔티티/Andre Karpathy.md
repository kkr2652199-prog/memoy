# Andre Karpathy

## 개요

Andre Karpathy는 AI와 대형 언어 모델(LLM)을 활용하여 개인 지식 기반을 구축하고 심층 연구를 수행하는 'LLM 위키' 방법론을 제안한 인물이다.

핵심 특징:
- AI 기반 연구 방법론 제안: LLM을 이용해 텍스트 파일, PDF 등을 분류하고 연결하여 복잡한 질문에 대한 깊이 있는 연구 결과를 얻는 새로운 접근 방식을 제시했다.
- LLM 위키 개념 도입: 개인의 지식 기반을 대형 언어 모델로 구축하여 연구 효율성을 극대화하는 시스템을 소개했다.
- Obsidian 활용 강조: LLM 위키의 데이터를 시각화하고 관리하기 위한 무료 도구인 Obsidian의 중요성을 역설했다.
- 단계별 구축 가이드 제공: LLM 위키를 직접 만들 수 있도록 구체적인 단계별 안내를 제공한다.
