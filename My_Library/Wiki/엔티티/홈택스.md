# 홈택스

## 개요

홈택스은(는) 배달대행 사업자의 복잡한 부가세 신고 과정을 간소화하여 홈택스 일괄 등록용 파일을 제공하는 대리 처리 서비스이다.

핵심 특징:
- 부가세 신고 간소화: 배달대행 사업자가 여러 업체의 정산서를 며칠 밤낮으로 정리할 필요 없이, 단 몇 번의 클릭으로 부가세 신고를 준비할 수 있도록 돕는다.
- 현업 개발 솔루션: 배달대행사 사장이 직접 개발하여 현업의 고통과 노력이 담긴 '부가세 해방' 솔루션으로, 실질적인 문제 해결에 초점을 맞춘다.
- 초간단 3단계 프로세스: 카카오톡/이메일 문의, 정산서 엑셀 파일 전송, 홈택스 일괄 등록용 최종 엑셀 파일 수령의 간편한 절차로 이용 가능하다.
- 홈택스 일괄 등록 파일 제공: 사용자가 전달한 정산서 엑셀 파일을 홈택스에 최대 50건까지 일괄 발급 가능한 형식의 최종 엑셀 파일로 변환하여 제공한다.
- 세무 업무 미제공: 세무 신고 대리, 세금 계산, 상담 등 세무 관련 업무는 일절 제공하지 않으며, 오직 파일 변환 서비스에 집중한다.
