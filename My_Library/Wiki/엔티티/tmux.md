# tmux

## 개요

tmux은(는) 단일 터미널 창에서 여러 세션을 관리하고 분할하며 유지할 수 있게 해주는 터미널 멀티플렉서입니다.

핵심 특징:
- 다중 세션 관리: 하나의 터미널 창 내에서 여러 개의 독립적인 터미널 세션을 생성하고 전환할 수 있습니다.
- 세션 영속성: 터미널 연결이 끊어져도 실행 중인 세션을 유지하여 나중에 다시 연결할 수 있도록 합니다.
- 강력한 개발 환경 구성: iTerm2, lazygit, zoxide 등과 함께 '클로드코드' 사용을 위한 최강 터미널 환경의 핵심 구성 요소로 활용됩니다.
- 통합 및 확장성: 다양한 터미널 유틸리티와 통합되어 사용자 맞춤형의 효율적인 작업 환경을 구축할 수 있습니다.
