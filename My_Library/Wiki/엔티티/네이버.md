# 네이버

## 개요

네이버는 동적 웹 콘텐츠를 제공하여 웹 스크래핑 시 JavaScript 렌더링 및 인코딩 문제 해결이 필요한 주요 웹사이트이다.

핵심 특징:
- 동적 웹사이트 특성: JavaScript 렌더링이 필수적이어서 requests.get과 같은 단순 HTTP 요청으로는 본문 데이터 추출이 어렵다.
- 웹 스크래핑 난이도: 복잡한 구조와 동적 콘텐츠로 인해 데이터 추출 실패 및 인코딩 문제가 자주 발생한다.
- 기술적 개선 필요성: httpx, playwright 도입을 통한 JS 렌더링 처리와 meta charset 파싱, chardet 활용을 통한 인코딩 문제 개선이 제안된다.
