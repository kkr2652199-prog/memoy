# Auto Hunyuan

## 개요

Auto Hunyuan은(는) Grok의 무료 사용 중단 이후 Hunyuan 1.5를 활용한 이미지 기반 AI 비디오 생성 과정을 자동화하여 비효율성을 해결하는 브라우저 확장 프로그램이다.

핵심 특징:
- 자동화된 AI 비디오 생성: 사용자가 이미지를 드래그 앤 드롭하고 프롬프트를 입력하면 자동으로 비디오를 생성한다.
- 클립 자동 관리: 생성 완료된 비디오 클립을 자동으로 다운로드하고 정리하여 사용자 편의성을 높인다.
- 브라우저 확장 프로그램 형태: Microsoft Edge와 Chrome에서 무료로 제공될 예정인 브라우저 확장 프로그램으로 접근성이 뛰어나다.
