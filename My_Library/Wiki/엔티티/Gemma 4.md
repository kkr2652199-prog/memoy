# Gemma 4

## 개요

Gemma 4은(는) 구글이 개발한 고품질 AI 모델로, 인터넷 연결이나 API 비용 없이 개인 컴퓨터에서 무료로 무제한 로컬 실행이 가능하도록 설계되었습니다.

핵심 특징:
- 로컬 실행 지원: 인터넷 연결이나 API 비용 없이 평균적인 PC에서도 뛰어난 성능으로 로컬에서 AI 모델을 실행할 수 있습니다.
- 비용 효율적인 성능: 특히 31억 개의 파라미터를 가진 버전은 성능 대비 비용 측면에서 우수한 투자 수익률(ROI)을 제공합니다.
- 이미지 분석 기능: 사용자가 업로드한 이미지를 분석하고 해당 이미지에 대한 질문에 답변할 수 있는 기능을 갖추고 있습니다.
- Ollama를 통한 접근성: Ollama를 통해 쉽게 다운로드하여 로컬 환경에서 다양한 작업을 수행할 수 있습니다.
