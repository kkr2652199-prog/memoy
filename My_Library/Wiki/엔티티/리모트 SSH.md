# 리모트 SSH

## 개요

리모트 SSH은(는) VS Code를 활용하여 원격 서버에 SSH로 접속, 직접 개발 환경을 구축하고 관리할 수 있도록 지원하는 기능이다.

핵심 특징:
- VS Code 기반 원격 개발: VS Code를 통해 원격 서버에 직접 접속하여 로컬 환경처럼 개발할 수 있는 통합 환경을 제공한다.
- 기존 SSH 클라이언트 대체: Termius와 같은 기존 SSH 클라이언트의 한계를 극복하고, 개발에 최적화된 통합된 경험을 제공한다.
- 상세한 설정 가이드: 설치부터 환경 설정, 가독성 향상 팁까지 원격 개발 환경 구축의 전 과정을 상세히 안내한다.
- 개인 개발자 최적화: 다른 에디터와의 비교를 통해 개인 개발자에게 VS Code 리모트 SSH가 적합한 이유와 장점을 제시한다.
