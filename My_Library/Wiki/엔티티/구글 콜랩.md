# 구글 콜랩

## 개요

구글 콜랩은(는) 구글에서 제공하는 클라우드 기반의 파이썬 개발 환경으로, 특히 머신러닝 및 딥러닝 모델 학습에 필요한 GPU/TPU 자원을 제공한다.

핵심 특징:
- 클라우드 기반 개발 환경: 웹 브라우저를 통해 Python 코드를 실행하고 머신러닝 모델을 학습시킬 수 있는 클라우드 기반 환경을 제공한다.
- GPU/TPU 가속 지원: 딥러닝 모델의 효율적인 학습 및 파인튜닝을 위해 GPU 및 TPU와 같은 하드웨어 가속기를 무료로 지원한다.
- 모델 파인튜닝 및 배포: Gemma 4와 같은 대규모 언어 모델을 파인튜닝하고, 학습된 모델을 GGUF 형식으로 로컬에 저장하는 등 모델 배포 과정을 지원한다.
- 외부 AI 플랫폼 연동: 허깅 페이스(Hugging Face)와 같은 외부 AI 플랫폼과 연동하여 사용자 정의 데이터셋 및 사전 학습된 모델을 활용할 수 있다.
