# LLM

## 개요

LLM은(는) 방대한 텍스트 데이터를 학습하여 인간과 유사한 언어를 이해하고 생성하며, 특히 대화형 인터페이스를 통해 지식 관리 및 개인화된 정보 구축에 활용되는 인공지능 기술이다.

핵심 특징:
- 대화형 지식 저장 및 연속성: 대화형 인터페이스를 통해 생성된 지식을 구조화된 파일 형태로 저장하고, 이를 다음 대화에 활용하여 지식의 연속성을 유지한다.
- AI 기반 지식 정리 및 페이지 생성: 사용자가 수집한 다양한 자료를 AI가 분석하고 정리하여 체계적인 위키 페이지로 자동 생성한다.
- 자동 지식 그래프 형성: AI가 기존 지식 페이지들 간의 연관성을 자동으로 식별하고 연결하여 지식 그래프를 구축, 정보 탐색 및 이해를 돕는다.
