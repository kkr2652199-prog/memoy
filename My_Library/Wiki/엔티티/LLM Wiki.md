# LLM wiki

## 개요

LLM wiki은(는) Andre Karpathy가 제안한, LLM(대형 언어 모델)을 활용하여 개인의 텍스트 및 PDF 자료를 분류하고 연결하여 심층 연구를 지원하는 지식 기반 시스템이다.

핵심 특징:
- AI 기반 연구 방법론: LLM을 활용하여 방대한 자료를 체계화하고 분석하는 새로운 연구 패러다임을 제시한다.
- 개인 지식 기반 구축: 사용자가 입력한 다양한 형태의 데이터를 LLM이 자동으로 분류하고 상호 연결하여 맞춤형 지식 저장소를 생성한다.
- 심층 연구 지원: 복잡하고 다층적인 질문을 통해 연결된 지식 속에서 심도 있는 통찰과 연구 결과를 도출할 수 있도록 돕는다.
- 연구 효율성 극대화: 자료 정리 및 분석에 소요되는 시간을 단축하여 연구자가 핵심적인 사고와 분석에 집중할 수 있도록 연구 효율성을 높인다.
- 시각화 도구 연동: Obsidian과 같은 무료 도구를 활용하여 구축된 지식 기반의 데이터를 시각적으로 탐색하고 이해할 수 있도록 지원한다.
