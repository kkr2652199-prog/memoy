# 클라우드

## 개요

클라우드는 데이터 저장 및 처리의 효율성을 높이는 기술로, 다양한 서비스에서 사용된다. 최근 1226회 로또 번호 예측 실험에서 로또PD가 1회부터 1225회까지의 1등 당첨 번호 데이터를 활용하여 AI를 통해 로또 번호를 예측했다. 네 개의 AI가 각기 다른 방식으로 번호를 선택하고, 지난주 결과와 미출연 번호를 분석하여 예측의 정확성을 높였다. AI별로 선택된 번호와 평균 적중 개수를 비교하며, 각 AI의 예측 방식과 결과를 상세히 설명했다.
