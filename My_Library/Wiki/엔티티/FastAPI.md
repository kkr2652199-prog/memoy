# FastAPI

## 개요

FastAPI은(는) 파이썬 기반의 현대적이고 빠르며 고성능 웹 프레임워크로, API 구축에 최적화되어 있습니다.

핵심 특징:
- 높은 성능: Starlette와 Pydantic을 기반으로 하여 매우 빠른 속도를 제공합니다.
- 자동 문서화: OpenAPI 표준을 준수하여 Swagger UI 및 ReDoc을 통한 자동 API 문서를 생성합니다.
- 데이터 유효성 검사: Pydantic을 활용하여 강력한 데이터 유효성 검사 및 직렬화를 지원합니다.
- 비동기 지원: `async/await` 구문을 통해 비동기 프로그래밍을 기본으로 지원하여 높은 동시성을 처리할 수 있습니다.

관련: [[SQLite]], [[Ollama]], [[D3.js]], [[Python]]
