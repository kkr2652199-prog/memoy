# LSTM 모델

## 개요

LSTM(Long Short-Term Memory) 모델은 시계열 데이터 분석에 효과적인 딥러닝 기술로, 주로 순환 신경망(RNN) 구조를 기반으로 한다. 최근 로또사주에서는 1218회 로또 번호 예측을 위해 LSTM 모델을 활용하여 행운의 번호를 추출하였다. 분석 결과, 예측된 행운의 번호는 3, 5, 7, 8, 13, 14, 17, 20, 22, 30, 35번이며, 실제 당첨 번호는 3, 28, 31, 32, 42, 45번으로 나타났다. 이 모델은 과거 데이터를 기반으로 미래의 결과를 예측하는 데 강력한 성능을 발휘한다.
