# Proton VPN

## 개요

Proton VPN은(는) 스위스에 기반을 둔 오픈소스 가상 사설망(VPN) 서비스로, 사용자 개인 정보 보호와 보안에 중점을 둡니다.

핵심 특징:
- 강력한 보안 및 암호화: AES-256 암호화, OpenVPN 및 WireGuard 프로토콜을 사용하여 데이터를 안전하게 보호합니다.
- 엄격한 노로그 정책: 사용자 활동 기록을 저장하지 않아 개인 정보 유출 위험이 없습니다.
- 시큐어 코어(Secure Core) 아키텍처: 여러 국가를 경유하는 다중 홉 VPN을 통해 고급 네트워크 공격으로부터 사용자를 보호합니다.
- 킬 스위치 및 DNS 유출 방지: VPN 연결이 끊어질 경우 인터넷 연결을 자동으로 차단하여 IP 주소 노출을 방지합니다.
- 스위스 기반 운영: 강력한 개인 정보 보호 법률을 가진 스위스에 본사를 두어 사용자 프라이버시를 보장합니다.
