# vibe coding

## 개요

vibe coding은(는) AI를 활용하여 코드를 작성할 때 시스템에 대한 깊은 이해나 비판적 사고 없이 피상적으로 접근하는 방식을 의미한다.

핵심 특징:
- AI에 대한 과도한 의존: AI가 생성한 코드에 대한 비판적 검토나 심층적인 이해 없이 전적으로 의존하는 경향.
- 시스템 이해 부족: 개발 중인 시스템의 아키텍처, 성능, 확장성 등에 대한 심층적인 지식 없이 코드를 생성하고 사용하는 방식.
- 사고 과정의 결여: 문제 해결 과정에서 필요한 분석, 설계, 최적화 등 개발자의 비판적 사고가 부족한 상태.
- 잠재적 문제 발생 가능성: 겉보기에는 작동하는 것처럼 보이지만, 실제 운영 환경에서 성능 저하와 같은 심각한 문제를 야기할 수 있음.
