# 위스크

## 개요

위스크는 구글 플로우(Nano Banana 2)를 활용하여 텍스트 대본으로부터 이미지를 자동으로 생성하는 크롬 확장 프로그램이다.

핵심 특징:
- 구글 플로우(Nano Banana 2) 기반 자동화: 나노바나 2 기술을 활용하여 콘텐츠 제작 과정을 자동화하는 프로그램이다.
- 텍스트 대본 이미지 자동 생성: 텍스트 파일의 대본을 읽어 최대 5명의 인물과 다양한 배경의 이미지를 자동으로 생성한다.
- 크롬 확장 프로그램 형태: 구글 드라이브에서 다운로드하여 크롬 확장 프로그램으로 쉽게 설치 및 사용할 수 있다.
- 사용자 맞춤형 속도 조절: 소스 코드 수정을 통해 이미지 생성 속도를 사용자가 직접 조절할 수 있다.
