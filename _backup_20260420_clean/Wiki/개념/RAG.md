# RAG

## 설명

RAG(Recall-Augmented Generation)는 정보 검색과 생성 모델을 결합한 접근 방식으로, 사용자가 필요로 하는 정보를 보다 정확하게 제공하기 위해 외부 데이터베이스에서 정보를 검색하고 이를 바탕으로 응답을 생성하는 기술입니다. 최근 자료에서는 Obsidian과 Claude Code를 활용하여 개인 지식 관리 시스템을 설정하는 방법이 소개되었으며, 이는 Andrej Karpathy의 개념을 기반으로 합니다. 사용자는 원시 정보를 입력하고 Claude Code가 이를 분석하여 위키 페이지를 생성하도록 하며, Obsidian을 통해 태그 및 자산 간의 관계를 효과적으로 관리할 수 있습니다. ⚠️ 모순: 기존 내용에는 Obsidian과 Claude Code에 대한 언급이 없습니다. 새 자료에서는 안드레아스 카파의 LLM 위키 개념이 지식 관리와 개인 지식 기반 구축에 관한 최신 트렌드와 기술적 접근 방식을 다루고 있다고 언급하고 있습니다.
