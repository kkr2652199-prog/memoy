# JS 렌더링

## 설명

JS 렌더링은 웹 페이지의 동적 콘텐츠를 표시하기 위해 JavaScript를 실행하는 과정을 의미한다. 현재 웹 스크래핑 시스템의 문제점 중 하나는 네이버와 유튜브와 같은 동적 사이트에서의 데이터 추출 실패로, 이는 JavaScript 렌더링이 필요하기 때문이다. 예를 들어, requests.get을 사용하면 네이버의 본문을 제대로 가져오지 못하며, 유튜브 채널 URL 처리에도 어려움이 있다. 해결책으로는 httpx와 playwright를 도입하여 JS 렌더링을 지원하는 방법이 제안되고 있다.
