# 증강 생성

## 설명

증강 생성은 기존의 생성 모델을 기반으로 하여, 추가적인 정보를 통해 생성된 콘텐츠의 품질과 유용성을 향상시키는 과정이다. 최근 Andrej Karpathy는 LLM Wiki라는 프레임워크를 발표했으나, 이 시스템은 자기 인식이 부족하다는 문제를 지적했습니다. LLM Wiki는 연구 자료를 정리하고 개념 간의 연결을 생성하지만, 사용자가 원하는 통찰력을 생성하기 위해서는 지식 그래프가 필요합니다. 이를 통해 사용자는 연구 아이디어를 시각화하고, 지식의 공백을 식별할 수 있습니다.
