# 메모리 관리

## 설명

메모리 관리는 [[컨텍스트 엔지니어링]]의 중요한 구성 요소 중 하나입니다. 이는 [[거대 언어 모델]](LLM)이 대화의 맥락이나 이전 상호작용을 기억하고 활용할 수 있도록 돕는 기술입니다. LLM의 기억 상실 문제를 극복하고 장기적인 대화나 복잡한 작업을 수행할 수 있도록 지원합니다. 효과적인 메모리 관리는 AI가 더 일관성 있고 유용한 응답을 생성하는 데 필수적입니다.
