# 렌더링

## 설명

렌더링은 컴퓨터 그래픽스에서 3D 모델이나 장면을 2D 이미지로 변환하는 과정으로, 시각적 요소를 최종적으로 표현하는 데 중요한 역할을 합니다. 최근에는 자동화 도구인 '오토플로우'가 등장하여, 사용자가 수동으로 작업하지 않고도 무한 연속 렌더링을 가능하게 하여 영상 제작의 효율성을 높이고 있습니다. 이 도구는 구글플로우를 완전히 자동화하여 사용자가 다양한 언어로 프롬프트를 입력할 수 있으며, 자동 번역 기능도 제공하여 글로벌 사용자들에게 유용한 지원을 합니다.
