# 데이터 구조

## 설명

이 강의에서는 개인의 지식을 기반으로 한 인공지능 에이전트를 만드는 방법을 설명한다. 강사는 10년간 쌓아온 데이터를 활용하여 AI가 개인의 기억과 지식을 연결하고 성장할 수 있도록 하는 방법을 공유한다. 이를 위해 옵시디안이라는 도구를 사용하여 데이터를 구조화하고, 구글의 AI 모델인 제마 4를 활용한다.
