# Agent mode

## 설명

[[Agent mode]]는 [[GitHub Copilot]]이 제공하는 독특한 기능 중 하나로, 특히 [[파일 검증]]을 통해 개발 워크플로우를 지원합니다. 이 모드는 [[AI 기반 개발자 도구]]인 [[GitHub Copilot]]의 다양한 기능 중 하나로, 코드 완성, 개념 설명, 편집 제안 등과 함께 제공됩니다. [[Visual Studio Code]] 및 [[JetBrains IDEs]]와 같은 주요 [[개발 환경]]에 통합되어 개발자의 생산성을 높입니다. [[Agent mode]]는 개발자가 잠재적인 문제를 사전에 식별하고 해결하는 데 도움을 주어 전반적인 [[개발 워크플로우]]의 효율성을 증대시킵니다.
