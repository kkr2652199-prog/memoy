# Inline suggestions

## 설명

Inline suggestions은 사용자 워크플로우 또는 편집기 인터페이스 내에서 직접 제공되는 실시간, 상황 인식 추천을 의미합니다. 이러한 제안은 사용자가 입력하거나 상호 작용할 때 관련 정보, 코드 스니펫 또는 작업을 제공하여 생산성을 향상시키는 것을 목표로 합니다. [[GitHub Copilot]]은 이러한 인라인 제안의 대표적인 예시로, 개발 워크플로우 전반을 지원하는 AI 기반 개발자 도구입니다. 이 도구는 [[코드 완성]], 개념 설명, 편집 제안, 에이전트 모드를 통한 파일 검증 등 다양한 기능을 [[Visual Studio Code]] 및 [[JetBrains IDEs]]와 같은 주요 개발 환경에 통합하여 제공합니다. GitHub Copilot은 개발자의 직무 만족도를 최대 75% 높이고 코드 작성 생산성을 향상시키는 데 기여합니다.
