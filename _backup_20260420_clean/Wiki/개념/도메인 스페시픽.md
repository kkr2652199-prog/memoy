# 도메인 스페시픽

## 설명

도메인 스페시픽은 특정 분야나 산업에 특화된 개념이나 기술을 의미한다. 예를 들어, 하네스 엔지니어링의 경우, 클로드 코드와 같은 도구들이 이미 하네스를 내장하고 있어 프로젝트에 맞는 커스텀 하네스를 추가하여 성능을 향상시킬 수 있다. 하네스의 기본 개념과 오픈 소스 하네스의 활용, 그리고 커스텀 하네스를 만드는 방법은 도메인 스페시픽 기술의 중요한 예시로, 프로젝트 규칙과 아키텍처 결정을 문서화하여 AI에게 전달하는 과정이 포함된다. 새 자료에서는 하네스 엔지니어링의 개념과 활용 방법을 상세히 다루며, 특히 프로젝트에 맞는 커스텀 하네스를 추가하는 방법에 대해 설명한다. ⚠️ 모순: 도메인 스페시픽의 정의와 하네스 엔지니어링의 구체적인 활용 사례가 명확히 연결되지 않음.
