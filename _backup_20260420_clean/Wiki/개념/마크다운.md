# 마크다운

## 설명

마크다운은 일반 텍스트 기반의 경량 마크업 언어로, 문서를 쉽고 빠르게 작성하고 서식을 지정할 수 있게 해줍니다. 이는 텍스트 편집기에서 특별한 도구 없이도 제목, 목록, 링크, 코드 블록 등 다양한 요소를 표현할 수 있어 개발자나 작가들 사이에서 널리 사용됩니다. 최근 AI를 가장 똑똑하게 활용하는 방법으로 'LLM 위키'를 구축하는 시스템이 소개되었는데, 이 시스템은 옵시디안과 같은 도구를 활용하며, 옵시디안은 마크다운을 기반으로 지식 관리를 수행합니다. 따라서 마크다운은 LLM 위키와 같은 AI 기반 지식 관리 시스템에서 정보를 구조화하고 연결하는 데 중요한 역할을 합니다.
