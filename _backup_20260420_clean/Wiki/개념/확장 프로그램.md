# 확장 프로그램

## 설명

확장 프로그램은 웹 브라우저의 기능을 확장하거나 개선하기 위해 설계된 소프트웨어 모듈입니다. 최근 구글 플로우(Nano Banana 2) 자동 생성 크롬 확장 프로그램이 출시되어 사용자가 쉽게 콘텐츠를 제작할 수 있도록 돕는 도구로 주목받고 있습니다. 이 프로그램은 5시간의 작업을 통해 개발되었으며, 사용자는 구글 드라이브에서 다운로드 후 크롬 확장 프로그램으로 등록하여 사용할 수 있습니다. 사용법은 텍스트 파일에서 대본을 읽어와 자동으로 이미지를 생성하는 방식으로, 사용자의 편리함을 극대화합니다.
