# 마케팅 콘텐츠

## 설명

마케팅 콘텐츠는 제품이나 서비스를 홍보하기 위해 제작되는 다양한 형태의 콘텐츠를 의미합니다. 최근 Matt Penny는 AI와 Claude를 활용하여 Canva에서 그래픽 디자인을 신속하고 전문적으로 만드는 방법을 소개했습니다. 그는 사용자가 Claude 데스크탑 앱을 설치하고 Canva와 연결하는 과정을 설명하며, 이를 통해 사용자가 원하는 디자인을 쉽게 생성할 수 있음을 강조합니다. AI가 생성한 디자인은 편집이 용이하며, 비용이 발생하지 않는 점에서 유용합니다.
