# 기억 한계

## 설명

AI의 기억 한계는 인공지능이 대화의 긴 맥락이나 이전에 설정된 규칙을 잊어버리는 문제를 야기합니다. 이는 AI가 일관성을 유지하지 못하고 '규칙 이탈' 또는 '맥락 상실'을 보이는 [[AI 규칙 준수 문제]]의 근본 원인 중 하나입니다. 이러한 한계를 극복하기 위해 [[맥락 주입]] 기법을 활용하여 AI에게 중요한 정보와 규칙을 주기적으로 상기시켜 주는 것이 중요합니다. 이는 특히 [[대규모 언어 모델]]의 성능 안정화에 필수적입니다.
