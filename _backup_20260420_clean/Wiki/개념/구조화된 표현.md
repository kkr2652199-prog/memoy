# 구조화된 표현

## 설명

구조화된 표현은 정보를 체계적으로 정리하고, 개념 간의 관계를 명확하게 나타내는 방법을 의미합니다. 최근 Andrej Karpathy는 LLM Wiki라는 프레임워크를 발표했으나, 이 시스템은 자기 인식이 부족하다는 문제를 지적받았습니다. LLM Wiki는 연구 자료를 정리하고 개념 간의 연결을 생성하지만, 사용자가 원하는 통찰력을 생성하기 위해서는 [[지식 그래프]]가 필요합니다. 이를 통해 사용자는 연구 아이디어를 시각화하고, 지식의 공백을 식별할 수 있습니다.
