# 챗봇 의존성

## 설명

챗봇 의존성은 사용자가 AI 챗봇의 제한된 기능에만 의존하여 잠재적인 AI 활용 범위를 충분히 탐색하지 못하는 경향을 의미합니다. 2026년 4월 18일 라이브 세션은 "단순 챗봇을 넘어선 'AI 워크플로우 구축' 전략"을 강조하며, 이러한 [[챗봇 의존성]]을 극복하는 것을 핵심 목표로 삼습니다. Gemini를 챗봇이 아닌 슈퍼컴퓨터처럼 활용하여 [[AI 워크플로우 구축]]을 설계하는 것은 사용자가 수동적인 '사용자'에서 능동적인 'AI 워크플로우 설계자'로 전환하도록 돕습니다. 이를 통해 [[Tool Use & API 연동]]과 같은 고급 기능을 활용하여 AI의 진정한 가치를 발휘할 수 있습니다.
