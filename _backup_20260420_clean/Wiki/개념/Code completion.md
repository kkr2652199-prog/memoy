# Code completion

## 설명

[[Code completion]]은 개발자가 코드를 작성할 때 자동으로 코드 조각, 함수 이름, 변수 등을 제안하여 입력 속도를 높이는 기능입니다. [[GitHub Copilot]]은 이러한 [[코드 완성]] 기능을 핵심적으로 제공하는 [[AI 기반 개발자 도구]]입니다. 이 도구는 [[Visual Studio Code]], [[JetBrains IDEs]] 등 주요 [[개발 환경]]에 통합되어 [[개발 워크플로우]] 전반에 걸쳐 지원됩니다. 이를 통해 개발자의 [[직무 만족도]]를 최대 75% 높이고 [[코드 작성 생산성]]을 크게 향상시킵니다.
