# Context Window

## 설명

[[Context Window]]는 [[대규모 언어 모델]]이 한 번에 처리하고 고려할 수 있는 입력 텍스트의 최대 길이를 의미합니다. 이는 모델의 '임시 기억' 공간으로, 현재 [[프롬프트]]와 대화의 이전 턴들을 포함하여 모델이 응답을 생성하는 데 활용하는 정보의 범위를 결정합니다. [[Gemma 4]]와 같은 LLM은 이 Context Window 내의 정보만을 기억하며, 이 범위를 벗어나는 과거 대화 내용은 자동으로 잊어버리는 [[기억 한계]]를 가집니다. 따라서, [[대화 맥락 유지 및 기억 능력]]은 이 Context Window의 크기에 크게 의존합니다.
