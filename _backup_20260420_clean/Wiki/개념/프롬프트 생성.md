# 프롬프트 생성

## 설명

프롬프트 생성은 사용자가 제공한 한국어 대본에 따라 각 씬에 맞는 영문 이미지 프롬프트를 생성하는 과정이다. 이지비디오 영문 이미지 프롬프트 생성기는 캐릭터 이미지를 분석하여 고유 특징을 영문으로 묘사하고, 이를 기반으로 프롬프트를 조립한다. 프롬프트는 캐릭터 묘사, 역할, 배경 시각화, 실존 인물 묘사로 구성되며, 모든 내용은 영어로 작성되어야 한다. 사용자는 씬을 나누어 프롬프트를 생성할 수 있다.
