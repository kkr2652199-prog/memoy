# 상태 비저장

## 설명

[[상태 비저장]]은 [[대규모 언어 모델]]과 같은 시스템이 각 상호작용이나 요청을 독립적인 이벤트로 처리하며, 이전의 대화나 상태 정보를 자동으로 기억하지 않는 특성을 의미합니다. 이는 모델이 이전 대화 내용을 자체적으로 저장하지 않고, 매번 새로운 입력에 대해 응답을 생성하는 방식으로 작동함을 뜻합니다. 결과적으로, [[Gemma 4]]와 같은 LLM은 현재 [[Context Window]] 내의 정보만을 활용하며, 이로 인해 [[대화 맥락 유지 및 기억 능력]]에 본질적인 한계를 가집니다. 진정한 [[장기 기억]]을 위해서는 외부적인 메커니즘이 필수적입니다.
