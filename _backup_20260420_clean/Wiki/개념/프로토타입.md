# 프로토타입

## 설명

Anthropic의 Claude Design은 Opus 4.7의 시각적 기능을 활용하여 웹, 앱, 슬라이드 디자인을 자동 생성하는 [[AI 디자인]] 도구입니다. 이 앱은 사용자가 [[디자인 시스템]]을 구축하고 이를 기반으로 새로운 프로젝트를 생성할 수 있도록 지원합니다. 특히, [[와이어프레임]] 또는 내용이 채워진 웹사이트를 생성하는 기능은 초기 [[프로토타입]] 제작 과정을 자동화하여 디자인 효율성을 높입니다. 이를 통해 디자이너는 아이디어를 빠르게 시각화하고 검증할 수 있습니다.
