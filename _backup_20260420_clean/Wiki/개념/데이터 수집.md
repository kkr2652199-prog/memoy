# 데이터 수집

## 설명

데이터 수집은 정보를 수집하고 정리하는 과정으로, 다양한 출처에서 데이터를 확보하여 분석과 연구에 활용하는 중요한 단계이다. 최근 Andre Karpathy는 AI를 활용한 새로운 연구 방법을 제안하며, LLM(대형 언어 모델)을 이용해 개인 지식 기반을 구축하는 'LLM 위키'를 소개했다. 이 방법은 사용자가 텍스트 파일, PDF 등을 입력하면 LLM이 이를 분류하고 연결해 주며, 사용자는 복잡한 질문을 통해 깊이 있는 연구 결과를 얻을 수 있다. 이러한 접근은 데이터 수집의 효율성을 높이고, 연구의 깊이를 더하는 데 기여할 수 있다. ⚠️ 모순: 기존 내용에는 LLM 위키와 관련된 정보가 포함되어 있지 않음.
