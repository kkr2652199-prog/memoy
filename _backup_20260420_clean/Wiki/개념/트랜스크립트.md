# 트랜스크립트

## 설명

트랜스크립트는 음성이나 비디오 콘텐츠의 내용을 텍스트로 변환한 것을 의미한다. 현재 웹 스크래핑 시스템의 문제점 중 하나는 네이버와 유튜브와 같은 동적 사이트에서의 데이터 추출 실패로, 이는 JavaScript 렌더링이 필요하기 때문이다. 유튜브 채널 URL 처리의 어려움 또한 트랜스크립트 추출에 영향을 미친다. 해결책으로는 httpx와 playwright를 통한 JS 렌더링 지원이 제안되고 있다.
