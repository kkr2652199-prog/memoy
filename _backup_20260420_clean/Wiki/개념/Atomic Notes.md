# Atomic Notes

## 설명

[[Atomic Notes]]는 하나의 명확하고 독립적인 아이디어를 담은 최소 단위의 지식 기록을 의미합니다. 이는 'LLM 위키 정신'에 기반한 [[지식 구조 설계]]에서 중요한 구성 요소로, 각 노트가 고유한 개념을 다루며 다른 노트와 [[상호 참조성]]을 통해 연결됩니다. 이러한 방식은 복잡한 지식을 관리하고 새로운 연결을 발견하는 데 용이하며, [[지식 관리 시스템]]의 유연성을 높입니다. 2026년 4월 19일 자료에서 언급된 '도서관(지식 구조)'의 자료 품질을 향상시키는 데 기여합니다.
