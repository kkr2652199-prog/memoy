# 대본 생성

## 설명

대본 생성은 한국 전통 야담 대본 작성을 위한 단계별 흐름과 구조를 설명하는 과정이다. 사용자는 장르를 선택하고, 주제와 제목을 제시받은 후, 인트로와 전체 대본을 작성하는 과정을 거친다. 각 단계는 구체적인 규칙과 예시를 포함하고 있으며, 대본은 Save the Cat 구조를 기반으로 한다. 최종적으로 대본은 약 90분 분량으로 구성되며, 각 비트의 시간 배분과 내용이 포함된다.
