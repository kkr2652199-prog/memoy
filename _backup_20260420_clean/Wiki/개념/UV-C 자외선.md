# UV-C 자외선

## 설명

UV-C 자외선은 파장이 짧은 자외선으로, 주로 멸균 및 소독에 사용된다. 스마트 제로 브이(Smart Zero-V)는 UV-C 자외선 기반 멸균 기술을 활용한 음식물 운송 전용 박스로, 저전력 회로를 통해 위생을 강화하는 제품이다. 이 제품은 배달 라이더가 식품 직접 취급자가 아니어서 법적 사각지대에 놓인 문제를 해결하고, 기존의 위생 문제를 개선하기 위해 개발되었다. 2025년부터 전국 주요 도시에서 시범 운용될 예정이며, 멸균 성능 검증과 특허 출원이 진행 중이다.
