# 이미지 생성

## 설명

이미지 생성은 컴퓨터 알고리즘을 사용하여 이미지 콘텐츠를 자동으로 만들어내는 과정입니다. 최근 [[그록 AI]]가 유료로 전환되면서 [[메타 AI]]가 무료 대체 도구로 소개되었으며, 이는 이미지와 영상 생성 기능을 제공하지만 한국에서는 VPN을 통해 접근해야 합니다. [[Gemini]]와 같은 다른 AI 시스템 또한 이미지 및 동영상 생성 기능을 핵심 기능으로 제공하여 사용자가 다양한 시각 콘텐츠를 손쉽게 제작할 수 있도록 돕습니다. 이러한 AI 도구들은 사용자가 원하는 이미지를 효율적으로 생성하며, 특정 [[워크플로우]]에 통합되어 활용될 수 있습니다.
