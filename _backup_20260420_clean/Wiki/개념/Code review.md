# Code review

## 설명

[[Code review]]는 소프트웨어 품질을 보장하고 잠재적인 오류를 식별하는 중요한 과정입니다. [[GitHub Copilot]]은 직접적인 [[코드 리뷰]] 도구는 아니지만, 그 기능들을 통해 [[코드 리뷰]] 프로세스를 간접적으로 지원하고 개선합니다. 이 [[AI 기반 개발자 도구]]는 [[코드 완성]], [[개념 설명]], 그리고 [[편집 제안]]을 통해 초기 단계부터 고품질의 코드를 작성하도록 유도하여 [[코드 리뷰]]의 부담을 줄여줍니다. 특히 [[에이전트 모드]]를 통한 [[파일 검증]] 기능은 잠재적인 문제를 사전에 식별하여 [[코드 리뷰]] 단계에서 발견될 오류의 수를 줄이는 데 기여할 수 있습니다. [[GitHub Copilot]]은 전반적인 [[개발 워크플로우]]를 효율화하여 더 효과적인 [[소프트웨어 개발]]을 가능하게 합니다.
