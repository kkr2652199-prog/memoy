# Large Language Model

## 설명

[[Large Language Model]] (LLM)은 방대한 텍스트 데이터를 학습하여 자연어 이해 및 생성 능력을 갖춘 인공지능 모델입니다. 그러나 기존 LLM은 학습 이후의 새로운 정보나 대화 맥락에 대한 '휘발성 기억' 문제를 가지고 있습니다. Andrej Karpathy의 [[LM-Milky]] 아이디어는 이러한 LLM의 한계를 극복하기 위해 고안된 개인 AI 지식 위키 구축 방법을 제시합니다. 이 시스템은 AI가 직접 위키를 생성하고 유지보수하도록 하여 LLM이 장기적인 지식을 축적하고 활용할 수 있도록 돕습니다. 이는 [[Retrieval Augmented Generation]]과 [[Knowledge Compiling]]의 중요성을 강조하며, [[Personal Knowledge Management]] 분야에 혁신을 가져옵니다.
