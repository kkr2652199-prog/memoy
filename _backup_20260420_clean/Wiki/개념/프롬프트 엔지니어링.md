# 프롬프트 엔지니어링

## 설명

프롬프트 엔지니어링은 사용자가 AI 모델에 입력하는 프롬프트를 최적화하여 원하는 결과를 얻는 기술입니다. 최근 구글의 'VEO 3.1' 모델을 활용한 무료 AI 영상 제작 방법이 주목받고 있으며, 사용자는 프롬프트를 입력하여 원하는 영상을 생성하고 다양한 기능을 통해 영상 편집과 음악 추가도 가능합니다. 또한, '그록 자동화' 확장 프로그램을 활용하면 반복적인 편집 작업을 자동화하여 창작에 집중할 수 있습니다. 최근에는 클로드 코드 사용 시 발생하는 비용을 줄이는 방법에 대한 논의도 이루어지고 있으며, 이는 정확한 프롬프트 작성과 작업에 맞는 모델 선택 등을 포함합니다. ⚠️ 모순: 프롬프트 엔지니어링의 정의가 구체적으로 설명되지 않음.
