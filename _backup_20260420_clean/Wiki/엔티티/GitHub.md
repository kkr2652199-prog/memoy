# GitHub

## 개요

GitHub은 소프트웨어 개발 프로젝트를 위한 웹 기반 호스팅 서비스로, Git을 사용하여 코드 저장소 관리 및 개발자 협업을 지원한다. 이 플랫폼은 코드 리뷰, 프로젝트 관리, 오픈 소스 및 개인 프로젝트 호스팅을 통해 개발 커뮤니티의 상호작용을 촉진한다. [[Build Your Second Brain With Claude Code, Karpathy’s Method]]와 같은 자료에서 개인 지식 관리 시스템 설정에 활용되며, [[Claude Code]]와 관련된 개발 과정에서 외부 API 연동 문제가 보고되기도 했다. ⚠️ 모순: 이번에 제공된 새 자료 요약은 [[Andrej Karpathy]]의 'LM-Milky' 아이디어를 활용한 개인 AI 지식 위키 구축에 대한 내용으로, [[GitHub]] 자체에 대한 직접적인 새로운 정보를 포함하고 있지 않다.
