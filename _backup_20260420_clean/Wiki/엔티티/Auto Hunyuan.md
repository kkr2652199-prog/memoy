# Auto Hunyuan

## 개요

Auto Hunyuan은 AI 비디오 자동화 과정을 혁신적으로 변화시키는 무료 브라우저 확장 프로그램으로, 사용자가 간단한 설정 후 여러 이미지를 한 번에 업로드하여 자동으로 비디오를 생성할 수 있도록 돕습니다. 사용자가 이미지를 드래그 앤 드롭하고 프롬프트를 입력하면 자동으로 비디오를 생성하며, 완료된 클립을 자동으로 다운로드하여 정리해줍니다. 이 도구는 Grok의 무료 사용 중단 이후, Hunyuan 1.5를 활용한 비디오 생성의 비효율성을 해결하기 위해 개발되었습니다. ⚠️ 모순: 기존 내용에서는 Auto Hunyuan이 소프트웨어로 언급되었으나, 새 자료에서는 브라우저 확장 프로그램으로 설명되고 있습니다.
