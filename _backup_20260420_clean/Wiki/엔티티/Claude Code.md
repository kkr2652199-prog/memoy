# Claude Code

## 개요

'[[Claude Code]]'는 개인 지식 관리 시스템 구축을 돕는 제품으로, [[Obsidian]]과 함께 활용된다. 이 도구는 [[Andrej Karpathy]]의 'LM-Milky' 아이디어를 기반으로 하는 개인 AI 지식 위키 구축 방법론에서 핵심적인 역할을 수행한다. [[Claude Code]]는 사용자가 입력한 원시 정보를 분석하여 위키 페이지를 생성하며, AI가 직접 위키를 생성하고 유지보수하도록 돕는다. [[LLM 위키]] 시스템 구축 시 AI 코딩 도구의 한 예시로 언급되며, [[Obsidian]]과의 연동을 통해 태그 및 자산 간의 관계를 효과적으로 관리하여 개인 지식 체계를 구성하는 데 기여한다.
