# Ollama

## 개요

Ollama는 고품질 AI 모델을 로컬에서 실행할 수 있는 혁신적인 플랫폼으로, Gemma 4 AI와 같은 모델을 무료로 무제한 사용할 수 있도록 지원합니다. 이 플랫폼은 인터넷 연결이나 API 비용 없이도 뛰어난 성능을 발휘하며, 평균적인 PC에서도 원활하게 작동합니다. 최근 My_Library 개인 지식 도서관 애플리케이션 개발에 [[llama3]] 모델과 함께 활용되어, 자료 임베딩 및 검색 기능 구현에 기여하고 있습니다. 2026년 4월 19일 기준으로 My_Library 프로젝트는 [[FastAPI]], [[SQLite]], [[D3.js]]와 함께 Ollama를 사용하여 172개의 자료를 관리하고 있습니다.
