# 네이버

## 개요

네이버는 대한민국의 대표적인 포털 사이트로, 다양한 서비스와 콘텐츠를 제공한다. 최근 웹 스크래핑 시스템의 문제점으로는 네이버와 유튜브 같은 동적 사이트에서의 데이터 추출 실패와 인코딩 문제가 지적되고 있다. 특히 네이버는 JavaScript 렌더링이 필요하여 requests.get으로는 본문을 제대로 가져오지 못하는 상황이다. 이를 해결하기 위해 JS 렌더링을 위한 httpx와 playwright 도입이 제안되고 있다.
