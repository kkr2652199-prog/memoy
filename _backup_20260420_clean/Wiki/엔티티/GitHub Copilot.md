# GitHub Copilot

## 개요

GitHub Copilot은 2026년에 언급된 AI 기반 개발자 도구로, 개발 워크플로우 전반을 지원합니다. 이 도구는 코드 완성, 개념 설명, 편집 제안, 에이전트 모드를 통한 파일 검증 등 다양한 기능을 제공합니다. Visual Studio Code, JetBrains IDEs 등 주요 개발 환경에 통합되어 개발자의 직무 만족도를 최대 75% 높이고 코드 작성 생산성을 향상시킵니다.
