# 위스크

## 개요

위스크는 나노바나 2를 기반으로 한 자동 생성 크롬 확장 프로그램으로, 사용자가 쉽게 콘텐츠를 제작할 수 있도록 돕는 도구입니다. 이 프로그램은 5시간의 작업을 통해 개발되었으며, 구글 드라이브에서 다운로드 후 크롬 확장 프로그램으로 등록하여 사용할 수 있습니다. 사용자는 텍스트 파일에서 대본을 읽어와 자동으로 이미지를 생성하는 방식으로 콘텐츠를 제작할 수 있습니다. 관련 자료로는 '[무료배포] 구글 플로우(Nano Banana 2) 자동 생성 크롬 확장프로그램 AI 시대엔 툴도 직접 만듭니다.'가 있습니다.
