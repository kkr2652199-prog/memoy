# LLM Wiki

## 개요

LLM Wiki은(는) 제품명 유형의 엔티티입니다. Andre Karpathy는 AI를 활용한 새로운 연구 방법을 제안하며, LLM을 이용해 개인 지식 기반을 구축하는 'LLM 위키'를 소개했습니다. 사용자가 텍스트 파일, PDF 등을 입력하면 LLM이 이를 분류하고 연결해 주며, 사용자는 복잡한 질문을 통해 깊이 있는 연구 결과를 얻을 수 있습니다. 또한, LLM Wiki는 사용자 자료를 진화시키기 위한 가이드를 제공하며, 개인의 지식을 AI가 관리하는 정원처럼 비유하고 있습니다. 자료를 진화시키지 않으면 '죽은 노트'가 되어 성장할 수 없다는 점이 강조되며, Obsidian과 AI 에이전트를 활용한 자동 정리 방법이 제시됩니다. 그러나 이 시스템은 자기 인식이 부족하다는 문제를 지적받았으며, 사용자가 원하는 통찰력을 생성하기 위해서는 지식 그래프가 필요하다는 점이 강조되었습니다. ⚠️ 모순: 기존 내용에서는 LLM Wiki가 자기 인식이 부족하다고 지적되었으나, 새로운 자료에서는 사용자 자료의 진화와 관련된 내용이 추가되었습니다.
