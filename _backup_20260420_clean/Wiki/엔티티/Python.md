# Python

## 개요

Python은 코딩 초보자를 위한 AI에게 요청하는 방법을 정리한 가이드와 관련된 프로그래밍 언어이다. 이 자료는 파일 구조, HTML/CSS/JS 용어, 요청 예시 및 피해야 할 요청 방식 등을 포함하여, 초보자가 명확하고 효과적으로 요청할 수 있도록 돕는다. 특히, 요청 시 구체적인 파일 경로와 내용을 명시하는 것이 중요하며, 이를 통해 더 나은 결과를 얻을 수 있다. 새 자료는 2025-10-02에 업데이트되었다.
