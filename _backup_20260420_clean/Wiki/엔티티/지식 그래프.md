# 지식 그래프

## 개요

지식 그래프는 다양한 정보와 개념 간의 관계를 시각적으로 표현하는 구조화된 데이터 모델이다. 최근 LLM 엔진을 활용한 지식 그래프 구축 방법이 주목받고 있으며, 이를 통해 개인의 작업 추적과 콘텐츠 생성이 가능하다. 이번 자료에서는 LLM을 통해 생성된 콘텐츠를 블로그와 연결하여 구조화된 정보를 시각화하는 과정이 소개된다. 새 자료는 2026년 4월 18일에 업데이트되었다.
