## 3. v3 공식 정의

### 기여 점수 (회차 N 종료 후, N+1 예측 전 누적)

```
정답 = 당첨 6 + 보너스 1 (7개)
각 정답번호 v를 k개 뇌가 잡음 → 각 뇌 1/k 점
뇌 i의 N회차 기여 = Σ(잡은 v의 1/k)
```

### 예측 (N+1)

```
w_i = Σ_{d<N} contrib_i(d) × 0.995^(N-d)   # recency decay
번호 n 점수 = Σ( n을 잡은 뇌의 w_i )
세트 점수 = 6번호 점수 합 → 상위 5세트
```

- walk-forward: **draw_no < N만** 사용, N+1 정답은 가중치에 **절대 미반영**
