## 3. ensemble.py 변경

- `predict()` (B안 ML) → `_legacy_B_predict()` 보존
- 신규 `predict()` → `hyena_commander.predict()` 위임
- fallback: hyena 빈 결과 시 `_legacy_B_predict()`

```python
def predict(draw_no, db_path, weights_override=None, *, final_score_weights=None):
    from app.lotto4.brains import hyena_commander
    out = hyena_commander.predict(draw_no, db_path, NUM_SETS)
    if not out:
        return _legacy_B_predict(draw_no, db_path)
    return out[:NUM_SETS]
```
