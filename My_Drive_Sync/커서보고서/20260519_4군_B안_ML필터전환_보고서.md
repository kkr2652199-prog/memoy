## 1. stat_generator — 통계 기반 후보 생성

**파일**: `app/lotto4/brains/stat_generator.py` (신규)

**로직 요약**
1. 최근 100회 빈도 → 상위 25개 번호 풀 (200세트 미달 시 30개로 확장)
2. 합계 100~175, 홀짝 2:4~4:2, 고저 2:4~4:2
3. 연번 최대 2쌍, AC≥7, 끝수 중복 최대 2
4. RNG seed = `target_draw * 20260519`, R13 준수

```python
def generate_candidates(target_draw, db, n_candidates=200):
    draws = load_draws_before(db, _history_cut(target_draw))
    rng = random.Random(int(target_draw) * RNG_SEED_MUL)
    for pool_size in POOL_SIZES:
        pool = _freq_pool(draws, pool_size)
        found = _collect_from_pool(pool, rng, n_candidates)
        if len(found) >= n_candidates or pool_size == POOL_SIZES[-1]:
            return found[:n_candidates]
```
