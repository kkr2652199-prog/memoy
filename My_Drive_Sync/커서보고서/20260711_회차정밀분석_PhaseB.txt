## UI 구성 (④ 섹션)

### 1~45 히트맵 (9×5 그리드)
- 누적 출현 횟수 + 최근 20회 `+N` 배지
- zone5 색 배경 (1~10 / 11~20 / … / 41~45)
- 출현 강도 heat bar (0~9단계)

### 오버레이 마커
| 클래스 | 의미 |
|--------|------|
| 당첨 | 실제 당첨 6개 |
| 급등 ↑ | `spike_nums` |
| 갑작출현 ❄ | `cold_comeback` |
| 예측적중 | 현재 뇌 best 세트 적중 |
| 예측오답 | 현재 뇌 best 세트 오답 |
| 놓침 | 실제만 출현 |

### 사이드 패널
- HOT / COLD top5 (클릭 → 히트맵 포커스)
- 6구간 막대 차트
- 핫쌍 · 보너스 프로필

### ②↔④ 연동
- 오답노트 예측/실제 번호 클릭 → ④ 히트맵 해당 셀 스크롤·펄스 강조
- 뇌 탭 전환 시 히트맵 예측 오버레이 갱신
