## `analysis_board` 필드 구성

| 키 | 설명 |
|----|------|
| `freq_grid` | 1~45 누적 출현 횟수 (1..N회 포함) |
| `freq_recent` | 최근 20회 출현 |
| `hot_top5` / `cold_top5` | 누적 HOT/COLD 상위 5 |
| `spike_nums` | 당첨 6개 중 급등 신호 |
| `cold_comeback` | 장기 미출(≥25회 gap) 후 출현 |
| `zone6` | 1~7 / 8~14 / … / 36~45 구간별 당첨 개수 |
| `zone5_color` | 1~10 / 11~20 / … / 41~45 색 구간 |
| `consecutive_runs` | 연속수 런 (2개 이상) |
| `pairs_in_draw` / `pair_hot_hits` | 당첨 쌍·역사 핫쌍(≥3회) |
| `bonus_profile` | 보너스 번호·구간·이월·gap·누적빈도 |
| `sum_total`, `odd_count`, `even_count`, `ac_value`, `ending_digits` | 조합 지표 |

### 컨닝 방지

- **당첨 분석판**: `draws_inclusive` = `_get_draws_before(N)` + N회 당첨 행
- **오답노트 태그**: `build_snapshot_context(draws_before)` — 예측 시점만 사용
