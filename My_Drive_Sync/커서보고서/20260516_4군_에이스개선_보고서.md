| **B 탐색형** | 3 | top-20 | `_sample_combo_focused(..., temperature=0.5)` |

**신규 함수:**
- `_sample_combo_legacy()` — 기존 로직 보존

**predict() 핵심 발췌:**


## STEP 3 – struct_brain 변경

### 기존 (문제)
- `_generate_candidates`: **10,000× random.sample(1..45,6)** + soft match ≥5
- 구조 예측과 무관한 **Monte Carlo**

### 신규 (구조 기반 결정론적)

1. **`_score_numbers(y_hat)`** — odd/high/decade/tail 4요소 가중합 → 1~45 점수
2. **상위 15번호** 풀 선정
3. **15C6 = 5,005** 전수 평가 → soft match ≥5 → struct_distance 정렬
4. Jaccard<0.5로 **5세트** 선택

**신규 함수:**
- `_generate_candidates_deterministic(y_hat)` — 결정론적
- `_generate_candidates_legacy(y_hat, rng)` — 기존 보존

**predict() 경로:** `_pick_sets_from_y_hat` → deterministic 우선 → legacy fallback
