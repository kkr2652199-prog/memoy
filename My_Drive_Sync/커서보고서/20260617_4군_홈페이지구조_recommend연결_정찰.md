## STEP2 — JS 렌더 패턴 (lotto4.js RAW)

### 공통 인프라

- API 베이스: `const API = '/api/lotto4/v13'` (8행)
- JSON fetch: `fetchJsonHandled(url, options)` — POST 지원 (`collect-draws`, `predict` 사례)
- 탭 전환: `switchView(name)` — `.view-panel` / `.nav-btn` active 토글 (212~218행)

### truth 탭 패턴 (추천 동일 적용)

```javascript
// 1) 렌더 함수
function renderTruthCards(data) {
  const host = document.getElementById('truthCardsHost');
  // … cards.forEach → host.innerHTML 조립
}

// 2) 로드 함수
async function loadTruth() {
  const st = document.getElementById('truthStatus');
  setStatus(st, '불러오는 중…', true);
  try {
    const data = await fetchJsonHandled(API + '/truth');
    renderTruthCards(data);
    setStatus(st, '검증 수치 ' + (data.cards || []).length + '카드 표시', true);
  } catch (e) {
    setStatus(st, '오류: ' + (e.message || e), false);
  }
}

// 3) 탭 클릭 시 lazy load (1834~1841행)
if (v === 'truth') loadTruth();

// 4) 새로고침 버튼 (1850행)
document.getElementById('btnReloadTruth')?.addEventListener('click', loadTruth);
```

### 번호 5세트 렌더 참고 (predict `renderSetsForBrain`, 626~654행)

- `.sets-grid` 컨테이너 + `.set-row` 카드 반복
- `renderBallsHtml(nums, null, null)` 로 공만 표시 (적중 비교 없음)
- recommend는 `combined_score`·`brain_contributions` 메타만 `.set-row-meta`에 추가

**결론**: truth(탭+load+render) + predict(볼·세트행) 패턴 혼합으로 **함수 1~2개**면 충분. 대공사 불필요.
