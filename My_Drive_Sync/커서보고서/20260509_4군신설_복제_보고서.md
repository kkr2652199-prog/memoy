## STEP 3 — 라우트 등록

- 본 프로젝트는 Flask가 아니라 **FastAPI** 이므로 `register_blueprint` 대신 **`app.include_router`** 사용.
- 변경 파일: `app/main.py`
  - `from app.lotto4.v13_routes import router as lotto4_v13_router`
  - `app.include_router(lotto4_v13_router)`
  - `startup` 에 `init_lotto4_db()` 호출 추가.
- 3군 라우터(`lotto3_v12_router`) **기존 그대로**.

- **4군 API 베이스:** `/api/lotto4/v13` (`v13_routes.py` 의 `APIRouter(prefix=...)`).
