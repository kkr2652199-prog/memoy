# 4군 테스트로또 상세페이지 — 증거 보관소 완성 보고서

📅 2026-07-11 KST · IMPLEMENT (app/testlotto만)

## 목표

상세페이지를 **「우리 논리의 증거 보관소」**로 완성 — 7뇌 각자의 근거(필살기) + 세트별 자신감(confidence) + 보조 4뇌 신호 노출.

## STEP 1 — 필살기 한 줄 (short_desc)

`app/testlotto/brains/registry.py` 7뇌 전원 `short_desc` 추가.

`get_draw_detail()` 반환:
- `brain_verdicts[].short_desc`
- `brains[].short_desc`
- `brain_meta` (7뇌 전체 메타)

## STEP 2 — 보조 4뇌 화면

- 신규 `app/testlotto/aux_analysis.py` — 보조뇌 신호 분석·`aux_brains` 섹션 빌드
- `aux_analysis_json` — brain_page에 lazy 저장 (조회 시 미기록이면 계산·UPDATE)
- `walkforward.py` — 신규 복습 시 aux_analysis_json 즉시 저장
- API `aux_brains` (예측3뇌와 분리), `signal_role: 신호/경고`, `scoring_note: 적중률 채점 대상 아님`

## STEP 3 — confidence 시각화

- `predicted_sets[].confidence` API 그대로 노출 (채점 로직 미변경)
- `most_confident_set_no`, `most_confident_score`, `confidence_summary` 추가
- UI: 신뢰도 바 · BEST/최고신뢰 태그 · 한 줄 요약

## 7뇌 노출 스크린 구성

```
① 당첨 확정
② 채점·비교
   ├ 3뇌 판정 스트립 (이름 + 필살기 + 등수 + best 세트)
   ├ 뇌 선택 스코어카드 (필살기 부제)
   ├ confidence 한 줄 요약
   ├ 5세트 (신뢰도 바 + BEST/최고신뢰 강조)
   ├ 놓친 패턴 / 학습 피드백
   └ 보조 4뇌 그리드 (신호/경고 · 실제당첨 + 예측3뇌별 신호)
③ 등수별 당첨
⑤ 뇌 학습 누적
④ 패턴·구조
```
