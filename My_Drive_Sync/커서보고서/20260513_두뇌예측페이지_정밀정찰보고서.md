## STEP 5 — 기능 상세 비교 (3군 vs 4군)

### 5-1. 3군 (코드 기준)

1. **회차 ◀▶**: `lotto3NavDraw`가 `_lottoDrawList`(예측이 존재한 회차 위주로 수집)에서 인덱스 이동 후 `lottoLoadSavedPrediction` → `GET /api/lotto3/v12/predictions/draw/{d}`.
2. **예측 실행**: `POST …/predict/{no}` 후 `all_predictions`로 즉시 렌더 또는 `predictions/draw` 재조회.
3. **저장 불러오기**: 별도 버튼 없음. 드롭다운/◀▶/선택 시 자동 `lottoLoadSavedPrediction`.
4. **“고정/고적중” 뇌**: 체크 시 `elite-tags`로 필터, 해당 뇌만 탭·카톡 대상. 매칭 0이면 안내 문구 블록.
5. **뇌 탭**: `BRAIN_LIST` 고정 8뇌; 탭 클릭 `lotto3SwitchBrainTab` → 동일 회차로 `renderPredictionsByBrain` 재호출.
6. **세트 카드**: 구간별 배경색 `lottoMiniBallBg`; 신뢰도는 `v<=1`이면 ×100 해서 `%`; `matched_count`/보너스로 1~5등·미당첨·추첨 전; 적중 볼 `is-hit`; DZ 플래그 가능.
7. **1~5등 모달**: 전용 패널, 실제 당첨 줄 + 등수별 섹션 + `renderMiniBall`, 모달 내 회차 점프 등.
8. **카톡**: 헤더/푸터 고정 문구, 뇌별 표시명, 세트 이모지, **역대 전적 한 줄**(`lotto3KakaoBrainRecordLine`), 엘리트 필터와 동일 뇌 목록.
9. **나노 전적**: 각 탭 버튼 아래 `lotto-brain-nano` (1~5등 누적 카운트).
10. **rank 뱃지**: 가중치 순위 `#` 형태는 탭에 없음 (대시보드 brain_power 기반 **누적 적중 나노**만).
11. **엘리트 강조**: 엘리트 필터 시 비대상 탭 `disabled` 등.

| 10 | rank 뱃지 | **4군만** 탭에 `#순위` (`brain/ranks`). 3군 탭에는 없음 → “불일치”이나 기능 상 **추가**에 가깝다. |
