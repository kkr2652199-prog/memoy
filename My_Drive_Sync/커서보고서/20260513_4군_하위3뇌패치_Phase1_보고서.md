`normalized(%)`는 응답에 포함되지 않음. 최대 가중치 `1.469` 기준으로 UI에서 `round(w * 100 / 1.469, 1)` 산출 가능.

    {"brain_tag": "v13_ensemble", "current_weight": 1.001, "rank": 2},
    {"brain_tag": "v13_trend", "current_weight": 0.565, "rank": 3},

### D-2 `GET /api/lotto4/v13/brain/status` → **200**

- `brain_profiles`: **8개** (뇌별 누적 적중 통계·method 등).
- **참고**: 응답에 **`weight` / `active` / `last_predict_draw` 필드는 현재 구현에 없음**. 가중치는 D-1·DB와 조합. DB상 v13 예측 `MAX(target_draw_no)`는 **1225** (1224 다음 회차 예측 존재).

```json
{
  "grade": "역전 마스터",
  "total_predictions": 45040,
  "best_record": {
    "brain_tag": "v13_ensemble",
    "target_draw_no": 789,
    "matched_count": 5,
    "bonus_matched": 0
  }
}
```

### 4군 페이지에서 확인 가능한 항목 (현 구조)

- **순위·가중치**: `brain/ranks` + (선택) 정규화 % 클라이언트 계산  
- **뇌별 통계·등급**: `brain/status`  
- **대시보드 요약·분포**: `dashboard-summary`  
- **회차별 적중 기록**: `predictions/{draw}`  
- **명예의 전당**: `hall-of-fame`
