## 1. Hyena 메커니즘 (가장 중요)

### 진짜 정체
> **"5뇌 25셋트 → 합의점수 → Top15 풀 → 5005조합 → tier1 → 5세트"**

### 전체 흐름
```
입력: 5뇌 결과 25개 셋트 (각 뇌 5세트씩)
  ↓
1단계: 모든 번호에 가중치 곱해서 합의점수 산출
       LSTM(2.5) > Fusion(2.0) > LLM(1.5) > Stat(1.0) = Markov(1.0) = Hyena(1.0)
       → FALLBACK_BRAIN_WEIGHTS
  ↓
2단계: 1~45번 중 점수 상위 15개 선정 (Candidate Pool)
       _select_candidate_pool(consensus, 15)
  ↓
3단계: 15C6 = 5005개 조합 전부 생성 (combinations)
  ↓
4단계: 각 조합의 합계점수 계산 → 정렬
  ↓
5단계: tier1_filter 통과한 것만 5개 추출
       (실패 시 30회 재시도 → 우회 로직)
  ↓
6단계: _is_diff_from_army1(combo, army1_sets, threshold=0.4) 검사
       → 이게 Snake가 외부에서 호출되는 부분
```

### 핵심 발견
- **명시적 "셋트 합성"은 없음**
- 5005 조합을 다 만들면 **부분 적중 셋트의 번호가 자연스럽게 재조합**됨
- → 형이 말한 "4개 적중 + 다른 셋트 1번호" = 5005에서 자동 발생
- **이게 1등 10회의 비밀**

## 2. 5뇌 각각의 핵심 로직

| 두뇌 | 알고리즘 | 입력 | 출력 |
|------|---------|------|------|
| **Stat** | 50회차 지수감쇠 + Gap 30~50 보너스 + 5회차 연번/빈도 + 피드백(Trap/Hit) | 최근 50회 + 피드백 | 5세트 |
| **Markov** | 전이행렬 + 현재번호에서 80스텝 Random Walk | 전체 회차 | 5세트 |
| **LSTM** | SEQ_LEN=50, Hidden 128, 2층 신경망 → PMF | 50회차 시퀀스 | 5세트 |
| **LLM** | Gemma(LM Studio) + Overdue + Hot Streak + 홀짝비율 | 텍스트 50회 | 5세트 |
| **Fusion** | 4뇌 확률벡터 결합 + DB 가중치 | 4뇌 PMF | 5세트 |
| **Hyena** | 25셋트 합의점수 → Top15 → 5005조합 → Top5 | 5뇌 25셋트 | 5세트 |

→ **6뇌가 모두 다른 입력/다른 알고리즘 = 자연스러운 다양성**

## 3. 보조 두뇌 3개 (형의 비밀 무기)

### Snake (V8 엔진) ⭐ 가장 중요
- **미당첨 분석 기반**
- **기존 30개 셋트와 Jaccard 유사도 0.4 미만인 조합만 채택**
- → 다양성 강제, 1군 셋트랑 일부러 다르게
- **외부 SOTA의 NCL을 더 단순하게 구현한 것**

### Cluster
- K-Means로 번호를 5개 그룹 분류
- 빈도/최근성/확률 기준
- 최근 당첨 그룹 가중치 ↑

### Entropy
- Shannon Entropy로 정보량 측정
- 평균적인 번호 가중치 보정 (0.85~1.5)

## 4. Filters / Feedback (1군 엔진룸)

### tier1_filter
- 합계: **80 ~ 210**
- 홀수 개수: **1 ~ 5**
- 구간 히트: **2개 이상**
- 최대 연번: **3개 이하**

### Feedback (Layer 3~5)
- **Hedge 알고리즘 eta 1.5**
- **Lottery-Aware Scoring** (1등 100, 2등 50, 3등 30, 4등 10, 5등 3)
- 매 회차 두뇌별 가중치 동적 업데이트
- **결과: LSTM 적중률 36% → 69%** (Layer 3.5에서 검증됨)

## 6. V12 설계안 (동생 추천)

**"1군 메커니즘 응용 + 2군 정체성(미당첨 틈새) 유지"**

|----------|-----------|-----------|
| stat | army2_stat 재태깅 | + Gap 30~50 보너스 |
| markov | army2_markov 재태깅 | + 80스텝 Random Walk |

### 2군만의 차별화 (정체성)
